        &lt;h3&gt;第三卷第一百九十五章  陈子澜的新年&lt;/h3&gt;
<br/>
<br/>        陈子澜躺在医院的病床上，这是她这几十年最憋屈的一个新年，手脚不能动。
<br/>
<br/>        原本想新年回家的，但是潘青松怕移来移去会对她造成伤害，所以就将家搬到了病房，他们住的是vip病房，里面的东西一应俱全，所以一家人就在医院过起了新年。
<br/>
<br/>        儿子潘禺得知母亲出了车祸，想办法休了探亲假回来陪着母亲过年。一家三口虽然是在医院里，也是其乐融融。
<br/>
<br/>        看着一老一少两张同样帅气的脸庞在眼前晃来晃去，陈子澜的心里满满的都是幸福。
<br/>
<br/>        潘青松做好饭，儿子潘禺先喂着母亲吃完，然后父子俩人坐在小桌子旁，边吃边聊。
<br/>
<br/>        “小禹，今天过年，咱爷俩喝一点。”潘青松很开心儿子能回来陪他们过年。
<br/>
<br/>        “行，您是老大您说了算。”潘禹在部队上也会喝一些酒，中国的酒文化是无处不在的。
<br/>
<br/>        潘青松拿了一瓶酒出来：“这个是你小姨给我拿的酒，平时工作需要我是很少喝酒的，今天过年我们爷俩可以多喝点，而且这个酒不错。”
<br/>
<br/>        潘禹和父亲碰了一下杯：“爸、妈新年快乐！”
<br/>
<br/>        “嗯，这个酒确实不错，入口甘甜软绵。”潘禹和父亲干了一杯酒，点点头评价。
<br/>
<br/>        “你这个臭小子，这个酒可是好酒，我都舍不得喝。”潘青松笑着说。
<br/>
<br/>        陈子澜侧头看着坐在小餐桌边的父子，人生如此，夫复何求？虽然她现在躺在床上，可是这种温馨的感觉真好。
<br/>
<br/>        她突然就想到了在甸城碰到的那几个孤儿，他们的新年该如何过？在那样一个冰冷的村子里，为什么会有那么多孤儿？而且那个村子人口并不多，孤儿从何而来？
<br/>
<br/>        陈子澜轻轻的叹口气，还是她的工作经验欠缺，贸贸然行动，断送了郑浩的性命，他们出去只有四个人知道，司机是他们自己找的，莫非，整个甸城都在黑色势力的布控之下？
<br/>
<br/>        想到这里，陈子澜不由得动了一下，痛苦也随之而来，她轻轻的“哼”了一身，正在喝酒的父子俩同时起身，同时到她的身边，一左一右看着她：“怎么了？”
<br/>
<br/>        陈子澜心里感动，父亲是她教出来的，儿子也是她教出来的，父子俩个人身手都很敏捷：“没什么，就是刚才想事情不由自主的动了一下。”
<br/>
<br/>        “想什么事情啊，你这个时候是要好好休养的，我还想等你好了赶紧再给我指导指导呢。”潘禹轻轻的替母亲捏了捏被角，低下头在母亲的额头上碰了碰。
<br/>
<br/>        潘青松看着母子俩人的互动，微微的笑了，只是眼里有泪花闪动，要知道陈子澜昏迷的那些天，他都在想，如果陈子澜醒不过来，他活着也就是行尸走肉，还好，还好，子澜终于是醒过来了，而儿子也有父亲和母亲。
<br/>
<br/>        “你妈妈那个脾气你又不是不知道，来，给我们讲一下，你想到了什么？”潘青松和儿子又重新坐到了餐桌前。
<br/>
<br/>        “我想到了在甸村看到的孤儿，不知道那些孩子是怎么过的新年，而且我很奇怪，那个村子人口不多，哪里来的那么多孤儿？况且郑浩当时闻到了不同的味道，那个味道就是我们需要找的东西。”陈子澜皱着眉头：
<br/>
<br/>        “我越想越觉得奇怪，我和郑浩去甸村的时候，除了我们四个人之外，只有司机知道，司机一没喝酒，二不存在瞌睡的情况，当时的车况良好，我们三个人，司机受伤的情况最轻。”
<br/>
<br/>        “一般情况下，除非是特别机警的人在出车祸的一瞬间能够采取措施，要不然司机是不会只受轻伤。”潘禹和父亲讨论过母亲受伤的情况。
<br/>
<br/>        “所以，我在想，甸城，怕是一个有组织，有计划的窝点。”陈子澜的语气里是透着肯定。
<br/>
<br/>        “你现在想什么都不行，这个只是你的判断，没有根据，我们讲究的事实，是证据。你啊，好好休息吧。”潘青松望着陈子澜那苍白的脸庞，他不允许他老婆以后有任何不好的情况发生。
<br/>
<br/>        甸城很多年前就是一个鱼龙混杂的地方，不过就是这几年有秦寿在，看起来很是太平，可是谁又能保证太平的下面会是一个什么样的光景？
<br/>
<br/>        “嗯，我就是心存疑虑，现在是什么都干不了。不过子昂说她要去甸城，我还是有些担心的，一旦牵扯了利益，情况就不一样了。”陈子澜的语气里满是担忧。
<br/>
<br/>        “妈，你也不用想那么多，小姨是生意人，她能走到今天也不是一般人，你想到的她也会想到，你要是担心她安全的话，到时候我帮她联系上一俩个保镖，陪着她不就行了。”潘禹只比陈子昂小几岁，小时候也是经常在一起玩的，对于陈子昂的聪慧是知道的，只是她心底善良，从不朝坏的方向去想，不过这也是陈子昂最大的软肋。
<br/>
<br/>        “保镖只是明面上的，暗地里的呢？”陈子澜还是忧心忡忡，不行，得找个时间给子昂好好说说，让她注意。
<br/>
<br/>        “妈，你别担心，好好养身体，我会给小姨提醒的，过几天我去北京看看小姨。”潘禹安慰着陈子澜。
<br/>
<br/>        潘青松：“子澜，张函对子昂念念不忘，而且他们孩子都有了，张函也会想办法保护子昂的，这个你放心。”
<br/>
<br/>        “你别说张函了，子昂要是和张函沾上边了，事情更麻烦，陆家怕不是会盯着子昂？”陈子澜并不知道新年陈子昂家里发生的一切，他们在担惊受怕的时候陈子昂家里看着却其乐融融。
<br/>
<br/>        “该来的一定会来的，躲也躲不过去，子昂怕是和张家不死不休了。”其实大家都明白，陈子昂有孩子的事情怎么可能瞒得住？如果说陈子昂找个人无声无息的结婚，可能没有人会去关注陈子昂的孩子，可是偏偏陈子昂单身。
<br/>
<br/>        一个单身了很多年的女人突然有了俩个孩子，而且这个单身的女人还不一般，在如今的社会难保不被人当做茶余饭后的调料。
<br/>
<br/>        三个人的新年因为讨论陈子昂感觉有点伤感，不过病房里还是温馨的感觉。
<br/>
<br/>        “小禹，子澜，不管怎么样，我们以后共同面对，我们是一家人！”潘青松举起酒杯：“祝我们的家人身体健康，幸福美满！
<br/>
<br/>        人生其实很简单，简单的都是希望自己身边的人身体健康，幸福美满或者是能够执子之手与子偕老。
