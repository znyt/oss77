　　　　当年，白家是白镇的最大家族，由于新社会成立，发展过快，后面迁居到了温城。
<br/>　　陈默忽然道：“你要小心了，你虽然现在修炼有成，但是你越强大，也就越危险”
<br/>　　“你在白镇数次显化，肯定会吸引一些隐藏家族的目光，况且你现在杀了白家的人”
<br/>　　“所谓觉醒，人类可觉醒，生物同样如此，况且这个世界现在是已人为尊，万物之长，古时候称之为修炼者，近代才叫觉醒者，人类觉醒，是正道，而生物觉醒，得到强大的力量，那么就是怪物，妖怪”
<br/>　　“在这个上知天文下知地理的高度文明时代，却修炼资源匮乏，到了现代，更是稀少，很多隐世家族反而需要这些怪物，这些觉醒类生物，捕抓了进行炼药修炼”
<br/>　　“炼药？”涂小安微微一愣，眉头挑了挑。
<br/>　　“对，觉醒者想要变的更强，就需要外力资源，而生物觉醒，最适合拿来炼药，它们的血跟肉，都成了天材地宝。”
<br/>　　陈默继续道：“这片山川大地，不知道隐藏多少人类觉醒者，也不知道隐藏多少觉醒类生物”
<br/>　　“觉醒类生物邪性者盗吃牲畜，食人脑髓，力求变强，而人类同样如此，发现有什么出世的觉醒类生物，会立刻捕抓，争夺资源”
<br/>　　涂小安静静的听着，终归是弱肉强食，你强你可以杀人，你弱，你被人杀，被人吃，被人炼药。
<br/>　　“不过觉醒类生物不少，但可没一种像你这样大张旗鼓的，生怕不被人知道”陈默淡淡的瞥了涂小安一眼。
<br/>　　好像看傻子一样。
<br/>　　你可以高调，你可以装逼，但是要付出代价的，这跟钱不露白是一个道理。
<br/>　　你露了，总会被有心之人，惦记上。
<br/>　　这是人类统治的时代，任务生物都要乖乖的蛰伏下来。
<br/>　　“当今社会，不仅仅有隐藏的修炼家族，就连国家，都专门有一个特殊部门，是由人类觉醒者成立”陈默幽幽的说：“但凡那个地方出现了觉醒类生物出来为非作歹，就会第一时间出来消灭”
<br/>　　涂小安咽了咽口水：“博士，那你的意思是说，我摊上事了呗”
<br/>　　他数次在白镇上空蛇影显化，看似威风凛凛，但背后却隐藏着杀机。
<br/>　　已人类的角度，出现怪物，那么就是出来降妖伏魔，实际上也算争夺资源，那么厉害的一条蛇，任何器官都是天材地宝。
<br/>　　要知道当初的江行泽可是生生的喝着吸血蛇的蛇血，直至最后觉醒。
<br/>　　“不，你不是摊上事，你是摊上大事了”陈默到有点恶趣味说。
<br/>　　呃...
<br/>　　他这是幸灾乐祸？
<br/>　　不过涂小安并不怕，他现在身负两种属性能量，雷电跟火焰，威力强大，杀人于无形。
<br/>　　谁敢来找他麻烦，作死。
<br/>　　况且，他也没做什么恶事。
<br/>　　涂小安脸上的傲气被陈默看在眼中：“小子，人外有人，天外有天，你纵然登基坐殿，成为蛇王，但是这天是人类的天，这地是人类统治的土地”
<br/>　　几个意思呀，这是让他跑路？
<br/>　　“博士，你说了那么多，到底是什么意思”
<br/>　　涂小安有点懵逼了。
<br/>　　“你现在继续待在白镇，恐怕没多久就会吸引很多觉醒者来抓你”
<br/>　　陈默想了下开口。
<br/>　　这还不是让自己跑路？
<br/>　　涂小安心里已有数：“谢谢博士的提醒”
<br/>　　他其实就想补偿一些未知领域的知识。
<br/>　　不管是人类觉醒者，还是别的觉醒类生物，对于涂小安来说。
<br/>　　人不犯我我不犯人，他的身份现在也有点复杂，他既是人类，也是一条蛇。
<br/>　　反正大丈夫生于世，最主要坦荡荡，无愧于心。
<br/>　　这个夜晚，涂小安跟蛇博士陈默聊了很多，一个小时后，他们散去了。
<br/>　　涂家后院，涂小安摸着黑耳小八漆黑如墨的毛发，淡淡的道：“小八，这个家，交给你，你能守护好吗”
<br/>　　“嗷...”
<br/>　　小八的阴阳眼在这个黑夜闪闪发光，内有威势。
<br/>　　涂小安不怕有人找上门来，但他一番思忖下还是决定先离开白镇。
<br/>　　为什么呢，因为他要变的更强，如今在白镇已经满足不了他了。
<br/>　　他的实力也很难继续得到长足的提升。
<br/>　　所谓潜龙入海，他要去温城玩玩，什么白家，涂小安根本就不怕。
<br/>　　他到是担心自己留在白镇，会为自己这个家，会为白镇带来麻烦。
<br/>　　另外涂小安也不能一直留在白镇，现在已经化形，应该出去闯闯了。
<br/>　　等他下来回来的时候，便是真正化形的时候。
<br/>　　不过既然准备离开，涂小安也要为这个家做点准备，光一只小八守护，跟一只骄傲的孔雀，涂小安还是有点不放心。
<br/>　　他利用系统，制造了三千只电蜂出来。
<br/>　　另外他现在拥有火焰之力，同时，也制造了三千只火蜂出来，蛰伏在这个家的周围。
<br/>　　就算到时候有觉醒者找上门，三千只电蜂，跟三千只火蜂，也会让来人灰飞烟灭。
<br/>　　几天后，等涂小安真正离开的时候。
<br/>　　九月处....
<br/>　　涂小安悄无声息的从白镇脚踏实地的离开，涂家的家蛇消失了，白镇的蛇王在往后的日子里也不在显化。
<br/>　　不过蛇王庙的香火还一直非常的旺盛，陆陆续续中，涂小安的信仰之力一直在涨动着。
<br/>　　一辆开往温城的大巴车上，涂小安泛着兴奋的心情一屁股坐了下去，准备沿途看看风景。
<br/>　　其实现在已他的手段，直接飞过去都行，但涂小安选择了最普通的方式。
<br/>　　大巴车行驶出去，按照白镇跟温城之间的距离，莫约差不多要掀开一个半小时的路程。
<br/>　　忽然，涂小安有种农村娃要进村的小激动，当初他就去过温城，只不过是被蛇博士硬抓了去。
<br/>　　逃出来之后，他也没在温城停留，顺着大山就回到了佘山。
<br/>　　此番进城，对于涂小安的意义完全不同。
<br/>　　就好像要去一座别的城市打工似的。
<br/>　　拿出手机，插上耳机，涂小安准备惬意的听一听音乐，沿途看看风景，他现在去温城并没有什么具体的打算，走一步算一步。
<br/>　　谁能相信，这区区一辆大巴车上坐的，居然是白镇人民无比信仰的蛇王，他要是显出原形，这辆大巴车都装不下。
<br/>　　恐怕都要原地爆炸。
