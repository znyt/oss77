r/>　　    他拿出传功玉符来，边看着边对着，才依依发现，传送阵地面上绘制的花纹，有十几处，被人给修改或者毁掉，这才造成传送阵的失效<br/>　　<br/>　　    陈默看到这个，才对整个事有了个概念<br/>　　<br/>　　    在传送到最后，贝宁云的仇敌，利用了阵法的改变，将贝宁云给强行留了下来，然后想着不能让人知道<br/>　　<br/>　　    但是却没有想到，贝宁云不仅有着保命的手段，这个手段还是一直从来没有给谁说过，所以仇敌也是不知道的。<br/>　　<br/>　　    因此，在最后紧要关头，贝宁云直接使用这个手段，不仅将仇敌给消灭了，还将自己也给连带着消灭了<br/>　　<br/>　　    他自己成为灵体后，是可以利用传送阵逃离的，但是却没有想到传送阵被人破坏，丝毫也跑不了<br/>　　<br/>　　    还有就是，通过贝宁云的记忆，陈默发现这个人对于修炼非常的衷，但是对其他手段，比如阵法和炼丹等，都是没有学习过，所以对阵法一无所知。<br/>　　<br/>　　    因此才会在阵法被人修改后，却不知道如何修复。<br/>　　<br/>　　    星级传送阵可不能乱来，不然传送一但出事故，也许下次传送可能就会将人送到太阳上去<br/>　　<br/>　　    不过，这些地方的纹路，对陈默来说，倒是简单的在传功玉符上，都有着详细的记载，只要通过这些记载，一一对照，就能将这个传送真修好。<br/>　　<br/>　　    至于说启动这个星级传送阵法，可就有些为难了，从哪里找到这么多的上品灵石<br/>　　<br/>　　    就算陈默他自己，都是没有见过上品灵石的，口袋里也就几块初等灵石。这次能见到上品灵石，还是阵法遗留下来的渣渣<br/>　　<br/>　　    等等，陈默似乎想到了什么&lt;b&gt;章节内容正在努力恢复中,请稍后再访问。&lt;/b&gt;想和更多志同道合的人一起聊《修真高手的田园生活》，微信关注“热度网文或者rdww444”与更多书友一起聊喜欢的书<br/><br/><br/>Ps:书友们，我是小学嗣业，推荐一款免费小说App，支持小说下载、听书、零广告、多种阅读模式。书友们快关注起来吧！<br/><br/><br/>
