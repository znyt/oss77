        林若琴是林贵的女儿，林贵又是傅老太爷的助理，林若琴若是杀害马智杰的真凶，大哥傅夜沉若是包庇林若琴的话……
<br/>
<br/>        想到这里，童筱颖又自顾自地摇了摇头。
<br/>
<br/>        大哥一定不是那种人！
<br/>
<br/>        只是，林若琴为什么要杀了马智杰？！
<br/>
<br/>        童筱颖想到这里，心里莫名其妙地不安起来。
<br/>
<br/>        “滋滋——”
<br/>
<br/>        手机的振动拉回了童筱颖的思绪，当她定睛看到屏幕上的来电显示时，不禁喜上眉梢。
<br/>
<br/>        “小卫！”童筱颖立马接听了电话。
<br/>
<br/>        “筱颍，今天一切安好？”傅昼景的声音格外温柔。
<br/>
<br/>        童筱颖只觉心窝里暖暖的，嘴里像含了蜜糖一样甜甜地说道：“嗯！今天报到有赵斌在一切都顺利，而且很快就和室友认识了。而且，我今晚还……”
<br/>
<br/>        “嗯？”
<br/>
<br/>        “还带着室友一起去临海城里最有名的小吃街吃东西。”童筱颖牵强地回答道，其实她本来是想说还遇到了大哥傅夜沉，后来想想还是算了。
<br/>
<br/>        “你开心就好。”傅昼景欣慰道。
<br/>
<br/>        童筱颖抿了抿唇，接着说道：“我们每个周五下午就没课了，所以，我每个星期都回来，好不好？”
<br/>
<br/>        “好！”傅昼景微笑着应道，此刻的他还坐在地利置业大厦的总经理办公室里加班，而季思妍的出现，让他脸上的笑容渐敛，“筱颍，你早点休息，我还得继续加班。”
<br/>
<br/>        “嗯，晚安。”童筱颖完全不知情地回应。
<br/>
<br/>        “晚安。”傅昼景与童筱颖通完电话后，便将电话给挂了，目光咄咄逼人地看向季思妍，冷冷地问道，“你怎么来了？”
<br/>
<br/>        季思妍踏着高跟鞋，“啪嗒啪嗒哒”地走了过来，指尖划过傅昼景的办公桌桌面，慢慢地挪到了他的肩膀上，并站到了他的身后，双手一搭，很自然地给他揉着肩膀按起摩来。
<br/>
<br/>        “我是来陪你加班的啊！”季思妍的声音有点低哑，听起来软绵绵的，没什么力气。
<br/>
<br/>        傅昼景刚想起身，却被季思妍两手摁住。
<br/>
<br/>        “我说过，我两现在只剩下合作关系。”傅昼景微微皱了下眉头。
<br/>
<br/>        季思妍微微俯身，双手环住傅昼景的脖子，将红唇凑到傅昼景的耳边，呵气如兰地接着说道：“阿景，我爸问起我两的关系，我一时嘴快，说我两是男女朋友关系了，怎么办？”
<br/>
<br/>        “你——”傅昼景欲言又止。
<br/>
<br/>        他现在与季思妍合伙开的公司，有百分之八十的资金来源是季思妍的父亲给予的。
<br/>
<br/>        现在，季思妍如果又闹这么一出的话，只会让他陷入尴尬的困境中。
<br/>
<br/>        “这又有什么关系呢？反正，你和筱颖是合法夫妻的事情，外人又不知道。而且，我爸爸知道你是我男朋友后特别高兴了！”季思妍柔柔地说，温热的红唇，轻轻地咬了咬傅昼景的耳垂，“我爸还说，我哥没出息。他很看好你。”
<br/>
<br/>        季思妍这话的另外一层意思是什么，傅昼景不用猜就知道。
<br/>
<br/>        季董事长是想让他傅昼景当上门女婿，至于日后，他是辅佐季运恒，还是继承季氏集团，还是一个未知数。
<br/>
<br/>        这与他现在在傅氏集团辅佐他大哥傅夜沉，没区别。
<br/>
<br/>        “阿景，如果，你跟我撇清了关系。仅仅只是合作伙伴的关系话，我想，你的发展空间也就局限在自己的交际圈里了。倘若，你成为我把的女婿。踏足我爸爸的交际圈，这对你来说，是无可厚非的资源。你难道，就不考虑考虑吗？”季思妍一边轻轻吻着傅昼景的脸颊，一边说着糖衣炮弹一样的话。
<br/>
<br/>        她如葱根的手指，顺着傅昼景的衣襟，慢慢地往下滑去。
<br/>
<br/>        已经跟过他那么多次了，他身上每一个敏感点，她都摸得一清二楚。

<br/>        就在她的手抵达他腹部以下时，傅昼景猛地捉住她的手腕，制止了她接下来的动作。

<br/>        “大哥已经对我起疑心了，这段日子，我得收敛收敛。”傅昼景冷静地说道，双眸如深潭里的水，平静无波。

<br/>        季思妍有些猜不透傅昼景的心思。

<br/>        “好。你不用这么快就拒绝我，我给你时间考虑。阿景，你要想清楚。筱颖给不了你，你想要的一切。她的存在，只会成为你的负担。”季思妍不打算把傅昼景逼得太紧，但也希望傅昼景能看清楚事实。
