        童筱颖会心一笑，一手托腮，一手拿着茶匙搅动着杯中的咖啡：“你挺了解他。如果你不喜欢一个人，那么就不会去了解那个人吧！可是，你却了解十爷。”
<br/>
<br/>        “我——”季雅姗欲言又止。
<br/>
<br/>        昨天晚上，他虽然跟她睡在了一起，却没有要了她的身。
<br/>
<br/>        他信守承诺，而且，睡在他怀里，让她很有安全感，甚至，她还在纠结，他会不会找筱颖来替换她。
<br/>
<br/>        从今以后，睡在他身边的女人，不再是她季雅姗，或许是筱颖。
<br/>
<br/>        季雅姗心里很矛盾，她有个时候觉得这样挺好，她终于可以解脱了，但是，有个时候心里又有些慌，不希望他身边出现其他女人。
<br/>
<br/>        “筱颖，你真的结婚了吗？”季雅姗忽然一本正经地问。
<br/>
<br/>        童筱颖也非常一本正经地点点头，应了声：“嗯，我真的已经结婚了！”
<br/>
<br/>        “幸好……”季雅姗心里暗自松了口气，就连她自己为什么要松口气的原因，她都没有任何这方面的意识。
<br/>
<br/>        自从她被十爷包养后，在物质上，十爷给她了想要的一切，甚至她有的，就连她那同父异母的姐姐季思妍都没有。
<br/>
<br/>        而唯独，十爷的那颗心，她不曾想过要去触及。
<br/>
<br/>        可就在昨晚，十爷说要调查筱颖的事情后，她心里莫名其妙地产生了一种危机感。
<br/>
<br/>        季雅姗心里一直在自我矛盾，情绪上也非常的纠结。
<br/>
<br/>        童筱颖微笑着说：“你放心好了，十爷不会拿我怎样，我是有夫之妇。而且，今天我老公出差回来了。”
<br/>
<br/>        季雅姗看着童筱颖脸上的笑容，双手托腮抵在圆木桌上，期待地问道：“将来，你若是和你老公有了宝宝，可不可以让我当宝宝的干妈？”
<br/>
<br/>        “好啊！这样一来，以后我的宝宝，就有两个妈妈宠着它了。”童筱颖情不自禁地憧憬起来。
<br/>
<br/>        以后……以后，她会和昼景过得越来越好；以后，她会和昼景有一双儿女；以后，她会和昼景白头偕老，子孙满堂。
<br/>
<br/>        童筱颖最向往的不是大富大贵，而是“面朝大海，春暖花开”。
<br/>
<br/>        她既不想当职场的女精英，也不想当商界的女强人，更不向往做什么豪门的贵太太。
<br/>
<br/>        她只想成为昼景一个人的贤妻良母。
<br/>
<br/>        季雅姗将自己那些压抑在心里的事情，向童筱颖全部托盘而出。
<br/>
<br/>        包括她的出身，她的家庭背景，她被十爷包养的前因后果，她全部讲给了童筱颖听。
<br/>
<br/>        而这些事情说完后，她忽然觉得自己豁然开朗，心里的包袱不再像以前那般沉重。
<br/>
<br/>        童筱颖非常耐心地听她说，同时，她也把自己的事情全部告知了季雅姗。
<br/>
<br/>        两个女人就这样彼此知根知底地凑在了一起，还莫名其妙地成为了闺中密友。
<br/>
<br/>        “原来，我们两个都是可怜的人。”季雅姗不禁感慨。
<br/>
<br/>        童筱颖却告诉季雅姗：“但我觉得我现在很幸福。”
<br/>
<br/>        “当然啊！你老公对你那么好！”季雅姗微微一笑。
<br/>
<br/>        虽然，童筱颖把自己的家世背景都说给了季雅姗听，但她却没告诉季雅姗自己救过一个身份悬殊的男人，更没告诉季雅姗自己的老公现在的经济状况。
<br/>
<br/>        季雅姗有问过她，为什么会在“夜惑”里卖唱，但她只说是自己的兴趣爱好，顺便还可以赚一些外快。
<br/>
<br/>        不过，从童筱颖的回答中，季雅姗忽然觉得自己比童筱颖要过得好。
<br/>
<br/>        至少包养她的男人是个大金主，她可以不再为钱的事情而发愁。
<br/>
<br/>        童筱颖却从未在这种事情上，和任何一个人去比较过，包括那个时候的苏漫雪。
<br/>
<br/>        两个人一起吃过甜点后，在店门口便道了别相背而行。
<br/>
<br/>        季雅姗走了几步，忽然停下了脚步，心思深沉地转过身去看了一眼童筱颖渐行渐远的背影。
<br/>
<br/>        其实，刚刚她去医院，是去买通鉴定科的医生的。
<br/>
<br/>        十爷想知道筱颖到底是不是婴宁的妹妹，不管鉴定结果如何，她都让那个出报告的医生写明被鉴定的两人是姐妹关系。
<br/>
<br/>        这样一来，筱颖成了十爷的亡妻的妹妹，十爷就不会对筱颖“下手”了吧！
<br/>
<br/>        季雅姗觉得这是为了筱颖好。
<br/>
<br/>        不知道阿十在得知筱颖是自己的小姨子后，会不会觉得很惊喜。
<br/>
<br/>        他终于完成了他妻子生前的遗愿，而筱颖，从此以后也有了亲人。
<br/>
<br/>        季雅姗希望自己这次做的事情，是一举两得。
<br/>
<br/>        童筱颖买了菜回来后，又将这小小的出租屋内打扫得干干净净，就连晚上的假她都请好了，还做好了晚饭，满心欢喜地等傅昼景回来。
<br/>
<br/>        傅昼景的电话还未打通，说明他还未下飞机，但童筱颖给他发了短信，相信他下飞机后一开机便能看到。
<br/>
<br/>        滨江城飞机场。
<br/>
<br/>        傅昼景和助理赵斌一起下的飞机，赵斌因为家里有事便先搭出租车离开了。
<br/>
<br/>        就在傅昼景准备拉开下一辆出租车的车门时，一只纤纤玉手突然抓住了他的手腕，阻止了他开门的动作，耳畔随即响起了一声柔柔的话语。
<br/>
<br/>        “阿景，我开车过来的！”季思妍微笑道。
<br/>
<br/>        傅昼景挪开了季思妍的手，淡漠地说了一句：“我自己坐出租车回去就好。”
<br/>
<br/>        “阿景，你这是在避讳我吗？怎么说，你现在也算是我师父啊！徒弟来接你，你就这么不给徒弟的面子吗？”季思妍落落大方地问。
<br/>
<br/>        出租车司机也忍不住落下车窗，不耐烦地催促了一句：“不走的话，麻烦让一让，别影响我载客。后面还有出租车等着。”
<br/>
<br/>        季思妍忙微微弯腰，冲着出租车司机，莞尔一笑：“抱歉啊！我们这就离开！”
<br/>
<br/>        她说完，淡定自若地拉住傅昼景的手，拽着他往自己的车子那边走去。
<br/>
<br/>        傅昼景将手抽了出来，季思妍也不再拽他，而是微笑着和他并排走在了一起。
<br/>
<br/>        “你不在的这段时间，我有认真完成工作噢！”季思妍没话找话地说道，认真的语气就像是一个乖巧的孩子。
<br/>
<br/>        傅昼景只是听着，却默不吭声。
<br/>
<br/>        而季思妍仍旧像高中时期那样，是个话痨，在平日里就很安静的他的耳边，可以不停地和她说任何话题。
