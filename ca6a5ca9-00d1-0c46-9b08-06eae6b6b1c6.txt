　　一周的时间一晃而过，整整七天内无论是南方边境还是中部基地，亦或是狼牙地界和埋骨之地，都风平浪静没有任何动静，好似这样的平静会一直持续下去，也让很多人在这样的平淡无波中忘了日子过的有多快。
<br/>　　白家难得五十名正统血脉的家族成员集体出动，他们来自不同分支中的佼佼者，被筛选过后跟随白忧大少一同前往阴阳谷。
<br/>　　这一传奇性的地域白家人当然知道它的历史，更清楚它的重要性，只是踏足并且还是跟着白忧一同前往，还是让他们心头忐忑。
<br/>　　这就是他们白家祖祖辈辈最向往却也最畏惧的地方？
<br/>　　与白家一样，罗家这次也大举出动，相对于白家的人数还有更多些，一共有八十名正统血脉成员，只是整体实力却不及白家的那五十人，罗家的成员大多数血脉不浓厚，只能堪堪到达正统血脉者的等级从而开启双基因锁。
<br/>　　高战力者寥寥无几不说，一堆嫡系成员之中，也只有罗杰的血脉看得过去。
<br/>　　只是在这八十几人之中，罗杰还带上了他所投资那座研究院中的一人，那就是楚云天。
<br/>　　胡梦浩原本是更合适的人选，毕竟年轻又是智力过人的门萨成员，只是可惜他双·腿残疾，坐在轮椅上行动不便，而阴阳谷毕竟是个地域巨大的地方，不是新人类甚至都没有那个脚程走完，更别说行动不便者。
<br/>　　于是罗杰只能退而求次，将各方面只比胡梦浩差一些的楚云天带上，虽然年纪大了些，也不是新人类，但至少是个能走路的。
<br/>　　想到这里的罗杰在半路上就有些暴躁异常，目光扫到混在在队伍中的楚云天，怎么看都怎么觉得不爽，就因为带了个普通人，他们的速度慢了很多，甚至还有很多地方需要注意。
<br/>　　罗杰是有一些其他心思，更不想被白家人知道研究院的情况，于是楚云天便被安排伪装成罗家成员一同进入。
<br/>　　想要那些罗家的神秘家族成员接受这一普通人与他们同等待遇，甚至朝夕相处中没有等级尊卑，这可不是一件容易的事，哪怕有着罗杰的强制命令，却依然让罗家人不适应和难受。
<br/>　　在至高的顶点站得久了便习惯性喜欢俯视，平等在他们心中本就是不存在的。
<br/>　　高白两家的两支队伍在预定的地点相遇，也就是在南沙港不远处的那一沙漠，这是阴谷的两座入口之一，相对于之前火族选择的那条入口，南沙港这处的漩涡虽然暴力，却因为地理优势的原因，让这批人的行动不被发现。
<br/>　　风沙会快速消磨脚印，更不会留下气温，哪怕前后脚的跟上，也不会被人轻易察觉。
<br/>　　“你们来晚了。”白忧看着罗家人出现，开头就是这么一句话。
<br/>　　罗杰随意笑了笑，道：“有个旁支在路上遇到一个女人，你懂得……”
<br/>　　白忧当即皱起眉满脸厌恶，转身看向入口处的漩涡：“谁先？”
<br/>　　“你决定好了，我无所谓。”罗杰还是那副无所谓的腔调。
<br/>　　白忧更加看不上此人，于是决定道：“我们先，你们过两分钟再进。”
<br/>　　“好。”罗杰比了一个‘请’的手势。
<br/>　　两分钟的前后脚，是一个不容易走散，但也能保证没有意外的时间，对于白忧的不信任，已经察觉到白家诡异的罗杰已经心中有底。
<br/>　　白忧并不耽搁，直接带着白家的五十名精英跳入阴谷入口。
<br/>　　哗——
<br/>　　一阵飞沙走石，疯狂旋转的黄沙漫天飞舞，像是搅拌机一样在那处深洞拼命拍打，发出阵阵刺耳的摩擦声。
<br/>　　罗杰站在入口边缘，嬉笑的目光为之一变，取而代之的是一股阴狠。
<br/>　　“杰少，我们现在进？”旁边的一名罗家人开口道，提议不遵循刚才与白家的约定。
<br/>　　罗杰摇头：“没必要打草惊蛇。”
<br/>　　“是。”那人点头，随后又看了眼身后：“那个楚云天不会被绞死吧？”
<br/>　　说起楚云天，罗杰便立即烦躁起来：“都包成粽子了还绞死那也没办法，罗家的那些防护服还是有点作用的，只希望此行这个老头别是拖累。”
<br/>　　“可以不带……”那名罗家人显然不认同罗杰的这一决定，小声嘀咕着。
<br/>　　罗杰冰冷的眼神扫过他，随口问出了另一问题：“狼牙那边安排好了吗？”
<br/>　　那人立即正色回答：“一切准备妥当，当我们都在阴阳谷内的时候，那帮人就已经开始行动了，放心吧杰少爷，这回楚涵必死无疑，白允儿估计杀不掉，但也会尽量让她重伤，我们的人会栽赃在白家身上，毕竟白忧这么一不在，白家剩下的那些人，可不会真心觉得一个混血杂种有资格当嫡系。”
<br/>　　“哼！”罗杰冷笑一声，阴寒道：“白家的传统真是恶心的够味。”
<br/>　　罗家成员不知如何接这句话，只有闭口不言。
<br/>　　此时的罗杰又忽然眼神一动，扭头笑道：“不过听说那个白允儿，国色天香啊？”
<br/>　　罗家成员思索了一下后回答：“能让白忧如此不顾及家族脸面，应该是绝色。”
<br/>　　“白家还有什么脸面？”罗杰笑出声，嘴角列开到最大：“他们家可是世世代代的兄妹结合，才保证了嫡系成员各个战力巅峰，不过每一代的嫡系数量都稀少，很多嫡系出生没多久就夭折。”
<br/>　　“这个家族就喜欢搞这种旁门左道。”罗家成员也不喜，皱起眉说道。
<br/>　　两人的谈话没有刻意掩盖，谈话内容在罗家人看来也并非什么秘密，他们却没有彻查楚云天的身份，以至于压根没有留意到队伍中有一人，会造成这次他们的阴阳谷之行不受控制。
<br/>　　楚云天穿着罗家人特地找来的防护服，包裹的臃肿站在人群里，他面色平静，就像一个极其普通的科研人员，除却自己的研究项目，对其他东西都不感兴趣的样子。
<br/>　　可就是这个看上去普普通通的老头，在一边平静听完两人的对话后，一边内心已经开始默默筹划起了什么……
