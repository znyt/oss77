　　“你们可有名字？”元始对着四人问道。
<br/>　　“吾等还没有名字，请老师决定！”
<br/>　　“没有，也好，那本座就给你们取个名字。”元始微微一笑，道。
<br/>　　“你是白鹤所化，丹砂作顶耀朝日，白玉为羽明衣裳。那你就叫白衣。”元始对着白衣男孩说道。
<br/>　　“多谢老师赐名！”白鹤，不，白衣对着元始躬身道谢。
<br/>　　“嗯，你为玄松所化，汝之名为玄青！”
<br/>　　“你为墨竹所化，幽篁一夜雪，疏影失青绿。莫被风吹散，玲珑碎空玉。既然如此，你为玉玲珑。”
<br/>　　“你为白莲所化，无情有恨何人觉，月晓风清欲堕时。你叫风晓月。”
<br/>　　之后，元始有对着玄松、墨竹、白莲说道，为他们取了名字。
<br/>　　“多谢老师赠名！”几人连忙感谢道。
<br/>　　“好了，不必多礼，嗯？”元始突然想到了什么，闭上双眼，一刻钟之后，睁开双眼。挥手之间，四道流光飞入四个孩童脑海之中。
<br/>　　“这是为师给你们的功法。”
<br/>　　几人连忙将意识探入识海之中查探，
<br/>　　白衣童子得到的是倾城剑诀，剑气既辉煌霸道，又缥缈如仙，为一门无双剑诀；
<br/>　　玄青得到的是不动青龙钟，是一门木属性的攻防一体的仙功；
<br/>　　玉玲珑得到的是天龙八音，为一门音道攻击仙法。
<br/>　　风晓月得到的是九天仙舞，九天舞，舞九天，九天仙舞舞九天，舞动之间，颠倒日月乾坤，鬼神皆惊。
<br/>　　“多谢老师赏赐！”
<br/>　　四人大喜，连声感谢元始，今日得到的惊喜实在是太多了，又是被老师收作弟子，又是赐名，又是赏赐功法，他们高兴得都快不知道自己姓什么了。
<br/>　　“好了，不必多礼，本座不喜欢那些礼节，随意就好。”
<br/>　　元始微微一笑，他绝不会承认，给他们的功法中有什么目的。两个男的还好，一个是剑诀，一个是攻防一体的功法。但是，那两个女的就不好说了，你看，不是跳舞，就是弹琴唱歌。你的目的到底纯呢，还是不纯呢？
<br/>　　要说这两门功法不好吧，它还绝对是顶尖仙法，绝对可以修炼到大罗金仙以上，而且威力巨大。但要说功法好吧，你怎么给他们的不是跳舞就是唱歌的，就没有别的了吗？
<br/>　　至于有没有别的目的，那就只有元始自己知道了。
<br/>　　“如今天地正值无量劫，你们就在这永恒岛上好好修炼，不可乱闯。要是你的两位师叔来了，就告诉他们我出去一趟，不日便会回来，叫他们回去闭关修炼，不可乱闯。”
<br/>　　元始打算出去将《玄黄不灭》提升上来再说，这段时间不再玉虚宫，故而特意向四个童子叮嘱道。
<br/>　　“是，老师！弟子遵命！”几人躬身告退。
<br/>　　看着四人退下后，元始出了永恒岛，朝着东海龙宫之处腾云而去。
<br/>　　东海龙宫，敖玉正在修炼之中，经过万年时间的修炼，再加上龙宫之中宝贝众多，当初他只是故意损伤龙珠，控制有分寸，所以现在已经好了差不多了，很快就能突破大罗金仙了，毕竟他作为龙祖第四子，血脉、天赋、资源，什么都不缺。
<br/>　　此时，他正在修炼，突然，内心深处接收到一种冥冥之中的召唤，让他不由自主地向着那召唤的方向走去。
<br/>　　良久，敖玉避开龙宫重重巡视，走出龙宫，向着大海之外飞去，一直飞到第一次遇见元始之处。
<br/>　　“啊？拜见尊主！”敖玉看见元始天尊站在虚空，屹立不动，顿时惊讶，倒头就拜。
<br/>　　“任务完成得如何？”元始面无表情道。
<br/>　　“启禀尊主，属下不负使命，已经将尊主所需的真龙精血及祖龙精血全部收集完毕。”
<br/>　　“哦？收集完毕？拿来！”元始淡淡道。
<br/>　　“是，尊主！”敖玉遵从元始天尊的命令，从识海深处取出一个绽放着琉璃光彩的玉净瓶。
<br/>　　元始伸手一招，玉净瓶就落入己手。神念微微一探，元始就知道这敖玉说的不错，他的确是收集完了自己所需的真龙精血。
<br/>　　“不错，你的确做到了，说吧，要本座怎么奖赏你？”元始收起玉净瓶，对着敖玉说道。
<br/>　　“为尊主效命，万死不辞！”敖玉双手抱拳，单膝跪地，低头道。
<br/>　　“嗯？本作看你本源龙珠受过伤害，想必是收集精血也不容易吧，也罢，看在你如此听话的份上，本座就赏赐你一场机缘，助你突破。”元始淡声道。
<br/>　　“喏，这是一颗黄中李果子，你拿回去将之炼化，不仅能够修复本源，还能顺利突破大罗金仙。”元始拿出一颗黄中李果子扔给敖玉。
<br/>　　敖玉能在龙族之中起的作用还是很大的，龙族马上就会发生大战，要是前世记忆没有出错的话，最后龙族等可是都会都会退出洪荒舞台啊。
<br/>　　而，敖玉作为龙祖的第四个儿子，将来有很大机会可以执掌龙族，这对于未来的自己非常有利，他还不想敖玉现在就将潜力耗尽。故而，元始给他一颗黄中李，不算太亏。
<br/>　　“多谢尊主赏赐！”敖玉感谢道。
<br/>　　“你龙族可有龙墓？”元始问道。
<br/>　　自己想要突破《玄黄不灭》第四重龙脉融龙骨，需要龙气充裕之地，没有什么地方比龙族的龙墓更好了。龙墓之中，葬有众多龙族高手，不仅会有这些龙族高手身上留下的龙源，散发成浓郁的龙气，还会有龙珠、龙族传承等。
<br/>　　“启禀尊主，龙族有龙墓，龙墓被藏在混乱虚空之中。”敖玉禀告道。
<br/>　　“哦？还真有，混乱虚空吗？不错，是个好地方，空间混乱，一不小心就会迷失在虚空之中。你将龙墓的位置报出来，再将你搜集到的龙族功法交出来，你就回去吧。”元始淡淡的声音响起。
<br/>　　“是，尊主！”接着，敖玉将龙墓的位置记载于一个玉简之中，再将这些年收集到的功法记载在玉简之中，给了元始。
<br/>　　元始心念一动，瞬间便接收到无数信息，在这些信息之中，大多是龙族的功法，还有一个地址，记载了龙墓所在之地。
<br/>　　几个呼吸的时间，元始便全部记住这些信息。当然，记住并不代表学会，毕竟种族不同，功法也不相同，想要完全参悟透，还需要多费些时间。
<br/>　　咔擦！
<br/>　　记住之后，元始便毁了玉简，对着敖玉道：
<br/>　　“你先回去吧，先将修为突破，之后争取掌控更多龙族势力，最终的龙凤大战，你就不要去掺和了。”
<br/>　　“是，尊主！属下告退。”
<br/>　　敖玉抱拳躬身告退，之后消失在天际。
<br/>　　看着敖玉消失后，元始也一步踏出，消失在原地，却是向着龙墓方向的混乱虚空飞去。
