　　“噗！”
<br/>　　血无影被叶尘一脚踏中，顿时狂喷了一口鲜血，身体下面顿时出现了一个凹坑，整个人完全陷入地下。
<br/>　　嘶~~
<br/>　　周围所有人全都倒吸了一口凉气，甚至都怀疑自己出现了幻觉。
<br/>　　巨门星域的第一天才，甚至动用血狱神碑的血公子，竟然被一个金丹境的小子一招击败，而且被他犹如死狗一般踩在脚下！
<br/>　　这个世界是怎么了？
<br/>　　简直太疯狂了！
<br/>　　“你、你到底是什么人？”
<br/>　　血无影瞪大着双眼，眼眸中透着浓浓的惊恐之色，不可思议的望着叶尘。
<br/>　　叶尘嘴角微微一扬，
<br/>　　“这个问题，你还是去地下问阎王爷去吧！”
<br/>　　说着，叶尘脚下微微用力。
<br/>　　“噗！！”
<br/>　　血无影再次狂喷了一口鲜血，继而强大的求生欲，让他强忍着五脏六腑的痛苦，大吼了起来，
<br/>　　“不！你不能杀我！我可是血月宗的少宗主！你杀了我，血月宗是绝对不会……啊！！”
<br/>　　还没等血无影威胁的话说完，叶尘早已经懒得去听，脚下猛然加大力道，
<br/>　　砰！
<br/>　　可怜这位巨门星域第一大天才，凶名赫赫的血公子，竟然被叶尘一脚生生踩爆！
<br/>　　“啊！！”
<br/>　　周围众人顿时个个脸色大变，
<br/>　　“血公子……就这么死了？”
<br/>　　“那可是血月宗的少宗主，竟然就这么被他给杀了！”
<br/>　　“难道他就一点也不担心血月宗的报复吗？”
<br/>　　“太彪悍了！”
<br/>　　……
<br/>　　众人议论纷纷，那道瘦削的少年身影，在他们的眼中，简直比恶魔还要可怕！
<br/>　　然而，叶尘却犹如做了一件微不足道的消失，随手一招，将血无影的血狱神碑收入掌中，然后抹掉上面的精神烙印，仍进了自己的乾坤镯之中。
<br/>　　这血狱神碑虽然是邪物，可毕竟也是一件准仙器，丢了就太可惜了。
<br/>　　做完这一切之后，叶尘方才转过身，缓步走到，早已经看傻眼的灵氏姐妹跟前。
<br/>　　“多谢前辈救命之恩！”
<br/>　　灵韵反应比较快，当即冲着叶尘躬身行礼，连称呼也变成了前辈，而且美眸之中，透着一抹浓浓的崇拜之色。
<br/>　　虽然叶尘的实力好像只要金丹境，可是在修真界，达者为尊！叶尘轻松斩杀了血无影，实力堪比化神天君，这一声前辈，自然也受之无愧。
<br/>　　更何况，叶尘杀了血无影，等于救了她的性命，灵韵是发自内心的敬佩和感激。
<br/>　　然而，
<br/>　　叶尘却连看都没有看灵韵一眼，伸手揉了揉灵溪的小脑袋，微微一笑，道：
<br/>　　“丫头，你不是想成为一名炼丹师吗？从今天开始，叶尘哥哥亲自来教你，好不好？”
<br/>　　灵溪顿时两眼一亮，
<br/>　　“叶尘哥哥你……是炼丹师？可是，我的身体，如果没有破灵丹的话，是没办法修炼的……”
<br/>　　叶尘哈哈一笑，
<br/>　　“区区四阶丹药而已，叶尘哥哥随便炼个十瓶八瓶，给你当糖豆吃都没问题！”
<br/>　　如果叶尘之前说出这种话，肯定没有人会相信，可是如今今非昔比，自然就大不相同了。
<br/>　　一旁的灵韵，在暗暗惊愕的同时，心里更是郁闷不已。
<br/>　　轻松斩杀了血公子，还至少是一位四品以上的炼丹师，关键又如此的年轻，这简直就是天才中的天才啊！
<br/>　　相比之下，就连她一向崇拜的柳云飞，也变得黯然无光了。
<br/>　　如果早知道这个少年，是如此的厉害，她之前肯定不会那般态度。
<br/>　　而如今，叶尘对灵溪温柔体贴，却对她视而不见，这让灵韵无比郁闷的同时，心里竟微微有些嫉妒。
<br/>　　就在这时，一旁的柳云飞也走了过来，一脸殷勤道：
<br/>　　“灵韵师妹，你没事吧？”
<br/>　　灵韵正一口闷气无处发泄，再联想起之前，柳云飞为了自保，将自己无情抛弃的场景，不由得俏脸一寒，
<br/>　　“不敢有劳柳师兄费心！”
<br/>　　柳云飞自然知道，灵韵为何生气，脸上不由得露出一抹尴尬之色，轻咳了两声，方才又道：
<br/>　　“既然没事的话，那我们这就一起回宗门复命去吧！”
<br/>　　柳云飞说着，便要去拉灵韵的胳膊，后者却微微一侧身，果断躲开一旁，
<br/>　　“不用了！我还要在黑岩城多待两日，柳师兄和于师弟，就请先回去吧！”
<br/>　　柳云飞被灵韵两次回绝，手臂悬在空中，顿时越发的尴尬，不由得有些恼羞成怒，
<br/>　　“灵韵师妹，我必须提醒你一句，这小……这位叶道友，虽然杀了血无影，血月宗的人岂会善罢甘休？而且你别忘了，此事可是因你而起！”
<br/>　　“你若是现在随我回宗门，我可以亲自为你向掌门师尊求情，飞雪宗必然可以庇护于你，你若是继续留在这里，等到血月宗的大军一到，哼哼！”
<br/>　　柳云飞这话一出口，不但灵韵脸色大变，其余黑岩宗的众人，也全都个个面如土灰。
<br/>　　是呀！
<br/>　　那可是血月宗！
<br/>　　巨门星域的第一大宗门！
<br/>　　如今血公子死在这里，血月宗必然会为他报仇。
<br/>　　看到众人的反应，柳云飞的脸上顿时露出一抹得意之色，感觉之前被叶尘压着的一口气，也终于可以一吐为快，望向叶尘的目光，也没有了方才的敬畏，
<br/>　　“叶道友，并非我看不起你，你就算再厉害，也毕竟只有一个人！”
<br/>　　“更何况，那血月宗可是有三位化神境的长老坐镇，你觉得凭你一人之力，能够对抗巨门星域第一大宗门吗？”
<br/>　　“如果我是你的话，还是赶紧逃命要紧！”
<br/>　　听到柳云飞这话，叶尘还没开口，灵山和黑岩城众高层，顿时全都急了，甚至有人忍不住高呼起来，
<br/>　　“不行！你不能走！你要是走了，血月宗的人杀来怎么办？”
<br/>　　“不错！人是你杀的，你要为此事负责！”
<br/>　　“对！不能让他走！”
<br/>　　……
<br/>　　眼看着众人越吵越凶。
<br/>　　“哼！”
<br/>　　叶尘直接冷哼一声，附带着一道精神攻击。
<br/>　　轰隆隆！
<br/>　　众人顿时感觉犹如被一道神雷劈中，脑袋轰轰作响，瞬间便安静了下来，
<br/>　　“我若要走，你们拦得住吗？”
<br/>　　黑岩城众人顿时惊呆当场，这才陡然意识到，
<br/>　　眼前这位，那可是连血公子能说杀就杀的猛人啊！
<br/>　　一时间，所有人噤若寒蝉，全部低下头，不敢再吭声一句。
