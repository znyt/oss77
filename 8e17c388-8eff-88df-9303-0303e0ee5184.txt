　　这些宝物，简直是充满了如今这大门背后的密室。
<br/>　　“怎么会有如此之多的宝物！”
<br/>　　白萱看了过去，眼神中都是极为贪婪，想要将这其中的宝物，全部尽收其中。
<br/>　　这些宝物，要是被一个人得到，那个人绝对会一跃之间，成为整个北域的大富豪。
<br/>　　不过面对着如此之多的宝物，在前方的无数武者，却是没有任何一人敢动。
<br/>　　无人敢越过雷池一步。
<br/>　　因为在那大门之上，却是有着一道巨大无比的禁制。
<br/>　　再次之前的几尊想要闯入其中的武者，都是一下子撞在了这禁制之上，当场陨落。
<br/>　　“那是？我们北邙学院的种子弟子落长恨！”
<br/>　　白萱扫视人群，突然看见其中一尊长相阴冷的男子。
<br/>　　这男子的长相，本就极为阴冷，他的脸上，更是仿佛随时随地充满着无数恨意一样，好像是他的心，都被恨意充满。
<br/>　　“这是？长恨之道！恨天恨地憎恨任何人的长恨之道！”
<br/>　　楚风眠顺着白萱，也是一眼看出了那落长恨。
<br/>　　毕竟这落长恨身上的气息，简直是太过明显，滔天恨意，一眼就可以认得出来。
<br/>　　这长恨之道，也是极为强大的一方武道，但是修炼的人，却是少之又少。
<br/>　　人的恨意，可以激起人的潜力，这长恨之道，便是憎恨一切，来获取力量。
<br/>　　虽然强大无比，但是寻常武者，根本不可能会有着如此强大的恨意，所以很难修炼。
<br/>　　只有一些经历过无数灾祸，才会有可能修炼这长恨之道。
<br/>　　曾经远古时代之中，便是有着一尊无上强者，名为恨天神王，此人便是少年家破人亡，一生中灾祸无数，最后憎恨一切，修成了长恨之道。
<br/>　　眼前这落长恨，虽然远远不能跟恨天神王这等人物媲美，但是却也是将长恨之道，修炼到了大成。
<br/>　　在他的身边，都无人敢靠近，因为一旦靠近了他，都有可能会被长恨之道所影响，引出身体中的恨意来，直接被恨意迷失自我。
<br/>　　“恩？”
<br/>　　楚风眠看向那落长恨的一刻，这落长恨似乎也察觉到了楚风眠的存在，一眼忘了过去。
<br/>　　楚风眠都感觉到，他的内心，似乎都变的愤怒无比。
<br/>　　“此人厉害！他的长恨之道，已经是足以影响我，不过他还只有神力境九重的境界，没有踏入生死境，不然他的长恨之道对我有着极大的威胁。”
<br/>　　楚风眠心中暗自想到，他催动力量，将心中的恨意一扫而空，眼神也变得平静起来。
<br/>　　“恩？”
<br/>　　看着楚风眠没有受到任何影响，落长恨的表情都微微有着一丝镇压，不过却并没有在关注楚风眠，而是将所有视线，放在了那大门的禁制上。
<br/>　　“这落长恨对你出手了？”
<br/>　　楚风眠身上的变化，瞒不过白萱的眼睛，她的脸上都是露出了几分忌惮之色道。
<br/>　　“这落长恨，虽然是北邙学院的弟子，但是却根本不值得信任，不能预知为友。”
<br/>　　“他修炼的长恨之道，憎恨一切，就算是北邙学院，他的憎恨，任何人他的憎恨，你可要小心一点。”
<br/>　　楚风眠点了点头，眼神中露出几分杀意说道。
<br/>　　“不错，修炼长恨之道，此人根本就是一个疯子，不过他之前想要阴我，等我找一个机会，定要将他斩杀！”
<br/>　　落长恨刚刚的出手，像是试探，但是实际上他也是带有着几分杀心的。
<br/>　　一旦是楚风眠实力不足，被他的恨意影响到，只怕是整个人都会废了。
<br/>　　敢对楚风眠出手，这落长恨在楚风眠的心中，已经是一个必杀之人。
<br/>　　“除了落长恨，北域的无数天才，居然都来了这么多！”
<br/>　　白萱扫视这人群，看着其中一道道熟悉的身影，惊叹道。
<br/>　　“这是武道世家沧澜家的沧澜霸，修炼的是纯真武道，他的实力跟落长恨相比，都不落下风！”
<br/>　　楚风眠看过此，这沧澜霸便是一尊纯正武者，一身武道精粹极致，似乎是要将武之一字，达到巅峰的任务。
<br/>　　“这是隐秘势力天棺宗的天才，枯朽天！他所修炼的极为诡异的腐朽之道！极为难缠，实力极强！”
<br/>　　枯朽天则是一袭黑衣，神秘无比，身上灵力极为诡异，周围的无数灵气，在他的身边，居然都开始枯萎腐朽消失。
<br/>　　隐秘势力，在这北域之中，除了北邙学院，地魔门这两大庞然大物，无数武道世家之外，还有着一些隐秘势力。
<br/>　　这些隐秘势力，向来不露山水，门下弟子极少，但是每一个人都是精英，跟武道家族极为相似。
<br/>　　不过武道家族都是依靠的是他们的血脉，而这些隐秘势力，都是各自的一些独门秘术，其中许多隐秘势力出身的弟子，都是极为诡异。
<br/>　　“还有地魔门的鬼谷圣子！战魔尊者这二人！”
<br/>　　这鬼谷圣子，战魔尊者二人楚风眠曾经便是见过一面，鬼谷圣子看起来根本就像是一尊七八岁的孩童一样。
<br/>　　但是实际上，他却是修炼无上魔功，才一直保持了童身，鬼谷圣子心狠手辣，据说喜欢吞食许多年轻女子的精血。
<br/>　　不是为了修炼，单单是吞食为乐，在北域都是臭名昭著。
<br/>　　“就连西域的终焉王朝的天才，都来了！”
<br/>　　白萱一个个看着，满脸惊叹。
<br/>　　这些人，许许多多平日里都根本不会在北域中走动出现的，如今看来这颜古龙帝洞府，居然是将这些人全部引了过来。
<br/>　　这颜古龙帝的洞府开启，简直就是一场天才之间的集会。
<br/>　　“没有生死境高阶的强者。”
<br/>　　楚风眠扫视了周围一圈，心中倒是安定了许多。
<br/>　　生死境高阶的强者，对楚风眠才有着致命性的威胁。
<br/>　　这些天才，虽然难以对付，却也不是无法对付。
<br/>　　这大门背后密室中的无数宝物，楚风眠一个人独吞虽然不能，但是从中抢夺几件，楚风眠还是足以做得到的。
<br/>　　只是，眼前这禁制，却是将所有人，都隔绝与无数宝藏之外。
<br/>　　在众人之中，最为强大的几尊武者，似乎正在交谈，突然间他们好像是谈拢了一样，分散了开来，无数灵力，开始凝聚。
