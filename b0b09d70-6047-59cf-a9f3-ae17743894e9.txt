　　任云都带着众人，在圣皇城中穿梭，没过多久，便来到了一座恢弘的建筑之前。
<br/>　　这座建筑，如同是一座巨大的城堡，占地方圆十数里，气势恢宏。
<br/>　　城堡的大门上方，金色的匾额之上，有着两个古朴的大字——兵府。
<br/>　　兵府大门之前，伫立着数十名穿着铠甲的兵士，个个手持战刀，气息强大。
<br/>　　这些士兵，赫然全部是武皇境的修为，这让苏莫暗暗心惊，连守门的士兵都是武皇境修为，当真是厉害。
<br/>　　来到兵府门前，任云都让众人在外等待，他一个人进入了兵府内。
<br/>　　“谢彬，这兵府是何地？”苏莫一边等待，一边向谢彬问道。
<br/>　　“苏兄，这兵府是圣朝调度兵马的地方，不过，只能调度圣朝一部分兵马！”谢彬解释道。
<br/>　　“原来如此！”苏莫点了点头。
<br/>　　“只不过，这掌控兵府之人，是圣朝祝家之人，乃是三皇子的人，和圣子殿下向来不对付！”谢彬说道。
<br/>　　“圣朝有几位皇子？”苏莫问道，他已经知晓古天意是圣朝八皇子。
<br/>　　“圣朝有十几位皇子，但天赋比较高的皇子，将来有望继承大统的只有三人，分别是大皇子、三皇子和圣子殿下！”谢彬道。
<br/>　　苏莫点了点头，心中有些明了，看来古天意也是对于圣皇大位有所垂涎，不然不会如此收拢人才。
<br/>　　“大皇子和三皇子，都是什么修为的武者？”苏莫又问道。
<br/>　　“大皇子和三皇子修为较高，一个是武尊境六重，一个是武尊境五重，在圣朝势力极大，相比较他们二人，圣子殿下由于年岁较轻，只有武尊境三重的修为，而在圣朝的势力也比较小！”
<br/>　　谢彬叹了徐徐说道，叹了口气，继续道：“不过，相比较天赋才情，还是我们的圣子殿下，更高一筹！”
<br/>　　苏莫默然的点了点头，心神沉思，这种夺嫡之争最是残酷，自己将来不会也卷进其中吧！
<br/>　　不过，沉思了片刻，他便不再多想，三十年时间转瞬即过，等到他在神武学府待满了三十年，修为估计早已达到武尊之境。
<br/>　　到那时天大地大，随处可去。
<br/>　　而且，到了那个时候，他就要离开古灵星河，前去明虚星河寻找青璇了，可不会留在皇祖圣朝，帮助古天意争夺什么圣皇大位。
<br/>　　众人等到了片刻之后，任云都走出了兵府。
<br/>　　“走，我们去黑山要塞！”任云都朗声说道。
<br/>　　随即，一众人立刻出发，离开了圣皇城，一路向西而行。
<br/>　　一路上，苏莫和谢彬聊了很久，对于赤粱星的格局，已经心中有数。
<br/>　　谢彬来赤粱星很多次了，大小战役都参加过，对于赤粱星的情况了如指掌。
<br/>　　赤粱星，皇祖圣朝是最为强大的势力，除了皇祖圣朝之外，赤粱星上还有四大古国，三大异族。
<br/>　　这四大古国和三大异族，任何一个都不是皇祖圣朝的对手，以前不敢与皇祖圣朝为敌。
<br/>　　但近百年来，四大古国和三大异族，居然联合了起来，不断的侵蚀皇祖圣朝的地盘。
<br/>　　可以说，若不是皇祖圣朝的圣皇，实力强大，威震整个赤粱星，这些古国和异族早就大举来犯了。
<br/>　　而黑山要塞，是皇祖圣朝和陀罗古国、血瞳族的三方交界之地。
<br/>　　此地非常的重要，周围是大地龙脉灵气勃发之地，有很多灵晶和灵石矿脉。
<br/>　　所以，陀罗国的高手以及血瞳族的高手，经常袭击此地，掠劫资源。
<br/>　　苏莫等人一路疾行，足足飞行了八日，于中午时分，终于来到了黑山要塞。
<br/>　　只见一片黑色的山脉延绵无际，方圆不知多少万里，仿佛一条巨大的黑龙盘旋与此。
<br/>　　而在山脉的边缘位置，屹立着一座巨大的要塞，将黑色的山脉守护在后方。
<br/>　　此要塞占地方圆数百里，形如城池，四方的城墙足有数百丈高，最主要的是，要塞的上空光芒闪烁，布置着强大的防御阵法。
<br/>　　嗖嗖嗖！！
<br/>　　苏莫等人纷纷身形降落，落在了黑山要塞的大门之前。
<br/>　　“我等是神武学府弟子，奉八皇子殿下之命，前来要塞职守，请速速打开阵法！”任云都望着要塞的城楼之上，朗声高喝。
<br/>　　要塞的城楼之上，密布着一名名身穿黑甲的士兵，个个面色冷漠的望着苏莫等人。
<br/>　　“没有祝隆将军的命令，任何人不能打开要塞！”冷漠的声音从城楼上传出。
<br/>　　“那快去通报！”任云都皱眉喝道，他没想到此地的兵马，居然也是祝家之人统领。
<br/>　　祝家是三皇子的人，估计这次不会给他们安排好的差事了。
<br/>　　随即，众人便在城门口等待了起来。
<br/>　　但是，这一等，就从中午等到了傍晚。
<br/>　　“怎么回事？”
<br/>　　“为什么还不让我们进去！”
<br/>　　“玛德，这是耍我们，若是陀罗古国和血瞳族的人杀来，那就麻烦了！”
<br/>　　众人都等急了，个个脸色难看，通报一下居然要这么久，这不是耍他们是什么？
<br/>　　若是在此刻，陀罗国的高手和血瞳族的高手杀来，那他们就危险了。
<br/>　　苏莫的脸色也是沉了下来，本以为这次历练，能轻轻松松再捞一笔资源，没想到这皇祖圣朝，也是一滩浑水。
<br/>　　“怎么回事，为何还不打开要塞？”任云都望着城楼之上，脸色难看的喝问道。
<br/>　　“没有祝隆将军的命令，任何人都不得打开要塞！”冰冷的话语，再次从城楼上传了下来。
<br/>　　“可恶！”
<br/>　　任云都气的咬牙切齿，随即朗声怒喝道：“通报一次能要大半天吗？你们如此做法，分明就是打击异己，我们现在就离开，将此事回禀八皇子殿下！”
<br/>　　任云都真的怒了，以往三皇子和八皇子，虽然明争暗斗，但在这种国家大事之上，还不会如此明目张胆的打击异己。
<br/>　　但现在，这个祝隆将军分明就是在给他们脸色。
<br/>　　所以，他也不准备再执行这次任务了，准备带着众人返回神武学府。
<br/>　　“呵呵！诸位何必心急呢！”
<br/>　　就在此刻，一声轻笑在城楼上响起，只见城楼之上出现了一名年轻的将领。
<br/>　　此人面目英挺，棱角分明，身穿一身银色的甲谓，显得威风凛凛，气度非凡。
