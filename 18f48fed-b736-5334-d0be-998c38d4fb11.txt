<br/>    万道天竹在手，陈明扫了一眼自己的面板，危险指数可谓是狂飙，直接涨了200，达到了840！
<br/>    
<br/>        还没有达到大妖王的地步，大妖王的危险指数，应该是破千的。
<br/>    
<br/>        自己得到血海魔刀的时候，可是没有这么大的增长，而万道天竹，可是给自己直接涨了200的危险指数，果然，最适合自己的才是最好的。
<br/>    
<br/>        只是现在自己这身扮相，有点问题啊，还得弄一身道袍，不然有可能被误会成太监.....
<br/>    
<br/>        一头长发也不能像这样随意披在身后了，还得弄个道簪。
<br/>    
<br/>        反正最近准备炼器，就给自己搞一身法宝吧。
<br/>    
<br/>        仙器都会炼制了，寻常的灵兵自然不在话下，陈明回到燕山，在燕山仓库里找到了青狮的皮，陈明准备直接用青狮的皮炼成一件青色道袍，挥手一道火焰打出，陈明开始炼制青狮的皮毛，将所有钢针般的狮毛全部炼化，变成一团青色的液体，流光溢彩，陈明将青狮的皮修修剪剪，终于是炼成了一件青色道袍的模样。
<br/>    
<br/>        别说，炼器师还真是自带裁缝技能！
<br/>    
<br/>        你以为这么简单就已经炼制成一件灵兵了吗？
<br/>    
<br/>        开玩笑，这不是扯淡吗？
<br/>    
<br/>        要是炼器这么简单的话，不是谁都会了，并不是所有的炼器都像是万道天竹那么简单，万道天竹那是浑然天成的仙器雏形。
<br/>    
<br/>        将青色道袍放在自己的身前，陈明开始在上面绘刻符印，到底绘刻什么符印好呢？
<br/>    
<br/>        这一件青色道袍乃是青狮的皮炼制而成，能够承受七个符印，符印也分很多种，比如兵器上经常绘制的锋锐，便是可以让灵兵变得更加锋利，兵器一般都需要不少坚韧，不然太容易折断，所以坚韧也需要绘制不少。
<br/>    
<br/>        至于像是道袍这种护身灵甲，倒是不需要锋锐这种东西，难不成你还想用道袍去砍了别人的脖子，等等，好像也不错的样子啊，其他人绝对猜不到我这道袍还能用来砍人！
<br/>    
<br/>        不过要是打起来，还得脱衣服，画美不看，算了算了。
<br/>    
<br/>        琢磨了一番，陈明最后毅然决然决定在上面绘制七个山岳符印，这样一来，就可以足够坚韧，不至于被破坏，可以有效防止穿透性伤害，震动性伤害陈明倒不是很怕，毕竟九九玄功还是有三十六层的。
<br/>    
<br/>        山岳符印，灌入法力激发之后，可以有效的阻挡敌人的进攻，如同有着土墙隐藏在道袍之中。
<br/>    
<br/>        七个符印落下，陈明一气呵成，正准备穿在身上，愕然发现，全是山岳符印，根本硬的穿不了，怪不得铠甲的防御力比灵衣要高。
<br/>    
<br/>        看来自己的思路有点问题，仔细想了一番自己会绘制的符印，最后想了想，还是绘制七个弱水符印吧。
<br/>    
<br/>        以柔克刚，道袍之中有着弱水符印，敌人的攻击打来，会有水流护体，这么一想，陈明想起了曾经得到的那一刻避水珠，那可是水系宝物，不如也炼制在这一件道袍之中。
<br/>    
<br/>        说干就干，陈明找到了避水珠，将避水珠研磨成粉，然后炼化在道袍之中，然后将七个弱水符印绘刻其上，花了半天的时间，陈明终于是将弱水符印绘制完毕，穿在身上，轻若无物，纯青色的道袍，穿上之后一改自己之前邪魅的形象，变得仙风道骨。
<br/>    
<br/>        可是头发，自己不会弄啊。
<br/>    
<br/>        找卓青瑶吧，女孩子 对于头发，可是擅长之极。至于为什么不找陈灵鱼，这种事情，当然是找更熟的人了。
<br/>    
<br/>        要是找陈灵鱼，陈明可以保证，不到三天，就会传遍燕山自己不会梳头。卓青瑶就没有这个担忧了，她一般都和记名弟子玩不到一起去。
<br/>    
<br/>        大师姐威严如山。
<br/>    
<br/>        嗯，冰山。
<br/>    
<br/>        来到卓青瑶的剑山，轻车熟路走进卓青瑶的院子，院子里并没有人，陈明自然而然的推开门，卓青瑶见到陈明来到此处，手忙脚乱，连忙将一个长布包藏在被子下面。
<br/>    
<br/>        嗯，这小妮子藏什么东西呢。
<br/>    
<br/>        陈明仔细看了一眼，三尺长，用布条装着，到底是啥，“青瑶，你藏什么呢？”
<br/>    
<br/>        卓青瑶面色微红，“没，没什么。”
<br/>    
<br/>        陈明笑道：“给为师看看，你到底藏什么呢？”
<br/>    
<br/>        卓青瑶愣了三息，然后道：“师父，你这样闯进一个女孩子的闺房真的好吗？”

<br/>        陈明切了一声，“屁大点丫头，还女孩子。”

<br/>        卓青瑶一挺胸脯，十四岁的卓青瑶已经开始发育，小胸脯开始微微鼓起。

<br/>        陈明忽然意识到，好像，是应该当做女孩子了，一年多不见，女大十八变，小胸脯开始隆起，小屁股也开始挺翘，以后确实不能随便进这些小丫头的闺房了。

<br/>        陈明道：“那个啥，你帮为师梳梳头。”

<br/>        卓青瑶只是扫了陈明现在的穿着，一身青色的道袍，双手在胸前交叉，握着一根拂尘，大致就估摸出了陈明的意思。
<br/>    
<br/>        让陈明在卓青瑶的梳妆台前坐好，不过一刻钟，陈明的头发，就被卓青瑶打理地整整齐齐，服服帖帖，毕竟是女孩与生俱来的技能。

<br/>        再看自己，头顶白玉冠，一身青色的道袍，可谓是仙风道骨，恍若谪仙人。
<br/>    
<br/>        指点了一番卓青瑶，陈明离去。
<br/>    
<br/>        卓青瑶连忙回到自己的闺房，找到那个长布条，摊开，是还有一丝凝固血迹的竹剑。卓青瑶在房间里扫了一圈，最后还是决定收在床底下。“嗯，这样，师父应该就不会发现了。”
<br/>    
<br/>        陈明回到燕山，开始决定写一本《炼器八步与符印详解》，一本《炼器纲要》。按照陈明的思路，燕山现在有三千六百名弟子，三千名弟子炼丹，丹药毕竟是消耗性物品，需要的多，六百名弟子炼器，炼器之后，只需要修复，不需要那么多记名弟子。

<br/>        炼器，最为难的就是掌握符印，符印众多，很难全部都掌握，需要大量的时间，所以陈明决定让一个弟子掌握一个符印，然后由最后一个弟子将所有符印贯通。
<br/>    
<br/>        和炼丹一个道理，就是分工，每个弟子都只掌握一个步骤，再笨也学的会！
<br/>    
<br/>        写了出来，修修改改又是几天，半个月之后，陈明终于是写完了这两本巨作，然后传给了那不在系统承认范围内的记名弟子。
