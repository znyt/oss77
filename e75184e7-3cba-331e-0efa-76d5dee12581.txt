　　丧尸王的逃跑只获得了楚涵一个余光的追随，就连旁边的尚九娣和陆玟丞，都没有对这个楚涵的手下败将有任何关注。
<br/>　　所有人，此时此刻都紧盯着白晟！
<br/>　　坐在轮椅上的中年人还是那副神情淡然的态度，好似根本没注意到丧尸王的离开，闲逛一圈后就再次回到原先的地点停下，与楚涵遥遥相对。
<br/>　　楚涵握住修罗战斧的双手越来越紧，敌不动，我不动。
<br/>　　现场气氛相当冷凝，大战一触即发！
<br/>　　尚九娣的心慌再起，她已经无法承受再一次的突然崩溃。
<br/>　　远处没有停止瞄准的陆玟丞副官，守在时光炮弹的旁边汗如雨下。
<br/>　　从极高的视野观看了整场大战，到最后白晟出现的叶永阳，当下更是紧张万分。
<br/>　　千年了，这就是千年后白家的当代家主！
<br/>　　那个曾经野心磅礴，将阴阳谷搅得天翻地覆白家！
<br/>　　他们逼得回炉守护者消失，让水火两族被迫远离阴阳制衡之地，更让四偏门来到现实社会与人类世界共存。
<br/>　　所有的一切，都是白家为主谋！
<br/>　　没有人敢在这个时候出声，连呼吸都小心翼翼，他们不清楚白晟的目的，一切都充斥着危险。
<br/>　　最主要也最关键的是……
<br/>　　制裁一门的当代制裁者此时此刻不在这里，何峰远在埋骨之地！
<br/>　　而上代制裁者何培源……他死了！
<br/>　　没有人能阻止重现天日的白家家主白晟，他想做任何事都有着绝对的时间。
<br/>　　情报部部·长陆玟丞是第一个反应过来的人，一想到何培源已死的事情还未汇报给楚涵，他就瞬间心慌无比，生怕楚涵仗着制裁者的存在，率先对白晟动手。
<br/>　　神秘家族至今不敢太过于放肆，也是明白除了上代制裁者何培源，还有着当代制裁者的存在。
<br/>　　而这个当代的制裁者身份，也很好的隐藏了下来，唯有与楚涵关系不一般的高少辉一人知晓！
<br/>　　陆玟丞不敢打草惊蛇，唯有赌。
<br/>　　就赌神秘家族成员对制裁者的恐惧！
<br/>　　只是让所有人都没有想到的是，白晟就在这紧张万分的时刻幽幽开口，语态平和：“你应该知道我这双·腿是怎么断的吧？”
<br/>　　楚涵又一次瞳孔微缩，在白晟出现后第一次回应：“这是制裁者留下的荣耀。”
<br/>　　“哈哈！”白晟忽然愉快的大笑，眼神光彩夺目：“你果然聪明，一句话仅仅十个字，处处是精髓。”
<br/>　　楚涵抿嘴不置可否。
<br/>　　他说出了制裁者，目的就是为了让眼前这个白晟掂量掂量，让他明白一旦出手的后果。
<br/>　　而用上了荣耀二字，更是一种反压制的强势态度，告诉白晟你的确强大，但也别想在人类的世界里嚣张。
<br/>　　但楚涵没想到的是，白晟听到这句话不怒反笑，甚至看向他的眼神还带着淡淡的欣赏。
<br/>　　“我就喜欢你这种是非分明毫不做作的后辈。”白晟毫不掩饰的夸赞，让楚涵皱起眉，也让周围的其他人神色莫名起来。
<br/>　　这时候的白晟长叹一口气，又道：“因为这是我最希望在我儿子身上看到的品质，但是可惜，他不仅没有丝毫这方面的气质，反而还越走越偏。”
<br/>　　白忧？
<br/>　　楚涵眼神闪了闪，他当然知道白忧这个大名鼎鼎的白家大少，但却对此人的气质作风了解不多，只知道狼牙的诸多险境都是他一手策划。
<br/>　　“我就只有一个儿子，我希望他敢作敢当有勇有谋。”白晟又道：“可惜他却偏偏敢做不当，有谋无勇，喜欢暗算，擅长诡计。”
<br/>　　楚涵毫不客气的当场翻了个白眼，打断此人的情绪沉浸道：“关我屁事。”
<br/>　　“不。”白晟立即反驳，直勾勾的看着楚涵：“你难道不觉得你们很像吗？一样的年轻，一样的万里挑一，一样的聪慧擅长谋略，甚至同样都是各方势力的领头人。只是你走阳关，他则独木，你用阳谋，他用阴谋，你遇到险境极限反胜，他选择绕道而行。你敢于面对问题，他却因为从小亲情的缺失而……”
<br/>　　“不好意思你搞错了。”楚涵再次打断他：“我一点不像你口中说的那样，我是个不择手段的人，要说阴险狠辣，实在抱歉我认为你的儿子不及我万分之一。”
<br/>　　楚涵直接的反驳和打脸，让气氛一度尴尬，也让一旁听得有些恍惚的尚九娣和陆玟丞神色迷茫。
<br/>　　在他们看来，白晟对楚涵的评价根本没有错，甚至夸赞的让他们很认同，也从心底感到愉悦和自豪。
<br/>　　楚涵忽然的打断，却硬生生的将这种愉悦的心情，像是扼喉般掐住，让人很难受，浑身难受的不行！
<br/>　　但这种难受的感觉只持续了一瞬间，尚九娣和陆玟丞又感受到诡异的精神一轻，再次看向前方时似乎耳清目明。
<br/>　　眼前的这个白家家主，身上那股柔和的气质貌似减弱了一分，神情看起来也不再那么大度光明。
<br/>　　因为白晟不知什么时候轮椅朝前滑动了颇长的一段路线，距离楚涵仅有十米，他的目光有些诡异的焦急，双手抬举在半空中朝着楚涵张开五指。
<br/>　　陆玟丞和尚九娣大惊！
<br/>　　他们在此之前，竟然根本没察觉到白晟的此举！
<br/>　　怎么回事？
<br/>　　刚刚他们怎么回事？
<br/>　　失忆！
<br/>　　还是幻觉！
<br/>　　这时候的楚涵，却嘴角斜斜勾起，一个标志性的张狂不羁笑容浮现：“老魔头，跟我玩心术？”
<br/>　　此时在远处高楼顶层的时光炮弹处。
<br/>　　那名狼牙情报部成员忽然发现到了一丝不对劲，呆呆的望着自己放开炮弹的双手：“我？我什么时候……”
<br/>　　一旁的叶永阳猛然一个激灵，惊恐万分的后退好几步，道：“白家秘术……不能听！快！快让他们捂住耳朵！”
<br/>　　“怎么回事？！”情报部的成员大惊，满是恐慌。
<br/>　　此时的叶永阳却不知道从哪里找出了一根尖锐物，狠狠一下子扎穿了自己的双耳！
<br/>　　鲜血溢出耳道，场面一度失控。
<br/>　　叶永阳忍着痛，将尖锐物递到情报部成员手中，声音颤抖：“快扎！白家秘术，攻其心，迷其魂！”
