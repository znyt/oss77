　　“王博士，你们几个就先在这里休息吧，这个地方之前受到过袭击，应该会很安全，刘刚，你带一百人留在这里保护王博士！若是王博士和其余专家少一根汗毛！提头来见！”
<br/>　　“是！”
<br/>　　一个中尉立即大喊道。
<br/>　　“其余人，跟我走，援助曙光基地！驱逐黑暗！”
<br/>　　周华大喊一声一群人立即上了车，林峰也翻身上了车，车队继续朝曙光基地赶去。
<br/>　　“王博士，接下来我会保护你们的安全，请你们放心，你们几个，立即把这里的尸体清理一下，免得吸引怪物前来，其余人布置战线，注意警戒！”
<br/>　　“是！”
<br/>　　百来个士兵立即开始分工行动起来。
<br/>　　林峰和周华坐在运输车内没有说话，此时此刻几百个士兵都没有说话，每个人都无比的沉闷，他们有的人也已经成家立业了，在曙光基地内有自己的父母有自己的爱人和小孩。
<br/>　　若是曙光基地没了，那么这一切也就全部没了，所以每个人都有势必要回去一决生死的决心。
<br/>　　而此时此刻的曙光基地。
<br/>　　大战依然在如火如荼的进行着。
<br/>　　数万只丧尸不畏生死的冲锋，他们是这一次大战之中的排头兵，他们的存在，就是用来消耗子弹的。
<br/>　　在曙光基地的墙脚下尸体已经堆积了足足七八米高，其中有人类的有丧尸的也有怪物的。
<br/>　　甚至其中还有几个A级怪物的庞大尸体。
<br/>　　原本平滑的驱逐之墙也已经被轰的破破烂烂，时不时还可以看到一根根尖刺。
<br/>　　这些都是变异怪物留下的。
<br/>　　“还真是够难缠的。”
<br/>　　两个黑袍人目光微微眯起，曙光基地的士兵之士气之背水一战的决心，让他有些意外。
<br/>　　哒哒哒。
<br/>　　每个士兵都疯狂的射击。
<br/>　　他们的眼睛都红了，甚至有的人耳朵都已经在这种炮火没有任何停歇的节奏下已经什么都听不到了，但是他们知道，自己不需要去听到，他们眼前需要做的唯一一件事情，就是扣动自己的扳机。
<br/>　　城在人在！！
<br/>　　“给老子狠狠的打！”
<br/>　　陆勇也掏出了他的配枪对着这些怪物疯狂射击。
<br/>　　然而他的手枪子弹直接被怪物的皮给弹飞了，根本丝毫未损。然而怪物群体却越来越猛。
<br/>　　“有意思，既然如此，那就轰开大门吧。”
<br/>　　两个黑袍人低喝一声。其中一人忽然是一下子伸出两根指头，狠狠的插入了自己的眼睛之中，把两个眼球给硬生生的挖了出来！
<br/>　　如此一幕，让战门之人吓了一大跳，心中更是畏惧，疯子！这第一邪教之人居然已经豪无人性到了这种地步！对自己都丝毫不留情！
<br/>　　鲜血顺着黑黢黢的眼眶留了下来，这个人剧烈喘气，很显然他也很疼。
<br/>　　不过他有着强大的信仰，可以让他忘记这些痛苦。
<br/>　　另外一个黑袍人，目光落在了几个战门之人身上。
<br/>　　“你们，要干什么。”
<br/>　　几个战门顿时感觉到了不好的念头。
<br/>　　但是下一刻他们一个个直接被扭断了脖子，包括那九阶的内门长老。
<br/>　　随后，他们的脑袋全部被拧掉，鲜血狂喷，两个黑袍人，蘸着鲜血，在地上花了些奇奇怪怪的东西，看上去似乎是拉丁文一样的字体。
<br/>　　随着最后一笔的落下，其中那个挖掉了自己双眼的黑袍人咬断了自己的舌头，然后喷在了地上。
<br/>　　“交给你了！用你的心灵蛊惑异能！让这群怪物为我们所用！！”
<br/>　　说完，他气绝而亡，到死之前他眼中都没有任何的痛苦，有的只是兴奋和死得其所，仿佛他做了什么了不得的大事。
<br/>　　另外一个黑袍人看着同伴死在自己面前也没有任何的表情波动，仿佛死的是一个微不足道的小草一样。
<br/>　　他继续写写画画，然后忽然朝怪物群之中一个变异的九阶水牛凌空一指。他的嘴巴念念有词，就好像念经的小和尚一样，听得到在说话，但听不清内容。
<br/>　　那A级变异水牛本来是在打酱油，忽然变得狂爆起来，头顶上的两个巨大牛角一下子变得更加粗大锐利，对着基地的大门就是一阵怼。
<br/>　　轰轰轰！
<br/>　　仿佛地动山摇一样，整个曙光基地之人都感觉到了这强大的震动感。
<br/>　　撞的这个水牛头破血流，甚至眼珠子都掉出来了，但他却依然还在撞击，仿佛他也不知道了疼痛。
<br/>　　“还不够。”
<br/>　　黑袍人继续朝怪物群之中一个八阶的变异龙虾一指，这个变异龙虾的两对钳子变得犹如锤子一样，他本来也在旁边啃着尸体，尸体也不吃了，直接也疯狂的朝基地大门咋去。
<br/>　　DuangDuangDuang。
<br/>　　就好像用了霸王洗发水一样，这股变异龙虾无比的野蛮狂霸，不断的锤击，直到后来他的锤子都烂了，他还在用头撞。
<br/>　　“还有你，也上！！用你的毒液去腐蚀！！”
<br/>　　黑袍人一指一个A级的变异眼镜王蛇，对方立即张开射嘴，不要钱的狂喷毒液，这些毒液，放在末世前，那寻常人沾到了一滴都会尸骨无存。
<br/>　　而现在毒性更加恐怖强烈，空气之中都散发出一股恶臭，而这毒液就好像水枪一样不断的喷射大钢门，好在这个钢门乃是特殊材质锻造，防爆防震防腐蚀，但即使如此，表面也被腐蚀冒着白烟。

<br/>　　陆勇等人看着这一幕都是瞪大了眼睛，他们曙光基地的大门是特别制造的，哪怕炸药都炸不开，这群怪物就好像一个个把这大门当做了自己的仇人一样。
<br/>　　“给我狠狠的打！不能让这群家伙破开大门！”
<br/>　　陆勇脑海之中有个大胆的想法，那就是这群怪物似乎在想着，不及一切后果都要攻破大门！
<br/>　　哒哒哒。
<br/>　　黑袍人疯了，怪物也疯了，而曙光基地的战士们也疯了。
<br/>　　“枪架断了！”
<br/>　　一个重机枪手破口大骂道。
<br/>　　“妈的，我来固定，你给我扣动扳机！！”
<br/>　　每个人都疯狂的扫射，甚至有个机枪手的副手，直接把一个重机枪抗在自己肩膀上来给射手稳固。
<br/>　　然而重机枪枪管连续射击会导致枪管发热，这还只是其次罢了。等机枪手射完了一个弹夹，这个副手也直接口鼻之中鲜血狂喷，瞪着眼珠子，就这样被机枪的后坐力给活生生的震碎了内脏。到死都还是保持着固定机枪的姿势。
