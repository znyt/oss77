　　离开天枢宫后，慕清霄就直接返回天璇宫。
<br/>　　此时，草谷正盘坐于一座黑色的丹鼎前，丹鼎中紫色兽火熊熊燃烧，将她那可爱的小脸蛋都映照的绯红一片。
<br/>　　不多时，丹鼎中传出一股浓郁的药香味。
<br/>　　只见她睁开双眼，其中闪烁着喜悦，将鼎盖开启，望着内部躺着的几颗丹药，眼底却闪过一抹失望。
<br/>　　“咳咳。”
<br/>　　熟悉的声音在耳边响起，草谷回头一看，不知何时，慕清霄已经出现在了她的身侧。
<br/>　　“长老哥哥，草谷明明是按照《朱子率》上的方式炼制的筑基丹，为何炼制出来后，品质和数量都没有达到要求呢？”
<br/>　　慕清霄走上前来，捏起她小手中的土黄色筑基丹打量了一眼，视线最后落在了她的丹鼎上。
<br/>　　“炼丹最主要的是技术，丹鼎其二，若是有一座好的丹鼎，炼制出来的丹药品质也会不同。”
<br/>　　视线从丹鼎上收回，慕清霄轻笑一声道：“你的丹鼎非常的普通，但能练出十余颗筑基丹，可见你在二品炼丹师中，技术也是拔尖的。”
<br/>　　草谷心灵聪慧，顿时了然，大眼睛中闪烁着喜悦，不知是因为了解原因，或者是被慕清霄夸赞。
<br/>　　对于草谷来说，慕清霄就是她的师傅，在她心目中，后者一直都是温和无比，待她像亲人一样。
<br/>　　回过神来，草谷的大眼睛中闪过一抹皎洁，视线落在了天璇宫中央的青色丹鼎上。
<br/>　　“哎哟……”
<br/>　　慕清霄没好气的敲了一下她的小脑袋，道：“那是为师的，不过念在你资质不错的份上，给你一件礼物。”
<br/>　　说着，慕清霄随手一挥，一道墨黑色丹鼎就出现在草谷的面前，丹鼎通体墨黑，其上纹路交错，以及各种魔兽的图案，正是万兽鼎。
<br/>　　“长老哥哥给草谷的吗？”
<br/>　　“不错。”
<br/>　　将万兽鼎上的灵魂印记抹除，慕清霄温和的笑道：“这座鼎叫做万兽鼎，给你炼丹绰绰有余，完全能支撑你将《朱子率》中的炼丹术全部学会。”
<br/>　　望着眼前的丹鼎，草谷撅了噘嘴，道：“长老哥哥，丹鼎上的图案好丑呀。”
<br/>　　慕清霄嘴角抽了抽，魔兽的图画确实狰狞的一点，可这小丫头真是身在福中不知福，有就不错了，居然还嫌丑。
<br/>　　要知道万兽鼎可是非常不错的丹鼎，在斗破苍穹世界中位列天鼎榜，许多老怪物想要都没有呢。
<br/>　　“将就着用吧，等你什么时候有能力了，自己去找更好的，收起来吧。”
<br/>　　由于草谷现在境界过低，无法凝聚灵魂印记，万兽鼎相当于是无主之物，不过有蜀山当作靠山，想必这丫头也不会被人劫了财。
<br/>　　“回去好好修炼，为师有事情需要处理，就不留你了。”
<br/>　　“喔。”
<br/>　　应了一声，朝着慕清霄吐了吐舌头，小丫头这才将万兽鼎收回纳戒，随后蹦蹦跳跳的离开了天璇宫。
<br/>　　待草谷离去，慕清霄的视线就落在了天璇宫中央的青色丹鼎上，蟠龙刻凤，简直与《上古神器志》有八成相似。
<br/>　　轻轻抚摸中丹鼎，慕清霄嘴角牵起一抹笑意，随后在其上印下了灵魂印记。
<br/>　　“系统，我需要这座丹鼎的信息。”
<br/>　　“叮，需扣除二十万掠夺积分。”
<br/>　　二十万掠夺积分，对于现在的慕清霄来说，已经不算很多，爽快道：“扣除吧。”
<br/>　　“神农鼎，上古神器（残缺），持此鼎可令百兽臣服，相传神农以此鼎炼药，无有不成。”
<br/>　　系统提示声刚落，慕清霄心潮澎湃。
<br/>　　果然是神农鼎，唯独让他失望的是，这神农鼎居然是残缺的，并非完整。
<br/>　　不过，虽然是残缺的神器，但也是超越了仙品的品质。
<br/>　　将神农鼎收回储存空间中，慕清霄想着，神农鼎残缺，太清真人肯定知道其原因，若是有机会，一定要将神农鼎复原。
<br/>　　定下心神，慕清霄来到天璇宫后的凉亭中。
<br/>　　灵气化雾，缭绕周身，一袭白衣，黑发及腰随风飘动，慕清霄站在湖泊前，仔细阅览起仙品剑诀。
<br/>　　经过一番思考，慕清霄决定学习六种仙品剑诀中的两种，《裂穹斩》与《御剑伏魔》。
<br/>　　《摧岳撼》对于剑有要求，需要重剑，慕清霄所持魂剑并不适合修炼此招式。
<br/>　　《刑锁念刃》则与《太虚剑》有些相似，都是将灵气凝聚成巨剑轰杀敌人，这招是大范围攻击，对于灵气的消耗过大。
<br/>　　所以，慕清霄暂时不打算修炼。
<br/>　　将《裂穹斩》剑诀仔细的阅览了一遍，慕清霄体内阴阳灵气翻滚不休，顺着右臂汇聚于掌心，随后一柄巨剑形成。
<br/>　　巨剑横空一挥，整片空间只是轻微的扭曲了一下，威压也并不是很强。
<br/>　　对此，慕清霄非常的不满，他感受过地仙强者的威压，现在手中的灵气巨剑威力根本不足以达到仙品。
<br/>　　将灵气驱散，慕清霄再度将灵气汇聚于掌心，剑意与威压只是比上次强了一些。
<br/>　　“果然，仙就是仙，凡就是凡，以自己现在的境界，想要修炼《裂穹斩》依旧是困难了些。”
<br/>　　……
<br/>　　……
<br/>　　光阴似箭日月如梭，两个月的时间转眼即逝。
<br/>　　天璇宫后，慕清霄屹立在湖泊前，星眸渐渐睁开，周围灵气狂涌，旋即一柄数十仗的巨剑渐渐凝聚而成。
<br/>　　“铿铿……”
<br/>　　清脆的剑鸣声响起，慕清霄右手猛的挥出。
<br/>　　伴随着剑刃破空之声，巨剑瞬间划过苍穹，在虚空中留下一道极长的黑色裂缝。
<br/>　　在恐怖的力量下，湖面泛起涟漪，涟漪渐渐变大，最后形成浪花……
<br/>　　巨剑崩碎，慕清霄嘴角牵起一抹笑意。
<br/>　　“虽然威力远不足仙品剑诀，但已经有了雏形，欠缺的只是自身实力。”
<br/>　　整整两个月，他寸步未离开此处，一直都在研究剑诀，不断的修炼，不断的完善，终究只能施展出仙品剑诀的雏形。
<br/>　　慕清霄完全有自信，等他成为地仙，完全能施展出《裂穹斩》，甚至威力会比其他修士更强。
<br/>　　深吸了口气，嘴中吐出一口气浪，星眸中光泽流转。
<br/>　　“也是时候，去灵溪谷一趟了。”
<br/>　　……
<br/>　　……
