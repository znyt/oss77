　　时间缓缓流逝，黄泉魔宗的众强者静静的等待。
<br/>　　约莫过去了一刻钟之后，前方的这位‘元老’，缓缓的睁开了眼眸，眸中漆黑的光芒一闪而逝。
<br/>　　“拜见宗主！”
<br/>　　“拜见宗主！”
<br/>　　“拜见父亲！”
<br/>　　霎时之间，一众黄泉魔宗的强者，纷纷单膝跪地，向这名‘元老’行礼。
<br/>　　原来，这根本就不是黄泉魔宗的元老，而是黄泉魔宗的宗主巫森。
<br/>　　武帝境界的强者，已经形成了真正的战魂，战魂不灭，便可以随意的夺舍他人的肉身。
<br/>　　显然，现在这位元老的肉身，已经被巫森夺舍。
<br/>　　至于是巫森强行夺舍，还是那位元老自愿而为，那就无法得知了！
<br/>　　而武帝境以下的武者，虽然也有特殊的方法夺舍，但并不完全。
<br/>　　比如巫山，虽然夺舍了李峰，但李峰并没有死，他只是暂据李峰的肉身而已。
<br/>　　只有在巫山突破到武帝之后，才能彻底泯灭李峰的灵魂意识，完成真正的夺舍。
<br/>　　“查探的怎么样了？”冷漠而又冰寒的声音，从巫森的口中响起。
<br/>　　“禀父亲，都查探清楚了！”巫山抱拳说道。
<br/>　　“说！”
<br/>　　“是！”巫山点头，略一沉吟，道：“如今的苍穹世界，苍穹神宫独霸天下，所有大势力尽皆臣服；苏莫在苍穹神宫和各大势力之间，建立了紧密的联系，力量完全可以拧在一起！”
<br/>　　“苏莫的实力呢？”巫森淡淡的问道，面对自己的儿子，他的面上也是没有任何表情。
<br/>　　苏莫的实力，才是巫森最为关心的问题，天下各大势力，根本无需考虑。
<br/>　　“不清楚！”巫山闻言，立刻摇了摇头，他们根本查探不出如今苏莫的实力。
<br/>　　因为，苏莫已经数年没有出手了，根本没有人知道其具体的实力达到了什么地步！
<br/>　　他们查探消息，根本不敢距离苍穹神宫太近，自然便无法查探到苏莫的修为实力。
<br/>　　巫森闻言，微微皱眉，不清楚苏莫的实力，倒是一件麻烦事。
<br/>　　“宗主，已经过了五年之久，想必苏莫很可能达到了准帝之境！”
<br/>　　一名黑袍中年人开口了，此人正是阴阳神宫的宫主阳鼎天。
<br/>　　现如今，苍穹神宫独霸天下，阴阳神宫和北堂世家已经没有了退路，他们都尽数加入了黄泉魔宗。
<br/>　　其门下的弟子，都隐藏在一起。
<br/>　　“我觉得很有可能！”
<br/>　　“以苏莫的天赋，达到准帝之境，应该没有问题！”
<br/>　　“五年时间，对于苏莫来说，提升到准帝之境很正常！”
<br/>　　阳鼎天的话音一落，顿时的到了很多黄泉魔宗强者的赞同。
<br/>　　众人都知道苏莫的天赋，所以都有这个猜测。
<br/>　　至于为何没有人猜测苏莫已经达到了武帝之境，那是因为一旦有人突破到武帝之境，整个苍穹世界，都会出现天地异象。
<br/>　　然而，这几年并没有任何异象发生，所以，苏莫不可能突破到武帝之境。
<br/>　　“我觉得不可能！”这时，一道否决的声音响了起来，顿时让洞穴中寂静了下来。
<br/>　　众人转头看向开口之人，目露疑惑之色。
<br/>　　开口之人正是皇甫擎，现如今的皇甫擎虽然没有了修为，但其在黄泉魔宗的地位，却是水涨船高。
<br/>　　因为皇甫擎拥有强大的肉身力量，足以堪比武皇境五、六重的武者。
<br/>　　“皇甫擎，怎么说？”二魔老开口问道。
<br/>　　皇甫擎面露沉吟之色，道：“想必各位都知道，苏莫曾经拜在我门下，我对此子深有研究！”
<br/>　　“此子不知道修炼了什么武学，所以战力超凡，而因此，他的修炼一直都需要大量的资源！是常人的百倍！”
<br/>　　“最近，我特意调查过，这几年苏莫从各大势力搜刮资源，简直将所有大势力搜刮了个干净，所以他根本没有足够的资源修炼！”
<br/>　　“我敢断定，他最多武皇境九重境界，最有可能是武皇境八重境界！”
<br/>　　皇甫擎徐徐说道，将自己的分析告知了众人，说的条理分明。
<br/>　　众人闻言惊疑，大多数人都是有些赞同皇甫擎的分析，的确有道理。
<br/>　　毕竟，皇甫擎是比其他人要更了解苏莫。
<br/>　　不过，众人的心中都很是凝重，即便苏莫只有武皇境八重的修为，也是非常骇人了。
<br/>　　毕竟，武皇境四重修为之时，苏莫就能打败武帝境的巫森，提升到了武皇境八重修为的苏莫，该有多么的恐怖！
<br/>　　估计现在，他们所有人联合起来，都不够苏莫两招镇杀的。
<br/>　　巫森沉默不语，眼眸中闪烁着森冷的寒光。
<br/>　　“诸位，我族的试炼之人，今年就会降临！”良久之后，巫森淡淡的说道。
<br/>　　众人闻言，顿时心中一震，而后个个面现狂喜之色。
<br/>　　终于要来了吗？
<br/>　　他们终于能离开苍穹世界了吗？
<br/>　　走向那浩瀚无边，万族争锋的宇宙星空！
<br/>　　所有黄泉魔宗的强者，他们都有一个共同的目的，那就是跟随巫森离开苍穹世界。
<br/>　　所以，即便黄泉魔宗沦落至此，成为了丧家之犬，他们也不离不弃。
<br/>　　阳鼎天和北堂平华两人，也正是这个原因，才会放下一切，帮助黄泉魔宗。
<br/>　　没有人怀疑巫森的话，因为巫森便是半个巫族之人。
<br/>　　巫森乃是上一届试炼者的子孙，和巫族有某种特殊的联系。
<br/>　　“待我族试炼之人降临，铲平苍穹神宫，我等一起离开苍穹世界！”巫森继续道。
<br/>　　“是，宗主！”众人齐声应和，个个面色激动，离开了苍穹世界，他们就能更好的追寻武道，将来拥有几千年、甚至是几万年寿命。
<br/>　　什么苍穹神宫？什么苍穹世界？与他们何干！
<br/>　　皇甫擎的目光抬起，仿佛洞穿了厚实的岩壁、无尽的海水，亿万里的虚空，落在了苍穹神宫。
<br/>　　“苏莫，你终究难逃一死！”皇甫擎低声自喃，眸中焕发出璀璨的神采。
<br/>　　他加入黄泉魔宗，只是为了离开苍穹世界而已，一旦离开，外界之大，他将逍遥自在。
<br/>　　唯一比较遗憾的是，他的几名弟子中，天辰和安暖被苏莫掠去，身边只剩下了花碎羽、君无惜和风少羽。
<br/>　　虽然几名弟子的生死，他不太在意，但毕竟是他耗费心血培养起来的，很是可惜！
