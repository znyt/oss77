　　虽说已经来到酆都，但鬼城并非极乐世界。
<br/>　　鬼城与极乐世界分别是由不同鬼王执掌的两块地域，整个鬼界也不可能只由一名鬼王执掌。
<br/>　　整个鬼界究竟有多么庞大，他并不清楚，总共有多少鬼王，他也不清楚，唯独知道火灵珠在谁的手中。
<br/>　　慕清霄需要的火灵珠，正是在执掌极乐世界的火鬼王手中，并非其他鬼王。
<br/>　　经过数千年的修炼，火鬼王具有极深的道行。
<br/>　　她居住在鬼界最下层，一直和阎罗鬼王作对，虽受到不少幽魂野鬼拥戴，但实力上，和阎罗鬼王相差太远。
<br/>　　火鬼王有火鬼界的熔岩作为天然屏障，阎罗鬼王也奈何她不得。
<br/>　　火鬼王本身就不惧怕岩浆，更何况有火灵珠在手，但别的鬼物，甚至鬼王却不行。
<br/>　　岩浆对于慕清霄来说无所谓，若真是在岩浆中战斗，火鬼王不见得能占据地形优势。
<br/>　　重要的是，他现在只知道极乐世界在鬼界深处，却不清楚明确的位置，甚至进入鬼界的方法，他都不得而知。
<br/>　　不过，他不知道，不代表某些鬼魂不清楚，比如‘鬼皇宫’中的鬼皇。
<br/>　　“嗯？”
<br/>　　就在这时，两道身影自‘鬼皇宫’中疾射而出，慕清霄嘴角牵起一抹若有若无的笑意。
<br/>　　原本，他就打算去‘鬼皇宫’，却没想到后者倒是先找上门了。
<br/>　　两道身影浑身充斥着阴寒的气息，自天际掠过，明显是鬼修，而且修为不低，实力堪比仙门中的化神期修士。
<br/>　　两位鬼修的速度不慢，十几秒就出现在慕清霄的眼前。
<br/>　　两者眼中古井无波，脸上却保持着恭敬的神色，微微行礼道：“大人，天鬼皇殿下有请。”
<br/>　　“天鬼皇……”
<br/>　　听到天鬼皇三字，慕清霄微微一愣，旋即眼中闪过一抹精芒。
<br/>　　他对于鬼界中其他鬼王的信息并不清楚，但偏偏火鬼王与天鬼皇，他都有一定的了解。
<br/>　　天鬼皇乃是鬼界的鬼王之一，同时也是天鬼族首领，隋文帝年间被关入锁妖塔。
<br/>　　被关入锁妖塔八年后，在李逍遥等人的帮助下重见天日，并且认李逍遥为恩公，事后也对帮助李逍遥等人度过困境。
<br/>　　天鬼皇身为鬼族，却极重义气，知恩图报。
<br/>　　后者虽没什么脑子，易受人利用，而且有一些爱慕虚荣，但人品还是有保障的。
<br/>　　见慕清霄陷入沉思，两位鬼修也不敢出声打扰，默默的站在一旁。
<br/>　　至于街头上的鬼族，纷纷被这一幕吸引，都怔怔的站在原地，心里却对被鬼王接见的慕清霄，充满了羡慕与好奇。
<br/>　　片刻后，慕清霄抬起头来，道：“前面带路吧。”
<br/>　　既然知道了后者的身份，慕清霄也不需要做任何的部署，因为后者的实力，根本不足以威胁到他。
<br/>　　既然派鬼修来与自己接触，可见后者必有所图。
<br/>　　根据天鬼皇的性格，慕清霄多多少少能猜出后者的目的，正好他也有目的，两者不谋而合。
<br/>　　跟随着两位鬼修来到鬼皇宫，偌大的宫殿中，氛围显得有些冷清。
<br/>　　鬼城并非大型城池，城外都是荒地，也没有附属城市。
<br/>　　金碧辉煌的宫殿是由天鬼皇自己建造，目的自然是满足小小的虚荣心。
<br/>　　毕竟，天鬼皇的性格有些爱慕虚荣。
<br/>　　对此，慕清霄并不感到奇怪，世间没有不喜欢权利的人，野心谁都有，只是没有实现野心相应的力量罢了。
<br/>　　来到正殿门口，两位鬼修已经悄无声息的退下，前面已经不是他们能踏足的地方。
<br/>　　慕清霄望着高高的殿门，星眸仿佛将其内部的事物看穿，面色如常，缓步走进其中。
<br/>　　来到殿内，寂静无声。
<br/>　　慕清霄抬头望去，只见殿上的黝黑的魔龙座椅上，躺坐着一名魁梧男子。
<br/>　　天鬼皇身穿黑袍，身子笼罩在偌大的披风下，容貌坚毅肃穆，显得颇有霸气风范。
<br/>　　“阁下来本座的鬼城，可有何事？”
<br/>　　天鬼皇声音沉稳，面无表情，冰冷的双眼盯着殿下的慕清霄，霸气测漏，完全与信息中的性格不同。
<br/>　　慕清霄微微一笑，道：“鬼王乃是鬼界霸主，但你却屈居于鬼界外围的小小鬼城中，看来鬼界暂且没有你能插足的地方啊。”
<br/>　　“请阁下慎言！”
<br/>　　天鬼皇脸色一沉，但语气中却听不出喜怒哀乐。
<br/>　　撇了撇嘴，慕清霄道：“如果在试探下去就完全没意思了，说说你的目的吧，或许我们能够站在同一战线上。”
<br/>　　被慕清霄一语道破，天鬼皇严肃的表情一凝，旋即露出一抹笑意，道：“阁下所言当真？”
<br/>　　鬼城各地都有他的眼线，在慕清霄出现在城内的第一时间，一举一动都完全暴露在了他的眼中。
<br/>　　看不透慕清霄的修为，天鬼皇思索片刻就晕下了决定，他需要慕清霄的帮助。
<br/>　　他在赌，赌慕清霄来鬼城的目的，而修士来鬼城的目的，那就只有一个，便是进入鬼界中。
<br/>　　正如慕清霄所言，他是鬼王，乃是一方霸主，可偏偏委身于小小的鬼城中，他不甘心。
<br/>　　慕清霄的出现，让他看到了希望，
<br/>　　见慕清霄不说话，天鬼皇语气平淡道：“阁下既然肯来鬼皇宫，想必也是有求于本座把？”
<br/>　　“不错，我需要进入鬼界的方法，还有极乐世界的明确位置。”
<br/>　　“火鬼王的领域吗？阁下的目的是火灵珠？”
<br/>　　听到极乐世界四个字，天鬼皇心中一惊，同时对于慕清霄的目的已经非常清楚。
<br/>　　除了火灵珠，天鬼皇再也想不到，极乐世界中有其他东西值得慕清霄冒险了，只是极乐世界中的火鬼王并不好惹，他也不想参合其中。
<br/>　　“既然你都已经知道了，该做出决定了吧，我也不想在这里浪费时间。”
<br/>　　天鬼皇凝重的点了点头，道：“本座只能带你进鬼界，并且告诉你极乐世界的位置，但本座也有条件。”
<br/>　　“说说看吧。”
<br/>　　慕清霄洗耳恭听，如果要求不过分的话，顺手而为也没什么。
<br/>　　“本座的目的是‘阴寒域’，其中由双煞鬼王兄弟执掌，只需要你帮本座除掉他们。”
<br/>　　……
