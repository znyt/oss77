　　第94章
<br/>　　“我有婚约在此，徐兰不能和你成亲！”
<br/>　　罗子开手中高举着一纸婚书。
<br/>　　这时代结婚讲究三媒六聘，问名纳彩等等繁琐的礼数缺一不可。如果罗子开手中这纸婚书是真的，今天这婚还真结不成。
<br/>　　“家丁，给我叉出去。”新郎断喝。
<br/>　　“贾武，你这卑鄙小人，你假意亲近我，却背着我勾引兰儿，枉我当你这狼心狗肺的人是朋友。”罗子开大喝。
<br/>　　新郎贾武脸上有些挂不住了，他喝道：“都干什么吃的，都给我打！”
<br/>　　几个人高马壮的家丁冲出来，今天的事情无论怎么着，对贾家都是丢人的事情。罗子开突然从身上抽出一把菜刀，大喝道：“你们谁敢过来，一刀两命，今天我也不活了。”
<br/>　　程大雷撇撇嘴：“为了一个女人如此疯癫，有失体面呐。”
<br/>　　横的怕楞的，楞得怕不要命的，罗子开一旦开始玩命，这帮家丁还真不敢过来。
<br/>　　贾武在军中当官，耍得一手好枪棒。他挥手夺过家丁手中的大棍，只是一回合就把罗子开砸倒在地。
<br/>　　“凭你也想和我玩命，你也配。”贾武今天已经丢人了，索性再丢得大些：“怪只怪你没本事留住女人，跑到这里撒什么野，来人给我打！”
<br/>　　“啧，抢人女人还这么理直气壮么。”程大雷继续吐槽。
<br/>　　一群家丁冲过来，对罗子开拳打脚踢，终究不过是个手无缚鸡之力的读书人，不多时就被打得半死不活。
<br/>　　他抱着脑袋在地上乱窜，突然脑袋一碰，撞到一条桌子腿上。
<br/>　　“给我往死里打，打死后丢到城外乱葬岗。”贾武今日被败了性子，胸中怒火焚烧，以贾家在城中的地位，打死个把人还真不叫什么事。
<br/>　　如果是罗家以前的势力当然不一样，可如今罗家破落了，所谓拔了毛的凤凰不如鸡。
<br/>　　罗子开倒在地上，眼皮黏着血，一条命究竟还剩下多少，如今却也说不太好。他迷迷糊糊倒在地上，心如死灰，便是今日死了也便是死了。
<br/>　　“喂，喂……”
<br/>　　迷迷糊糊中，感觉有只手拍他的肩膀，他挣扎着睁开眼睛，看到一张大脸杵在自己面前。
<br/>　　“喂，少年，你相信正义么？”
<br/>　　罗子开脑子有些不清楚，稀里糊涂中听到这句话，一下愣住了。
<br/>　　程大雷哈哈笑笑，长身而起，一脚将桌子踹翻，锵啷拔出身上佩剑。
<br/>　　“蛤蟆寨在此做事，今天只找贾家麻烦，无关人等速速退下！”
<br/>　　徐神机和刘悲此刻也站起来，拔出身上短刀，莫看二人是文职，但毕竟是山贼，杀起人的凶悍绝对胜过普通人。
<br/>　　声音落地，宛若平湖起惊雷，一众人哇哇乱叫，有向外跑的，有往桌子下面躲的。
<br/>　　贾武提着大棍向程大雷冲过来，程大雷持剑和对方拆了几招，不是对手，程大雷不是贾武对手。论本事二人其实差不了多少，程大雷关键输在兵器上。
<br/>　　剑这种兵器，用来充排面是相当不错的，当真战场杀敌，其实没多少杀伤力。
<br/>　　程大雷心里有些发虚，再打下去就要输啊，这第一脚要是踢不响，以后的事情就不好干了。
<br/>　　正此时，一棍凌空飞来，重重砸在贾武腰肋上。贾武身子一歪，半跪在地上，程大雷立刻把剑压在对付脖子上。
<br/>　　“动一动，要你的脑袋！”
<br/>　　秦蛮，林少羽，张肥也从里面打将出来，这一棍就是秦蛮丢的，有秦蛮开路，自然无人能敌。最惹眼的是张肥，手持两把菜刀，三人里面数他本事最弱，数他叫得最响。
<br/>　　秦蛮过来，三下两下就拆下来贾武的膀子，贾武两肩脱臼，手臂无力垂下。
<br/>　　徐神机刀压贾武脖颈，口中大喊：“杀呀，杀呀！”
<br/>　　“杀什么杀，再不走就走不成了。”程大雷踹了徐神机一脚，目光注意到一旁的新娘。
<br/>　　刚才大乱，新娘已经吓得瑟瑟发抖，她脸上还戴着盖头，除了新郎外其他人是不能揭的。
<br/>　　程大雷一剑就给她挑开了，随着红布垂落，露出一张吹弹可破的俏脸。因为恐惧而颤抖，眼中的神情却更加动人。
<br/>　　“好俊的妹子！”程大雷伸出大手捏了捏对方肥嘟嘟的脸蛋，留下一个黑印：“果然好俊，如此也不愧两个男人为你争一场。”
<br/>　　生得如花似玉的新娘吓得脸色煞白，却是一动不敢动。
<br/>　　“程当家，程当家……”一个锦袍老者从里面跑出来：“程当家，我们和你远日无怨，近日无仇，还求程当家放过犬子性命，程当家需要什么财物，老朽一定满足。”
<br/>　　“什么叫无冤无仇，我令你们散尽家财，你们做了没有。今天强抢民女，被我抓个正着！”程大雷道：“还说什么守不住女人只能怨自己没本事，今天本大王要抢你儿媳妇，你也答应么？”
<br/>　　“若是程当家看得上，送给程当家便是，只求放小儿一条性命。”
<br/>　　“啧……”
<br/>　　程大雷无语了，世界观不同，还真是无法交流啊。他也不和这老者废话，此处和自己拖延时间，恐怕早就安排人去喊城内守军了。
<br/>　　“带走带走！”
<br/>　　押着贾武，几人浩荡离开，刚出贾府，贾家的家丁就追了出来。这件事不可托大，几人带着贾武加快了脚步。
<br/>　　一路从小道出城，雪橇和马都在城外隐蔽处藏着，只要上了雪橇，追兵的速度根本追不上大家。
<br/>　　相继从墙上跳下来，积雪颇深，倒也不用担心摔伤。
<br/>　　程大雷目光转了一圈，目光忽然呆住：“马，我的马呢？”
<br/>　　原先藏马的地方空空荡荡，除了散乱的马蹄，一根马毛都没有。
<br/>　　“大当家，会不会被路过的人牵走了？”徐神机。
<br/>　　“这地方根本不会有人经过，你以为我藏马的时候没想到嘛。”程大雷气得七窍生烟：“这特么是冲我下手啊！”
<br/>　　“把人给我就地埋了。”
<br/>　　秦蛮一棍将贾武敲晕，倒栽葱埋在雪地中，只有两只脚露在外面。
<br/>　　城里响起闹哄哄的声音，追兵已经近了。可没有马，在雪地中想要摆脱追兵，势比登天还难。
<br/>　　“特么，终日打雁竟被家雀抓了眼，敢劫到我身上，让我知道是谁，绝饶不了他！”
<br/>　　“大当家，现在怎么办？”徐神机问。
<br/>　　程大雷听着渐近的追兵，猛地一咬牙：“进城！”
<br/>　　
<br/>(本章完)
