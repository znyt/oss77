　　楚风眠这样大摇大摆的追杀小魔主，在这群山周围的强者自然不可能知晓，如今有人出手，直接要救走小魔主。
<br/>　　这一股灵力，压迫在楚风眠的身躯上，其中的雄厚，简直是要超出楚风眠的想象。
<br/>　　这出手的强者，比起什么锁魂魔尊，震旦魔尊等人，都要强大的多，一身灵力雄厚，简直是深不见底。
<br/>　　尤其是此人，也同样是生死境巅峰的境界，还没有超越生死，达到碎命之境。
<br/>　　但是这一身灵力雄厚程度，只怕是丝毫不比真正碎命之境的武者要差。
<br/>　　楚风眠都有些吃惊，没想到在这天羽神域中，还有着如此可怕的人物。
<br/>　　不过今日，就算是谁来，这小魔主他也是要必杀，不杀小魔主，终究是心头大患。
<br/>　　“剑势！”
<br/>　　楚风眠身上剑意猛然爆发出来，这一股剑意，锋利如芒，将他身上压迫的灵力，全部斩断，令楚风眠脱离压制，剑光再次冲着小魔主斩杀过去。
<br/>　　“哦？居然足以化解我的压迫，天羽神域，什么时候来了这样一号人物，居然还是一位剑修？不错，在这天羽神域中，终于又来了一个剑修。”
<br/>　　这神秘的声音，再度响起。
<br/>　　“既然你要杀小魔主，那我就跟你比一下剑道吧！”
<br/>　　突然，就在楚风眠身上压迫的灵力，全部消失不见，取而代之的，则是一股剑意，笼罩在了楚风眠的身上。
<br/>　　这一股剑意，锋利入剑，笼罩下来的一刻，其中的杀意便是毫不掩饰的展示出来。
<br/>　　一股股剑意，全部化剑，要将楚风眠直接斩杀。
<br/>　　尤其是这剑意，更是从遥远的地方传递而来，楚风眠感觉到，这出手的人，并不在附近。
<br/>　　只是动用力量，打算来救下小魔主，还足以达到这种剑意，着实也让楚风眠吃惊。
<br/>　　“天羽神域中，真正拥有这等实力的武者，只有天剑宗的剑祖荒，圣王宗的鹿战神跟荒古门的荒圣君，鹿战神，荒神君都不可能拥有如此剑意，你的剑祖荒？”
<br/>　　感受到这剑意的一刻，楚风眠便是突然明白过来，这出手之人的身份。
<br/>　　在天羽神域中，能做到这一切的，唯有剑祖荒，天剑宗的超级天才。
<br/>　　剑祖荒冷哼一声，开口道。
<br/>　　“哦？没想到你居然知道我，看你的实力，我们天剑宗弟子，应该都是死在了你的手中了吧。”
<br/>　　楚风眠轻描淡写的回答道。
<br/>　　“是又如何？不是又如何？”
<br/>　　“是，今日你就是必死无疑，不是，你追杀小魔主，我不会杀了你，但是你必须要跪下自废修为，可以免去一死。”
<br/>　　剑祖荒冰冷的开口道，他的语气，便是至高无上的审判一般，似乎是要宣判楚风眠的命运一般。
<br/>　　“不错，是我的杀的又如何？七大宗门的人，都是该死之人，现在你又来阻挡我，你也得死！”
<br/>　　楚风眠斩钉截铁的开口道，语气中都是蕴含这浓浓杀机。
<br/>　　剑祖荒又如何？
<br/>　　七大宗门的人，都得死，在绝世天才，也要被楚风眠斩杀。
<br/>　　“七大宗门的人都是该死之人？好！够狂！”
<br/>　　剑祖荒冷哼一声，他的话像是夸奖，但是其中的每一个字，都是蕴含有着无上剑意。
<br/>　　顿时笼罩在楚风眠身上的剑意，在这一刻之中，猛然暴涨了无数倍。
<br/>　　并且这杀意，要更加的恐怖，更加的蕴含杀机。
<br/>　　楚风眠的一番话，激起了剑祖荒心中的愤怒，令他起了杀心。
<br/>　　“单单想要以剑意，就想要斩杀我，我该说你是狂傲呢，还是愚蠢呢？”
<br/>　　楚风眠面对这剑意笼罩下来，嘴角露出几分不屑。
<br/>　　在剑道上，能胜他之人，都寥寥无几，更何况还单单是想要以剑意来压死楚风眠。
<br/>　　让天剑宗的宗主前来，都未必能做到这一点。
<br/>　　“剑意无极！”
<br/>　　楚风眠轻喝一声，他身上的剑意，也是猛然展现出来，他身上的剑势，疯狂攀升，猛然冲天而上。
<br/>　　简直是如同一只沉睡的猛兽，在这一刻，瞬间苏醒，要撕裂眼前挑衅的蝼蚁一般。
<br/>　　“给我破！”
<br/>　　楚风眠的剑锋，直冲天上，这笼罩下来的剑意，被瞬间击破。
<br/>　　同时在这瞬间，楚风眠的一道剑光，也是向着小魔主斩杀过去。
<br/>　　小魔主的身躯，顿时被这一道剑光贯穿，身躯都是化为粉碎。
<br/>　　就在这剑祖荒的面前，楚风眠杀了小魔主。
<br/>　　“畜生！”
<br/>　　一声爆喝，从天上猛然爆发出来。
<br/>　　剑祖荒，被彻底的激怒了，怒火中烧，他的愤怒已经是压抑不住。
<br/>　　“还从未有人敢在我的面前杀人！小子，我要将你碎尸万段！将你彻底毁灭！让你承受世间的一切痛苦！”
<br/>　　“我会让你知道，你的剑道，在我的面前，不过是不堪一击罢了！”
<br/>　　就在空中，无数剑意突然凝聚，逐渐的，化为了一名白衣男子。
<br/>　　这白衣男子，脸庞如剑，目光锋利，整个人仿佛都是剑的化身。
<br/>　　这便是只有将剑道修炼到极高的层次，才足以达到的境地，人剑合一，人为剑，剑为人。
<br/>　　“死！”
<br/>　　这白衣男子凝聚而出的一刻，顿时一剑之间，突然斩了过来。
<br/>　　这一剑斩出一刻，周围仿佛都是化为了剑的世界，无穷无尽的剑意笼罩下来，压迫这楚风眠的身躯，令楚风眠移动一下，都极为艰难。
<br/>　　楚风眠陷入到这剑之世界之中，好像是根本无法跟这白衣男子抗衡，面对这一剑袭来，就算是想躲，都无法躲开。
<br/>　　“想要杀我，让你的本体前来吧，一缕剑意凝聚出的化身罢了，也敢在我面前放肆？我就斩了你这一道化身，也让你损失一道剑意吧！”
<br/>　　楚风眠身躯一震，他的身边似乎无数剑光，突然闪耀一下，将周围的剑之世界，全部击碎。
<br/>　　“嘶！”
<br/>　　同时楚风眠身躯一动，祖龙至尊剑猛然出窍，一道剑光，猛然斩了过去。
<br/>　　两道剑光碰撞，这剑祖荒的剑光，被直接斩断，同时这一抹剑气，更是向着剑祖荒这一缕化身斩杀过去。
<br/>　　楚风眠冷笑一声道。
<br/>　　“这一缕化身，也凝聚了你不少剑意，杀了他，也会对你实力有损吧！”
