　　所有人都认为苏莫和千幻邪君陨落了，毕竟，身躯被强大的攻击直接粉碎，元神基本上也无法存活了。
<br/>　　然而，让众人意外的是，千幻邪君并未彻底陨落，身躯崩溃之后，一股气息诡异的气流，立刻向山下逃窜。
<br/>　　这气流呈乳白之色，正是千幻邪君的千幻元神。
<br/>　　千幻邪君实力强大，而且作为万古天骄，自然不会这么轻易的陨落。
<br/>　　他的身躯遭到了毁灭般的打击，元神却是保存了下来，立刻开始逃窜。
<br/>　　他的元神虽然也很强，但是，失去了肉身的保护，即便没有人攻击，他在剑山之上，也撑不了多长的时间，所以要飞快的逃离剑山。
<br/>　　然而，就在此刻，异变陡生。
<br/>　　只见，苏莫身躯崩溃之地，混乱的血肉碎片和散乱的混沌之力，如同一大团浓雾，直接向千幻邪君的元神扑去。
<br/>　　浓雾速度非常之快，比之千幻邪君元神逃离的速度，要快的多。
<br/>　　几乎是眨眼之间，浓雾便扑到了千幻元神之上，将其一口吞没。
<br/>　　“怎么可能？”
<br/>　　一道惊恐无比的吼叫声响起，真是千幻邪君的声音，但是，下一息便戛然而止。
<br/>　　随即，浓雾瞬间浓缩，再次化为了苏莫的身躯。
<br/>　　混沌不灭体！
<br/>　　不死不灭！
<br/>　　即便拓跋流明修为高达四品中级，强大无比，也无法击杀苏莫。
<br/>　　“这是……！”
<br/>　　“混沌不灭体！”
<br/>　　看到这一幕，无论是拓跋流明还是云沐，亦或是韩生、古剑山、公孙嘉，还是后方的其它人，均是心中一震，目瞪口呆。
<br/>　　他们完全没有料到，苏莫居然练成了混沌不灭体。
<br/>　　“你居然修炼的是大混沌五行神诀？”
<br/>　　拓跋流明的脸色，瞬间阴沉如水，因为，除了混沌神帝的正宗道统之外，其它的混沌类的道统，是无法修炼成混沌不灭体的。
<br/>　　“没有人能阻挡我的脚步，你们，也不行！”
<br/>　　苏莫面色冷漠，转头看向拓跋流明和云沐，眸中冷芒闪烁。
<br/>　　言语之间，他在快速炼化千幻邪君的元神，对方被他吞噬，神智已经被抹杀，但是元神还需要一小段时间的炼化。
<br/>　　不过，唯一遗憾的是，对方的身躯崩溃之后，血脉之力被剑山的锋锐之气湮灭，他没有得到对方的虚无玄体。
<br/>　　“是吗？”
<br/>　　拓跋流明眼眸一寒，心中杀机大起，因为，他感受到了威胁。
<br/>　　这威胁来自于苏莫，当然，这并不是说苏莫威胁到了他的安全。
<br/>　　而是，在他看来，苏莫有登上剑山的潜力。
<br/>　　天赋如此妖孽，又拥有混沌不灭体，有登上剑山之巅的可能性。
<br/>　　嘭！
<br/>　　毫不犹豫，拓跋流明抬起脚步，向回走去。
<br/>　　因为，他距离剑山之巅太近，沉声的压力巨大，所以出手的话，所能展现的实力有限。
<br/>　　他只需后退几丈，承受的剑山锋锐之气减弱，他所能发挥的战力，将会大增。
<br/>　　苏莫见此，面无惧色，因为，他吞噬了千幻邪君，已经是后退了一些距离，实力自然也更强了。
<br/>　　云沐和韩生见此，面色阴晴不定，心中急思。
<br/>　　对于韩生而言，他不想现在和拓跋流云交手，因为必败无疑。
<br/>　　但是，他也不能置之不理，任由拓跋流明攻击秦陨。
<br/>　　而对于云沐而言，苏莫是她的威胁，拓跋流明和韩生同样也是她的威胁。
<br/>　　当然，对于她来说，苏莫和韩生，威胁更大。
<br/>　　毕竟，终极神王殿的弟子，是不会允许外人登上剑山之巅的。
<br/>　　而拓跋流明，若是自己登不上剑山，肯定不会阻止别人蹬上，古剑山也是同理。
<br/>　　所以，稍一沉吟，云沐便转头看向韩生，同样出手了。
<br/>　　这一次，云沐不再出剑，而是动用了她的绝技，天罚之力。
<br/>　　云沐的身上，升起神圣的光芒，她的双眸之中，闪过璀璨的精光。
<br/>　　璀璨的精光，如同一道虚空闪电，急速飚射而出，直击韩生。
<br/>　　虚空闪电，大气磅礴，仿佛蕴含着天道意志，恐怖无比。
<br/>　　面对云沐的攻击，韩生自然不会坐以待毙，立刻还击，剑光耀眼，威势毁天灭地。
<br/>　　韩生，乃是终极神王殿毁灭剑宫的弟子，修炼毁灭剑道，攻击力极度强大。
<br/>　　而在同一时间，拓跋流明也同样出手了，一剑斩向苏莫。
<br/>　　这一剑恐怖无比，比之前一掌威力大的多，只见一剑斩出，浩大的剑气形成了一道巨大的风刃，横断虚空，破灭一切。
<br/>　　风刃速度快如疾电，裹挟滔天之力，誓要一击击败苏莫。
<br/>　　苏莫面无表情，面对拓跋流明的攻击，他全力抵挡，元神之力和血脉之力，包括法则之力，尽数催动到了极致。
<br/>　　巨大的混沌剑轮，在法则之力的加持之下，威力暴增，瞬间便与风刃相击在了一起。
<br/>　　轰！
<br/>　　一声震天爆响，混沌剑轮虽然强大，但是，依旧不敌巨大风刃，立刻被斩碎。
<br/>　　强大的风刃，又一次的击在了苏莫的身上。
<br/>　　不过，风刃的威力，也所剩不多，只是将苏莫击退五丈，并没有再次击溃苏莫的肉身。
<br/>　　轰隆隆！
<br/>　　与此同时，韩生的毁灭剑气，硬挡天罚之力，同样掀起震耳欲聋的爆响。
<br/>　　韩生的实力，亦是不如云沐，同样被击退数丈。
<br/>　　而且，韩生的防御不是很强，护体源力差点被虚空闪电破开，险些受伤。
<br/>　　可以说，现在的局面是，苏莫和韩生这两位终极神王殿的弟子，完全的处于下风，是弱势的一方。
<br/>　　苏莫面色凝重无比，眼眸直直的盯着拓跋流明，此人极度强大。
<br/>　　他比对方落后了二十丈左右，承受的剑山压力小的多，但是依旧不是对方的对手。
<br/>　　若是正常的公平一战，他根本挡不住对方一招，修为差距太大了。
<br/>　　“可恶！”
<br/>　　拓跋流明怒了，以他的修为和实力，居然连续两招，都无法击败苏莫，让他心中怒气升腾。
<br/>　　他堂堂四品中级的修为，居然无法打败二品高级修为之人，这简直是天方夜谭。
<br/>　　他抬起脚步，再次向下方走去，只有距离剑山之巅越远，他才能发挥出更强的战力。
<br/>　　“拓跋流明，你解决秦陨，我解决韩生！”云沐朗声娇喝。
<br/>　　拓跋流明闻言，并未回答，他一步步向下走去，不断的接近苏莫，行动已经证明了一切。
<br/>　　最上方，古剑山看到下方发生的一切，心中暗笑，下方众人冲突，对于他来说，自然是好事。
<br/>　　没有人打扰他，他就更有机会登上剑山之巅。
<br/>　　随即，他再次踏出了一步，距离剑山之巅，已经只有八丈有余。
