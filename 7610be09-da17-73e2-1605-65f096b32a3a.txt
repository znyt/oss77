　　神武学府内，真是各种各样的建筑云集，让人眼花撩乱。
<br/>　　各式各样的阁楼、宫殿、院落、简直数之不尽。
<br/>　　而且，建造这些建筑的材料也各不相同，有普通的岩石、有上好的檀木、有精美的白玉，也有闪耀的金属。
<br/>　　总之，这里汇聚了各式各样的建筑，风格迥异，几乎各不相同。
<br/>　　苏莫看的暗暗惊奇，这神武学府的建筑，正是让人奇怪，建造的完全没有章法。
<br/>　　甚至有的地方建筑连绵，非常紧密，一眼望不到尽头，而有的地方却又只有零零散散的几座建筑。
<br/>　　一路上，苏莫看到了不少神武学府的弟子，个个修为都有武帝之境。
<br/>　　有些年轻的武者，看起来只有二十岁左右，居然也是武帝境的强者，当真不凡。
<br/>　　这让苏莫颇为心惊，不愧是传承百万年的至高学府，果然不同凡响。
<br/>　　青年带着苏莫，在神武学府中走动，穿过了一片片连绵的建筑群，不多时便来到了一座大门之前。
<br/>　　这座大门是一座白玉石门，高约十数丈，造型古朴大起，其上刻画着无数繁杂的纹路。
<br/>　　最让人奇怪的是，这座大门只有一个光脱脱的门框，而且耸立在一片荒地之上。
<br/>　　大门的前后左右，都没有任何物体，仿若此门只是一个摆设。
<br/>　　此刻，在大门之前，正伫立着一名年轻的女子。
<br/>　　此女年约二十七八岁许，穿着一身纯白色的长裙，身姿婀娜，凸凹有致。
<br/>　　此女的面容亦是极美，白皙的脸庞上五官非常的精致，琼鼻凤眼，黛画之眉，朱唇殷红，组成一张倾城之颜。
<br/>　　只不过，此女的脸色非常的冷，冰冷如霜，一副生人勿近的模样。
<br/>　　“木导师，就是此人求见火元圣师！”青年指了指苏莫，向白衣女子抱拳说道。
<br/>　　“嗯！”
<br/>　　白衣女子闻言微点螓首，随即瞥了苏莫一眼，淡漠的道：“你跟我来吧！”
<br/>　　言罢，白衣女子一步踏出，瞬间走进了大门之中，而后消失无踪。
<br/>　　“嗯？”苏莫见此，顿时双眉一挑，此女居然不见了？
<br/>　　难道这座大门，是通往一个小世界的入口不成？
<br/>　　苏莫看了一眼带路的青年，目露疑问之色。
<br/>　　“这是通往内府的入口！”青年说道。
<br/>　　苏莫闻言顿时恍然，原来这神武学府，也有内府和外府之分。
<br/>　　随即，他不再犹豫，同样一步踏出，走进了大门之中。
<br/>　　刹那之间，苏莫眼前场景变换，来到了另一个世界之中。
<br/>　　“这是……？”看清眼前的场景，苏莫顿时倒吸了一口凉气。
<br/>　　内府，居然是如此的模样。
<br/>　　此刻，苏莫所在之地，是一片一望无际的星空，辽阔无边，广阔无垠。
<br/>　　而在这片星空之中，正漂浮着数以万计的山峰，壮观无比。
<br/>　　这些山峰均不是很大，小的只有数十丈高，最大的也不过数千丈高。
<br/>　　而这些山峰排列的非常有序，中央的山峰最是高大，外围的山峰呈一道道环形，包围着中央最大的山峰，越是向外围延伸，山峰便越是矮小。
<br/>　　几乎所有的山峰之上，都有精美的建筑群，仿佛每一座山峰之上，都是一片仙宫。
<br/>　　有的山峰之上不仅有武者的身影，还有一只只异兽飞舞。
<br/>　　苏莫怔怔的看着眼前的一幕，他完全没想到，这内府居然如此的壮观，如此的震撼人心！
<br/>　　“跟我来！”白衣女子清冷的声音在苏莫身边响起，随即白衣飘飘，向前方飞去。
<br/>　　苏莫深吸一口气，不敢怠慢，立刻便跟了上去。
<br/>　　苏莫跟着白衣女子，飞驰在虚空之中，掠过一座座山峰，飞向了中央区域。
<br/>　　不多时，两人便飞临到了一座高达三千余丈的山峰之上。
<br/>　　这座山峰位于中央区域，虽然不是最大的一座山峰，但比此山峰更大的山峰，绝对不会超过十座。
<br/>　　山峰上有不少宫殿，白衣女子款步而行，带着苏莫径直来到了中央的一座宫殿之前。
<br/>　　而后，白玉女子脚步不停，推开殿门，带着苏莫走进了宫殿中。
<br/>　　宫殿中极为空旷，只有一名灰袍老者盘膝坐在蒲团上，双眸紧闭，静若磐石。
<br/>　　这名老者身形微胖，脸庞宽大，脸庞上布满了皱纹，满头的白发，颌下还留有一缕雪白的胡须。
<br/>　　“导师，人带来了！”白衣女子走到老者身前，躬身一拜道。
<br/>　　随即，灰袍老者的双眸缓缓的睁了开来，略显浑浊的双眸，仿佛无尽的深渊，深邃浩瀚。
<br/>　　老者的目光落在了苏莫的身上，顿时让得苏莫心中微颤，仿佛自己的一切，都被对方一眼看穿了一般。
<br/>　　他仔细的打量了老者一番，丝毫看不出对方的修为，对方仿佛就是一片汪洋大海，深不可测。
<br/>　　“精神力不错，的确适合炼器大道，难怪柴晋推荐你来此！”老者开口了，声音沧桑而又悠远。
<br/>　　“晚辈拜见前辈！”苏莫立刻抱拳，向对方躬身行礼，同时他心中暗暗震惊，对方居然一眼就看出了他精神力强大。
<br/>　　老者微微颌首，随即淡声问道：“你叫苏莫？”
<br/>　　“正是！”苏莫点头。
<br/>　　“你是否愿意加入神武学府？”老者再次问道。
<br/>　　“晚辈愿意！”苏莫立刻同意，神武学府如此不凡，他当然不会拒绝。
<br/>　　“嗯！”
<br/>　　老者见此再次颌首，道：“你以后，就留在这火元峰吧！好好学习炼器大道！”
<br/>　　“这……！”苏莫闻言一怔，随即面现迟疑之色，他虽然炼器天赋极高，但他的追求并不是炼器，而是武道。
<br/>　　这位火元圣师，明显是一位炼器师，若是他留在这座山峰上，怕不是以后要终日学习炼器了吧！
<br/>　　这不是苏莫的追求，所以他立刻便迟疑了。
<br/>　　“怎么？你不愿意？”老者见苏莫迟疑，顿时便是眉头一皱。
<br/>　　站在一旁的白衣女子，亦是面上带有疑惑之色，难道面前的这个小子，不知道眼前之人的身份？
<br/>　　能跟随火元圣师学习炼器，这是古周星乃至整个古灵星河，无数炼器师根本不敢奢望的梦想。
<br/>　　而面前的这个小子，居然还一副不太愿意的模样？
