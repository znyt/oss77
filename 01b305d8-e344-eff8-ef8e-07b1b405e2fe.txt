　　轰！轰！轰隆隆！！
<br/>　　恐怖的爆响，不断的响起，仿佛无休无止，整片虚空都陷入了彻底的混乱之中。
<br/>　　咆哮的龙力，崩溃的混沌源力，炸开的虚空之力，彼此纠缠在一起，不断的涌动。

<br/>　　苏莫毫无罢手的意思，疯狂的攻击，涛涛剑气绵绵不绝，不断的斩向洪荒祖龙。
<br/>　　洪荒祖龙很强大，手段也是很多，龙力从口中喷薄，化为无数的百丈神龙，迎击苏莫的剑气。
<br/>　　不过，洪荒祖龙终究还是弱了三分，在苏莫巅峰战力爆发，不断的攻击之下，渐渐开始后退。

<br/>　　不仅如此，他连吞噬法则之力，都动用了。
<br/>　　可以说，他此时此刻，是前所未有的强大，一身战力几乎施展到了十二成。
<br/>　　不过，洪荒祖龙是何需人也，即便苏莫疯狂的攻击，也只是落入了下风，短时间之内不会落败。
<br/>　　就这样，连续攻击力接近一刻钟，苏莫的面色彻底的凝重了下来，因为他消耗过大，混沌源力不多了。
<br/>　　他的混沌源力虽然无比的浑厚，是同阶武者源力的数十倍，但是，他之所以能这么强大，是因为动用终极神剑，此剑极为消耗源力。
<br/>　　每一剑所消耗的源力，都是普通攻击的百倍，即便他源力无比的浑厚，也坚持不了多久了。
<br/>　　虽然他身上源石极多，但是，源石是需要炼化的，在战斗之中，根本没有足够的时间去炼化源石。
<br/>　　“必须尽快结束战斗！”
<br/>　　苏莫心中急思，他还有三种元神可以动用，那就是天梦元神、千幻元神、吞噬元神。
<br/>　　而前两种元神，对于战力的加成并不大，唯有吞噬元神，辅助效果比较大。
<br/>　　嗡~！
<br/>　　毫不犹豫，苏莫立刻直接释放出了吞噬元神，庞大的吞噬旋涡，直接悬浮于头顶上空。
<br/>　　浩大的吞噬之力，席卷而出，浩浩荡荡，笼罩方圆万里，将洪荒祖龙笼罩在了其内。
<br/>　　同时催动四种强大元神，苏莫的精神力几乎达到了极致。
<br/>　　果然，吞噬元神一出，效果是极为显著的，强大的吞噬之力，对于同阶的洪荒祖龙，明显有很大的束缚作用。
<br/>　　不仅让洪荒祖龙的动作有所减缓，连洪荒祖龙身上的龙力，都有所流逝。
<br/>　　苏莫大喜，顿时更加疯狂的攻击，体内的混沌源力，如同流水一般急速的流逝。
<br/>　　唰唰唰！！
<br/>　　几乎是在一个眨眼之间，苏莫连斩九十九剑，几乎是倾尽了体内的源力，成败在此一举了。
<br/>　　九十九剑斩出，苏莫体内的源力，只剩下一心半点了，他立刻取出大量的源石，直接吞噬一空，急速的炼化。
<br/>　　这一轮攻击，不知能否成功，他必须尽快炼化源石，补充体内的源力。
<br/>　　若是没有成功，体内的源力得以补充，说不定还有翻盘的机会。
<br/>　　轰隆隆！！
<br/>　　九十九道剑气，组成了一个巨大的剑气风暴，如同宇宙狂风，向庞大的洪荒祖龙袭杀而去。
<br/>　　面对强大的剑气风暴，洪荒祖龙根本无法躲避，其一他身形庞大，其二，他的身形受到了吞噬之力的影响很大。
<br/>　　眼见庞大的剑气风暴来临，洪荒祖龙巨大的眼眸之中，居然留露出极度凝重之色。
<br/>　　昂！
<br/>　　扬天长啸，龙吟震天，只见洪荒祖龙身上彩色的光芒陡然大盛，在它的周身形成了一层密密麻麻的符文。
<br/>　　符文将洪荒祖龙的庞大的身躯，全部包裹在了其内，而后，只见洪荒祖龙龙尾一摆，撕开吞噬之力的束缚，急速向剑气风暴冲了过去。
<br/>　　庞大的龙躯，如同一根巨大的彩色神箭，瞬间便和剑气风暴相击在了一起。
<br/>　　轰隆隆！！
<br/>　　巨响声不断的爆开，现场一片混沌，暴炸的混沌剑气，如同末日的狂风，不断的席卷。
<br/>　　庞大的洪荒祖龙，处在剑气风暴之中，不断的咆哮。
<br/>　　九十九道剑气风暴的威力，何等恐怖，就连洪荒祖龙都承受不住了，全身不断的被撕裂，庞大的身躯不断的扭动。
<br/>　　轰隆隆！！
<br/>　　整个剑气风暴，全部炸开，气浪席卷，缓缓的散开。
<br/>　　而洪荒祖龙，虽然没有被灭杀，但是，其庞大的身躯，已经布满了一道道可怕的剑痕。

<br/>　　苏莫见此，顿时眼眸一凝，随即定晴一看，这西瓜般大小之物，哪里还是洪荒祖龙，分明只是一团血液。

<br/>　　随即，苏莫意念一动，立刻将龙腾从空间戒指之中，强行的拉了出来。
