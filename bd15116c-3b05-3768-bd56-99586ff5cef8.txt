　　苏莫目光转动，扫过巫梵尊者，扫过巫尹尊者，扫过满脸怨毒的巫天御，以及其他四十多名巫族天才。
<br/>　　他狠狠的咬了咬牙，必须要早做决断了，不然必死无疑。
<br/>　　巫梵的威压极强，一直如同擎天巨山般镇压在苏莫的身上，让他倍感压力，长时间下去，必然力有不逮，被对方寻到机会。
<br/>　　至于那巫天御，今日肯定是杀不了了，宏青璇只能待日后再救了。
<br/>　　苏莫本想用巫娴来要挟这两位尊者，让他们命巫天御交出宏青璇，但权衡利弊，他还是放弃了这个想法。
<br/>　　因为，如此的话，这两位尊者必然知道他在意宏青璇，到时候说不定会反过来要挟他，那样的话，就彻底完了！
<br/>　　虽然宏青璇很重要，但和他神图空间中苍穹神宫百万人的性命相比，孰轻孰重，苏莫还是清楚的。
<br/>　　意念一动，苏莫立刻向身边的秦不死传过去一道意念，而后两人一起，开始向侧翻飞退。
<br/>　　两人的速度并不是多块，但也是瞬息百里，并时刻注意着巫梵和巫尹两位尊者。
<br/>　　“你们逃不掉的！”巫梵冷喝一声，虚空踏步，瞬间便追上了苏莫两人。
<br/>　　老妪巫尹速度也是奇快无比，身形一闪，刹那间便临近了苏莫和秦不死。
<br/>　　苏莫见此，顿时左手一探，直接抓在了巫娴的肩膀上。
<br/>　　咔嚓！
<br/>　　啊！
<br/>　　而后，清脆的骨骼碎裂声响起，伴随着巫娴的一声痛呼。
<br/>　　巫娴肩膀的骨骼被瞬间捏碎，从未吃过苦头的她，立刻痛的俏脸都扭曲了起来。
<br/>　　与此同时，秦不死手中劲力一吐，他手中的两名人质，顿时口中鲜血狂喷。
<br/>　　“什么？”巫梵和巫尹两位尊者见此，顿时面色一变。
<br/>　　“别追过来，不然我就废了她的修为！”苏莫厉声大喝，而后，他和秦不死的速度，骤然暴增十倍，瞬息千里，急速暴退。
<br/>　　“找死！”巫梵勃然大怒，他怎么可能会让苏莫两人逃走？
<br/>　　虽然他没追上去，但却是袖袍一挥，一道浩大的气浪顿时汹涌而出。
<br/>　　气浪奇快无比，肉眼难辨，刹那间便卷过虚空，形成了一个硕大的气罩，将苏莫两人笼罩在了其内。
<br/>　　“不好！”苏莫和秦不死立刻停下了身形，两人俱是面色大变，身形被困住，那他们就危险了！
<br/>　　“你们逃不掉，老夫再给你们最好一次机会，放了他们三人。否则杀无赦！”巫梵怒声大喝，音如天雷滚滚，震动九天十地。
<br/>　　霸道的声音无孔不入，直接钻进了苏莫两人的耳膜之中。
<br/>　　呃！呃！
<br/>　　苏莫和秦不死，被震得闷哼一声，口中再次喷出鲜血。
<br/>　　在这一声暴喝之下，秦不死思维停歇，眸光涣散，大脑已经出现了短暂的空白，巫梵和巫尹若是出手，必能轻易击杀秦不死。
<br/>　　但是，两人都没有妄动，因为在巫梵一声大喝之下，苏莫虽然也被震得口吐鲜血，但其大手却捏的更近了，几乎要捏碎了巫娴的脖颈。
<br/>　　其实，以巫梵的实力，一声怒喝，音波之力完全可以在不伤害巫娴三人的情况下，击杀苏莫和秦不死，但他不敢。
<br/>　　因为他担心苏莫在死亡的那一刻，会拉巫娴一起陪葬。
<br/>　　巫梵早已看出了苏莫的不简单，即便是面对他的威压，面对他蕴含劲力的音波，眸中都是一片清明。
<br/>　　他已经尝试了三次，每一次用音波将苏莫和秦不死震伤，苏莫都是清明无比，并没有出现像秦不死那种眸光涣散，大脑空白的情况。
<br/>　　巫梵知道，这是因为苏莫的精神力强大，远超同阶武者数倍不止。
<br/>　　“是你们逼我的！”突然，苏莫面现狠厉之色，怒喝一声，一掌狠狠的拍在了巫娴的后背上。
<br/>　　嘭！
<br/>　　一声闷响，即便苏莫没有用多少的力量，也是瞬间将巫娴的内脏震碎了。
<br/>　　哇！
<br/>　　巫娴顿时惨叫一声，口中鲜血狂喷，鲜血中更是混杂着大量的内脏残渣。
<br/>　　“立刻撤去气罩，不然我立刻杀了她！”苏莫怒声大喝，眸中尽是疯狂之色，让人不敢怀疑他的决心。
<br/>　　“孽畜……！”巫梵面上怒火冲天，手掌一抬，就想一掌将苏莫拍死。
<br/>　　他堂堂一名武尊境武者，居然连番被一个武皇境的蝼蚁威胁，几乎怒火冲天了。
<br/>　　不过，巫梵刚刚抬起手臂，一直枯瘦的手掌，顿时拉住了他。
<br/>　　“不可！”巫尹尊者顿时向巫梵摇了摇头，事关巫娴的性命，千万不能鲁莽。
<br/>　　“我告诉你们，她们的识海中都有我二人留下的意念玄力，只要我们一死，意念玄力就会立刻爆炸，她们必死无疑！”苏莫大声喝道。
<br/>　　“什么？”巫梵和巫尹闻言，顿时面色微变，随即脸色彻底的难看了起来。
<br/>　　“梵尊者、尹尊者，他们并未说谎！”
<br/>　　“他们的确留下了意念玄力！”
<br/>　　“我们的识海中都有！”
<br/>　　远处传来几声呼声，正是之前裹挟在秦不死身边的几名人质。
<br/>　　几人被震开之后，立刻便被其他的巫族天才救起，解除了封印，正在努力祛除识海中的玄力意念。
<br/>　　这几人识海中的意念玄力，均是秦不死所留，秦不死担心彻底激怒两名巫族尊者，便也没有引爆这些意念玄力。
<br/>　　“放我们两人离开，如若不然，她们三人与我们同归于尽！”
<br/>　　苏莫高高举起了口中鲜血横流的巫娴，继续道：“我们两个就是两个土著，贱命两条，能有高贵的巫族美女天才陪葬，也不失为一件美事！”
<br/>　　“不错！大不了一死，能有三名巫族天才陪葬，黄泉路上也不会孤单！”秦不死回过神来，立刻大声喝道，面现决绝之色。
<br/>　　“可恶！”巫梵尊者闻言，胸膛起伏，气的简直要控制不住心中的怒火了。
<br/>　　“梵……梵尊者、尹尊者，让……让他们离开！”就在此时，巫娴开口了，断断续续的说道。
<br/>　　巫娴无法动用玄力，身受重伤，虚弱无比，她的脖颈被苏莫死死的捏着，声如蚊蝇，简直是从牙缝中挤出来的。
<br/>　　巫娴的眼眸中露出一丝恐惧之色，现在她真的怕了，她怕自己真的被苏莫杀死。
<br/>　　正如苏莫所言，她是何等高贵的身份，支族大祭司的曾孙女，若是死在这个土著的手中，那真是太不值了。
<br/>　　所以，她开口让两位尊者放苏莫两人离开，只要两人无生命危险，暂时应该不会杀她。
<br/>　　而且，她巫族强者如云，高手如雨，日后有准备的情况下，击杀苏莫将她救出，是轻而易举之事。
<br/>　　“这……！”巫梵闻言顿时眉头一皱，苏莫和秦不死杀死他这么多族中天才，如何能放两人离开！
<br/>　　“梵尊者！”
<br/>　　这时，巫尹尊者瞥了巫梵一眼，微微点了点头，眸中闪过一丝莫名的光芒。
<br/>　　巫梵见此顿时明白了对方的意思，略一沉吟，便点了点头。
<br/>　　唰！
<br/>　　随即，巫梵手臂一挥，笼罩苏莫两人的硕大气罩，顿时随风消散。
<br/>　　“老夫放你们离开！但她们三人若有事，你们必死无疑！”巫梵冷声说道。
<br/>　　嗖！嗖！
<br/>　　气罩乍一撤去，苏莫和秦不死两人顿时心中大喜，毫不犹豫，两人立刻爆发出了最快的速度，身形如电，瞬息千里，急速逃离。
<br/>　　几乎是眨眼之间，苏莫和秦不死两人，便消失在了无尽的虚空尽头。
