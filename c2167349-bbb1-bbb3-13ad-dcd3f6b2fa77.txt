　　珍宝港，位于太平洋中部的夏威夷群岛，是米国控制太平洋的海上大本营。
<br/>　　同时也是第三岛链的核心，前沿部署兵力的大后方，米国本土前出防御的最后一道堡垒！
<br/>　　当年二战时期，岛国与米国开战，便是选择偷袭珍宝港海军基地，让米国海军，瞬间陷入了瘫痪状态。
<br/>　　这也足以说明，此处对于米国的重要性！
<br/>　　“小叶将军该不会是想……就这么硬闯米国海军大本营吧？”
<br/>　　“这珍宝岛海军基地，可是米国太平洋战区指挥中心！岛上驻扎着上万米军，还是数十艘舰艇的母港！”
<br/>　　“米国可不同于岛国，他们的海军实力，绝对不是个人武力所能抵挡的！”
<br/>　　……
<br/>　　与此同时，
<br/>　　不光华夏军方众高层，对于叶尘的这番举动个个惊呆。
<br/>　　就连米国军方，也在密切关注着叶尘的一举一动，同样百思不得其解。
<br/>　　“这家伙想干嘛？难道他还真想，以一己之力，对抗我们米国整个太平洋战区？”
<br/>　　“他这是自投罗网！自寻死路！”
<br/>　　“传我的命令下去，通知战舰拦截，不得让此人靠近珍宝港半步！”
<br/>　　……
<br/>　　叶尘带着苏珊一路疾驰，来到离珍宝岛还有百里左右的地方，前方海域之上，突然出现一排战列舰，一字排开，拦住了他们的去路。
<br/>　　“叶先生，前面就是珍宝岛，是米国海军在太平洋战区的总指挥中心，您确定要这么冲过去吗？他们可不会因为我这个人质，就不对你开火！”
<br/>　　苏珊见叶尘丝毫没有回头的意思，忍不住再次开口向叶尘提醒道。
<br/>　　不料，叶尘一言不发，忽然屈指一弹，一道赤红色火焰，自叶尘的指尖发出，瞬间没入苏珊的体内。
<br/>　　苏珊猛然感觉到体内，多出了一股极度危险能量，似乎随时都能将她整个人吞噬，脸色不由得大变，
<br/>　　“你……你对我做了什么？”
<br/>　　叶尘淡淡道：
<br/>　　“一些防止你逃跑的小手段罢了！乖乖在这里等着我，等我灭了你们海上军事基地，你给还得给我继续带路！”
<br/>　　说完这话，也不管苏珊同不同意，叶尘直接朝前方那五艘战列舰冲了过去。
<br/>　　苏珊听到这话，先是微微一愣，继而露出一抹喜色，望着那道狂奔而去的人影，嘴角不由得露出一抹幸灾乐祸的神情，低声道：
<br/>　　‘难道他还真想，一个人单挑整个米国海上军事基地？简直不知死活！’
<br/>　　也难怪苏珊会不看好叶尘。
<br/>　　整个珍宝岛上，有数万名装备精良的米国海军，三艘航空母舰，以及数十艘军舰，上百架战斗机！
<br/>　　再加上，米国军队装备的，都是世界上最为先进的武器。
<br/>　　在她看来，别说灭掉整个珍宝岛海军基地，便是眼前的这五艘战列舰，就足以将宙斯那个级别的强者，打的抱头鼠窜。
<br/>　　而叶尘，就算实力要强过宙斯不少，却还是远远不足以对抗整个海军基地。
<br/>　　……
<br/>　　“开火！！！”
<br/>　　随着米国海军指挥部一声令下，那一排共五艘战列舰，几乎同时开炮！
<br/>　　砰！砰！砰！砰！
<br/>　　随着一道道轰鸣声响起，漫天的炮弹，划过寂静的夜空，朝叶尘两人的方向，纷纷爆射而来！
<br/>　　轰！轰！轰！
<br/>　　紧接着，
<br/>　　那一枚枚炮弹，犹如漫天的雨点，自上空砸落，然后爆炸开来。
<br/>　　看到这一幕，后方的苏珊，不由得露出一抹轻蔑的冷笑，
<br/>　　“虽然战列舰上装备的普通炮弹，应该还不足对你构成威胁，可是这长千上万枚炮弹一起轰炸，就算是你是钢铁人也扛不住吧！”
<br/>　　然而，苏珊自言自语的声音，才刚刚落下，接下来看到的一幕，顿时让她目瞪口呆。
<br/>　　“什么！”
<br/>　　只见，在那漫天炮火之中，忽然发出一道璀璨的金芒！
<br/>　　继而，苏珊便看到了她永生难忘的一幕。
<br/>　　只见，一身金光的叶尘，在漫天的炮火之中，踏空而行，竟然一步步朝那五艘战列舰逼近！
<br/>　　而任凭那些炮弹如何轰炸，全部都被叶尘身上的那道金色护罩弹开，根本无法靠近他们十米之内！
<br/>　　不一会的功夫，叶尘已经逼近到，那五艘战列舰十里之内。
<br/>　　“今日，就让天下人，见识一下我叶狂仙真正的力量！”
<br/>　　叶尘忽然朗声开口，声音犹如春雷滚滚，在辽阔的海面上，远远回荡开来，清晰无比的传到那五艘战列舰上，每一个米国大兵的耳朵之中！
<br/>　　虽然，很多人并不懂华夏语，可是却能感受到叶尘语气中的强大煞气！
<br/>　　“他想做什么？”
<br/>　　“那是什么东西！”
<br/>　　“他竟然真的冲过来了！”
<br/>　　……
<br/>　　在众人无比惊骇的目光之下，叶尘的身上，猛然发出一道妖异的紫色光芒！
<br/>　　继而，整个人的速度，也猛然暴涨，瞬间便闯过漫天的炮弹，冲到五艘战列舰千米之内！
<br/>　　“快！拦住他！绝对不能让他冲过来！”
<br/>　　随着米国海军指挥部一声令下，十多架早已经准备就绪的F35战斗机，也加入了战局之中，自上千米的空中发射导弹，朝叶尘攻击而来，想要阻止他朝战列舰靠近，
<br/>　　“哼！”
<br/>　　叶尘顿时冷哼一声，忽然身体一沉，下落到海面之上，从容躲开了那些导弹袭击。
<br/>　　那F35战斗机中的驾驶员们，顿时一个个精神一震，当即操控着战斗机，也俯冲了下去，想要对叶尘进行追击。
<br/>　　然而，谁也没有想到，叶尘落入海面上之后，猛然在海面上一踩，
<br/>　　彭！
<br/>　　海面之上，顿时出现了一个，深达数十米的深坑！
<br/>　　瞬间掀起了一道道高达上百米的海浪！
<br/>　　而与此同时，叶尘借助这一踩之力，整个人已经腾空而去，瞬间达到上千米的高空！
<br/>　　“死！”
<br/>　　唰！唰！唰！
<br/>　　随着叶尘一声暴喝，一道长达数百米的紫色剑芒，在空中留下道道残影。
<br/>　　而片刻之后，那十几架F35战斗机，竟然全部被那道紫色剑芒击毁，无一幸存！
<br/>　　“什么！”
<br/>　　无论是后方的苏珊，还是米国海军基地指挥中心的众高层，都是全都一个个目瞪口呆。
<br/>　　那可是十几架地球上最先进的战斗机，竟然被对方像揉捏玩具一样，瞬间击溃！
