　　第三城之内，陷入了一片寂静之中，苏莫的狠辣，再次让众人震惊，
<br/>　　这真的是不计后果，完全的不留活口啊！
<br/>　　若是他们，轻易不会下杀手，毕竟这里的每个人，背后都有强大的势力支撑。
<br/>　　也就混沌神朝、亘古魔国，天星神朝、天龙神朝等七大神朝之人，敢无惧一切的杀人，但也不会这么肆无忌惮。
<br/>　　“好了，现在一切都处理完毕，接下来，我们就是要团结一致，对付其它八座城池！”苏莫身形缓缓从空中的落了下来，环视众人，朗声说道。
<br/>　　他知道这些人，肯定不会真心团结，但是，还是可以一用的。
<br/>　　“苏莫，我们现在当务之急，是搞清楚每座城池之中是哪些人？”公孙长绝率先开口道，这每座城池，都有无形的阵法之力，他们无法用神识窥探到其它城池之中。
<br/>　　“这是自然！”
<br/>　　苏莫点了点头，略一沉吟，道：“我准备派人分头行动，前去监视每一座城池，看看每座城池之中有哪些人！”
<br/>　　“苏莫，这不妥！”龙双双摇了摇头，白净的脸庞上露出凝重之色，道：“如果派太多的人出去，一旦被擒拿，我们的底细也就泄露了！”
<br/>　　“的确，我们不清楚其它八座城池的情况，其它人同样不知道我们的情况！”公孙长绝道。
<br/>　　“苏莫，我们应该派出一位擅长隐匿之人，独自前去查探！”亘古魔国的魔野声音洪亮的说道。
<br/>　　苏莫闻言，微微的点了点头，三人之言在理，但是，这样也有不妥。
<br/>　　因为，只拍一人的话，带回来的消息不一定真实。
<br/>　　比如，让魔野出去查探消息，对方肯定会暗中帮助古魔太子，带回来有误导的消息。
<br/>　　只有他信任之人，才能去执行这个任务，比如龙双双。
<br/>　　但是，若是派龙双双出去，别人发现了龙双双，也可能猜到他在第三城之中。
<br/>　　“有没有暗影族之人？”苏莫环视众人，朗声问道。
<br/>　　众人纷纷环视众人，此地并没有暗影族之人，暗影族掌控天池八十一个界域之一，算是天池八十一个超级势力之中，比较强大的存在，但是，此地并没有暗影族之人。
<br/>　　“公孙兄，不如你亲自出去查探如何？”苏莫看向公孙长绝，只有对方出去查看，才能保证消息的真实性。
<br/>　　因为对方，不会帮助任何一个神朝，只会帮助自己。
<br/>　　而且，若是其它永恒神宫的弟子，知晓了公孙长绝在第三城，说不定还会暗中帮助公孙长绝。
<br/>　　“我去？”公孙长绝闻言一怔，这种跑腿的事情，他是不想做的。
<br/>　　不过，他略一沉吟，也考虑到了苏莫的想法，便点了点头，道：好吧，我去！”
<br/>　　言罢，公孙长绝的身形，瞬间就消失无踪，除了其适才所在之地，空间微微波动了一下之外，没有任何的征兆。
<br/>　　“嗯？”苏莫见此，顿时心中一惊，大感震惊。
<br/>　　空间法则！
<br/>　　这公孙长绝，居然精通空间法则，而且造诣还不敌，这里的空间应该都有阵法之力加固，比外界还要凝实，对方还能如此，便可见一斑。
<br/>　　“苏莫，公孙长绝不仅精通空间法则，还精通时间法则！”龙双双站在苏莫的身边，见苏莫面露惊讶之色，提醒道。
<br/>　　这公孙长绝的实力，在同阶之中，正面一战之下，或许不如古魔太子、霸云公子等人，但绝对是最难对付的人，没有之一。
<br/>　　“空间法则和时间法则！”苏莫闻言，面露凝重之色，之前，他一直没将公孙长绝放在心上。
<br/>　　在他看来，本来修为只有六品高级的公孙长绝，实力肯定不如古魔太子等人，并没有任何的威胁，完全无法与他争锋。
<br/>　　但是，现在看来，并不是那么回事，就算第一个环节胜利了，第二个环节也要小心此人了。
<br/>　　也对！
<br/>　　公孙长绝是永恒神宫之人，而永恒神宫是时空界域的霸主，当初时空至尊所创建的势力。
<br/>　　这公孙长绝作为永恒神宫的万古天骄，参悟空间法则和时间法则，也在情理之中。
<br/>　　想必，不止是公孙长绝，其它的永恒神宫弟子，多多少少都精通一些时间法则或者是空间法则。
<br/>　　深深的吸了口气，苏莫大感这次千选战的困难，但是，再困难他都不能输。
<br/>　　“各位，你们查看一下这第三城，看看有没有护城阵法！”苏莫环视众人，朗声吩咐道。
<br/>　　“嗯！”
<br/>　　“好！”
<br/>　　众人纷纷点头，随即，便有一部分人在城中开始查看了起来。
<br/>　　这第三城，果然有护城阵法，在一处城墙下竖立着一块八尺高的石碑，就是控制阵法的主阵基。
<br/>　　“打开阵法！”
<br/>　　在苏莫的吩咐之下，众人立刻打开了护城大阵，一个土黄色的阵法光罩，在第三城上空升起。
<br/>　　有了护城大阵，就不担心别人的查探了，毕竟，有些人手段高明，说不定偷偷潜入进来，他们都无法发觉。
<br/>　　随后，众人便纷纷在城中空地上盘坐了下来，等待公孙长绝回来。
<br/>　　这一等，就是一个多时辰的时间。
<br/>　　一个多时辰之后，第三城的护城大阵，突然剧烈颤动了起来。
<br/>　　“是我！”
<br/>　　公孙长绝的声音，通过护城的阵法，传递到了城池之中。
<br/>　　不需要苏莫吩咐，立刻有人稍稍打开了护城大阵，人影一闪，公孙长绝出现在了苏莫的身前。
<br/>　　“公孙兄，怎么样？”苏莫急忙起身，面色郑重的问道，同时，他心中暗暗震惊，这公孙长绝在空间法则和时间法则的帮助之下，隐匿的本事比暗影族还要厉害。
<br/>　　所有人都聚拢了过来，环绕着公孙长绝，静等对方告知情况。
<br/>　　“并不乐观！”
<br/>　　公孙长绝微微摇了摇头，道：“我将八座城池都查探了一遍，第一城也开启了护城大阵，我无法进入，也没有发现其内有人出来，情况不明；第二城在内战……”
<br/>　　“都有什么人？”苏莫迫不及待的问道，第二城在内战，要么就是没有特别厉害的人物，要么就是厉害的人物太多，导致无法齐心，在争夺统领之权。
