<br/>　　“神剑山！”绫裳道。
<br/>　　“神剑山？”苏莫闻言疑惑，神剑山算什么至宝？
<br/>　　“神剑山，是终极剑尊留下的终极神剑所化，蕴含无边剑意，是剑修的至宝，你进入神剑山内部，参悟终极神剑的意志，对你有好处。”绫裳道。
<br/>　　“原来如此！”苏莫恍然。
<br/>　　“带紫霄神王归来，再医治芊寻月师尊！”绫裳道。
<br/>　　“好！”苏莫点头，毫不犹豫的答应了，这种好事，他自然不会拒绝。
<br/>　　况且，紫霄神王不在，他只能等待了。
<br/>　　绫裳略一沉吟，又问道：“帝一魂是否知晓你来到了神域？”
<br/>　　“已经知道了，估计目前在到处寻我！”苏莫叹了口气，若不是帝一魂，他哪里会如此艰难！
<br/>　　“那你就留在终极神王殿吧！这里足够安全，我会为你消除一切痕迹！”绫裳告诫道。
<br/>　　只有在终极神王殿，苏莫才能有最好的隐藏，因为，她能消除一切苏莫在此的痕迹。
<br/>　　苏莫轻轻的点了点头，能留在终极神王殿，自然更好。
<br/>　　他资源充足，只要有个安稳的地方，他的修为就能快速精进。
<br/>　　至于时空之城，还是过段时间再去吧，那里也不太安全，很容易被帝一魂找到。
<br/>　　“你需要改头换面，并且改名换姓！”绫裳又道。
<br/>　　“这个简单！”苏莫微微一笑，随即立刻动用了天幻神丹。
<br/>　　在绫裳的眼中，苏莫立刻变成了一位身形消瘦，面目狭长的青年。
<br/>　　“你居然有如此高深的隐藏之术！”绫裳惊讶，难怪苏莫能活到现在，这改头换面之术，连她都看不穿。
<br/>　　“以后，我叫秦陨！”苏莫道。
<br/>　　“嗯，以后你就是终极神王殿绫裳剑宫的弟子。”绫裳点了点头，随后，她又向苏莫讲述了一些终极神王殿的殿规，以及神剑山的玄妙，最后便命人为苏莫安排一切。
<br/>　　……
<br/>　　一座普通的阁楼中，苏莫一个人端坐，这是绫裳为他安排的弟子居所，并没有什么破例，和其他弟子一样。
<br/>　　“神剑山！”
<br/>　　苏莫低声自喃，这的确是个好地方，现在，他是终极神王殿的弟子，自然不能错过这个机会。
<br/>　　少倾，他起身离开了阁楼。
<br/>　　绫裳剑宫，并不是很大，走在宫中的小道之上，也看不见几个人影，显得寂静无比。
<br/>　　不多时，苏莫走到了一座大殿之中，到了这里，才看到络绎不绝的弟子。
<br/>　　殿中，有一个高达一丈有余的门户，乃是由刺目的白光组成，不时的有一个个弟子，都光门之中进出。
<br/>　　毫不犹豫，苏莫走进了光门之中。
<br/>　　霎时之间，斗转星移，他从光门内走了出来，但是眼前却是另一番场景。
<br/>　　这是一片浩大的虚空，白茫茫一片，虚空之中，好似有无数剑影飞舞，一股浩大沧桑的剑之威势，在这片空间中浩荡。
<br/>　　站在此地，仿佛是置身于剑冢之中，只有一种剑意在升华。
<br/>　　“这就是神剑山内部吗？”苏莫低声自喃，眼眸扫视四方，在他目力所及之处，发现了数百上千人。
<br/>　　这些人分散在各地，或盘膝而坐，或在练习剑招，俱是在各自的修炼。
<br/>　　有些人修炼的剑招极为强大，不断的发出震耳欲聋的轰响。
<br/>　　随即，苏莫身形一闪，向远处飞去。
<br/>　　这里空间极大，苏莫飞行了片刻之后，也没有飞到尽头。
<br/>　　不过，周围已经看不到任何人了，属于决对的安静之地。
<br/>　　随即，苏莫便盘膝坐了下来，感受这无穷无尽的剑道意志。
<br/>　　乍一闭上眼眸，沉下心神，他就感觉自己被万千神剑包裹住了。
<br/>　　这一刻，他的心跳突然加快了起来，心脏之处，更是散发出淡淡的荧光。
<br/>　　这是剑心，苏莫当初了吞噬了天心剑客的肉身，虽然没有太大的变化，但是在此地，剑心的效用，便被彻底激发了出来。
<br/>　　“剑道……终极剑道……！”苏莫心中沉吟，神念融入了这片天地，仔细感受这冥冥之中的剑道意志。
<br/>　　在他的精神感知之中，身前仿佛有万千剑影舞动，每一道剑影都独一无二，散发出不同的气势。
<br/>　　按照绫裳所言，终极剑道，是剑道的极致，包含了所有的剑之大道。

<br/>　　苏莫不求自己参悟终极剑道，他只求自己能找到属于他的剑道。
<br/>　　因为，在他看来，剑道哪有终极，只有适合他的剑道，才是最强的剑道。
<br/>　　时间悄然流逝，苏莫如同石化了一般，已经忘却了时间的流逝。
<br/>　　一晃眼，就是一年过去了。
<br/>　　越是感知此地的剑道意志，苏莫对于剑，便了解的越发深刻。

<br/>　　“难道所谓的终极剑道，就是包含了一起属性？亦或者说是融入了万千法则？”
<br/>　　苏莫呢喃自语，他感觉所谓的终极剑道，就是包含了这一切。
<br/>　　但是，这又好像不对，怎么可能有人，能练成所有的属性，以及参悟万千法则！
<br/>　　这根本就是不可能的事情，就算比他妖孽一百倍，也做不到这一点。
<br/>　　那么，终极剑尊，是如何修炼的终极剑道？
<br/>　　“不对，是意志，这意志来自于意境，意境发自于心灵！”

<br/>　　如何将心中的意境，转接到招式之中，这是至关重要的一点。
<br/>　　他试着心中生出意境，但是一击剑指击出，攻击并没有丝毫的变化。
<br/>　　思索了良久，苏莫也找不到其中的关键，无法把握住其中的玄妙。
<br/>　　“算了，终极剑道不急于一时，先摸索出自己的剑道。”

<br/>　　这一点，要比参悟出终极剑道，简单的多，只不过也要花费一些时间。

<br/>　　苏莫面带喜色，随即立刻闭上眼眸，再次的参悟了起来。
<br/>　　不过，仅仅是片刻的功夫，一道轻灵的声音，在他的耳畔响起。
<br/>　　“苏莫，两年了，紫霄神王回来了！”
