　　“谢兄，我准备走了，你是否和我一起离开？”苏莫低声问向身边的谢彬。
<br/>　　现在经过两天的拍卖，已经时而有人离开了。
<br/>　　所以，他现在离开的话，也不会太引人注意了。
<br/>　　“苏兄，现在拍卖会的重宝都没有出现呢！这么早就离开？”谢彬面带迟疑之色，不见识拍卖会的压轴宝物，他哪里甘心离开。
<br/>　　“不等了，若是你不想离开，那我就先走了！”苏莫摇了摇头道，重宝在拍卖会上一般都是在后期，作为压轴的存在。
<br/>　　他现在哪有心思等到最后啊，必须赶在巫人王之前离开。
<br/>　　“这……那苏兄你就先走吧！”谢彬略一沉吟，便点了点头，这种拍卖会数年难得一见，他不可能能现在就离开！
<br/>　　苏莫叹了口气，随即便长身而起，走下了坐席区域，大步向拍卖场的大门走去。
<br/>　　苏莫的离开，引起不少人的注意，因为他之前出手太阔绰了。
<br/>　　不过，注意到他的人也没有太过在意，提前离场再正常不过了。
<br/>　　不过，万事都有例外，就在苏莫即将走到拍卖场大门之时，一道冷冷的声音突然响起。
<br/>　　“这不是真圣榜第一的苏莫吗？为何这么着急离开？”
<br/>　　这声音的音量很高，仿佛是故意让所有人听到一般，而开口之人，自然便是尤天韩。
<br/>　　只见尤天韩嘴角噙着一丝冷笑，眸中尽是戏谑的笑意，他就是要整苏莫。
<br/>　　苏莫不是隐藏本来容貌吗？那他就当场揭穿。
<br/>　　苏莫不是被排在真圣榜第一吗？有这个资格吗？
<br/>　　虽然所有人都没有异议了，但肯定很多人心中都不是很服气，只要揭穿苏莫的身份，想必会很精彩吧！
<br/>　　这可是真圣榜第一啊！只要打败了苏莫，那不就名震天下了？取而代之了？
<br/>　　尤天韩话音一落，本来正热火朝天进行拍卖的拍卖场，突然的安静了下来。
<br/>　　随即，所有人均是目光转动，落在了苏莫的身上。
<br/>　　这是苏莫？
<br/>　　真圣榜第一的苏莫？
<br/>　　苏莫居然在拍卖场中？
<br/>　　所有人都是有些惊疑，眼前的人和苏莫并不相像，当然也有可能是改变了容貌。
<br/>　　“苏莫？”巫人王亦是神情一怔，随即阴冷的目光瞬间落在了苏莫的身上，眸中精光顿时暴涨。
<br/>　　不过，他并不能立刻确定这就是苏莫，因为他以前接触的只是巫尘，从未接触过苏莫，也就只在刚才的影像中，才直观的认识了苏莫。
<br/>　　但是，他灵识一探之下，果然发现此人的气息，和苏莫一模一样。
<br/>　　此刻，正快要走出拍卖场的苏莫，闻听尤天韩的话，顿时心中咯噔一声，暗道不好。
<br/>　　随即，他脚步不停，一步跨出，毫不犹豫的瞬间冲出了拍卖场。
<br/>　　“站住！”
<br/>　　就在此时，一声暴喝惊天动地，只见巫人王身形如一道黑色的闪电，同样瞬间冲出了拍卖场。
<br/>　　嗖！
<br/>　　苏莫的速度很快，冲出拍卖场之后，立刻身形拔天而起，直刺苍穹，急速逃离。
<br/>　　他不敢有丝毫的停留，现在的巫人王，可不是上次与他分身一战之时，那种修为遭到压制的状态，而是巅峰状态。
<br/>　　虽然对方的天玄武体血脉，被他吞噬了近八成，但也不是他能够抗衡的存下。
<br/>　　对方不仅是武帝境九重修为，还是战力强大的天才，绝对不比一般的武尊境一重的强者弱。
<br/>　　“哪里走！”
<br/>　　暴喝声再次的响起，巫人王的速度，比之苏莫更加的快速，紧追在苏莫的身后。
<br/>　　而后，巫人王直接展开了攻击，一拳暴轰而出。
<br/>　　昂！
<br/>　　霎时之间，龙吟震天，耀眼的紫色拳茫化为了庞大的紫色巨龙，咆哮着向苏莫飞扑而去。
<br/>　　这一击，强大到极致，庞大的巨龙横贯天地，龙威震天，磅礴浩瀚的劲力如同爆炸的火山，简直威不可挡。
<br/>　　“不好！”
<br/>　　苏莫见此顿时心中大惊，若是被对方缠上，他怕是再也走不掉了。
<br/>　　瞬息之间，他立刻催动了‘隐身战魂’，一股无形的气流顿时将他全身包裹，而后他的身形突兀的消失在了半空中。
<br/>　　轰隆隆！！
<br/>　　庞大的巨龙撕裂长空，暴击十万里，最后才自行崩溃消失，连苏莫的毛都没有碰到一根。
<br/>　　“人呢？”巫人王满脸疑惑之色，他转头看向四方，灵识扫荡方圆数万里，根本就没有苏莫的影子。
<br/>　　“怎么可能？”
<br/>　　巫人王顿时脸色铁青，一个大活人凭空消失了，这不是扯淡吗？
<br/>　　他也不傻，立刻便猜测到，苏莫可能是运用了某种强大的秘法，隐藏住了身形。
<br/>　　“可恶！”巫人王气的怒吼连连，好不容易碰到了苏莫，居然还让对方跑了！
<br/>　　此时，拍卖场中很多人跟了出来，远远的观看。
<br/>　　见到苏莫凭空消失，所有人都是一脸的惊讶，这苏莫果然不凡啊！
<br/>　　更让众人感到惊讶的是，这巫族青年，居然和苏莫有深仇大恨，上来便下杀手。
<br/>　　尤天韩也出来了，更是一脸的懵逼，他只是想整一整苏莫，可没想到巫人王和苏莫有仇啊！
<br/>　　拍卖场中，玄净尊者身形不动，目光却仿佛是洞穿了层层虚空。
<br/>　　“真是不简单！”
<br/>　　少倾，玄净尊者点了点头，他已经发现苏莫隐藏的身形，正在急速逃走，眨眼间便消失在了他的感知之中。
<br/>　　玄净尊者也认识巫人王，知道对方是巫族天冥星一脉的顶级天才，有不弱于真圣榜上天才的战力，武帝境九重的修为，居然丝毫无法拿下苏莫！
<br/>　　本来，他们也是准备将巫人王排进真圣榜，但是后来听说，巫人王在天冥星巫族的少司选拔中，败给了同阶的巫尘，所以便没有让其上榜。
<br/>　　至于那位巫尘，据说是被天冥星一脉的巫族大祭司抹杀了，自然也就没有上榜。
<br/>　　虚空中。
<br/>　　巫人王铁青着脸，略一沉吟，便身形撕裂长空，急速离去，既然苏莫逃走了，他自然要立刻行动起来。
<br/>　　而后，观看的众人纷纷返回拍卖场，虽然所有人都很好奇此事，但拍卖会还要继续进行。
<br/>　　尤天韩眸中精光闪烁，随即冷笑一声，他没有再回拍卖场，而是身形一闪，向着巫人王离去的方向而去。
