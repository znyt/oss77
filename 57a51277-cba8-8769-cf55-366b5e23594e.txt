        &lt;h3&gt;第三卷第二百五十九章  潘青松见张函&lt;/h3&gt;
<br/>
<br/>        江俞轩送的资料，其实张函也有掌握，只是张函觉得还不到时机，所以他迟迟都没有动，在他的心目中并没有把夏颜当回事，没有想到夏颜和冯睿纠缠到一起，还攀上了陆家。
<br/>
<br/>        看着江俞轩送来的资料，他心里叹了口气，江俞轩都开始行动了，他也只好提前行动。
<br/>
<br/>        西城这几年的风气不好，发展中的社会都会是有利有弊，这似乎是一种特性，没有谁能逃得过这种劫数。
<br/>
<br/>        张函一路行来，见证了社会发展的变迁，也看透了世间炎凉、官场的复杂，有些事情不是凭一己之力就可以做到的，好的是，现在已经步入正轨，一起都在向好的方向发展，要不然他也不会是现在的这种地位和身份。
<br/>
<br/>        心有所属，便无所欺。他们的心中都有一杆秤，一份光。
<br/>
<br/>        张函对张倩楠说：“江俞轩送来的资料可以交给潘青松，让潘青松安排人去查一查。这样夏颜会认为是陈子昂和她作对。”
<br/>
<br/>        张倩楠不理解：“那这样岂不是更让夏颜对陈子昂仇视了？”
<br/>
<br/>        张函：“想要一个人堕落之前必须使其疯狂。陈子昂很聪明，不可能不提防夏颜，尤其是他们公司发生变故之后。”
<br/>
<br/>        张函沉默了一会：“再说了，我需要一个锲机，就让陈子昂作为这个契机的导火索吧。”
<br/>
<br/>        “你这样岂不是将陈子昂至于危险地步？”张倩楠这会不理解张函了，他不是那么想要保护陈子昂母子吗？
<br/>
<br/>        “相对于在泥潭里苦苦挣扎的陈子寒来说，陈子昂是幸福的，我想，陈子昂也会愿意的。要不然她会去甸城？”
<br/>
<br/>        “可是，毕竟陈子昂并不太清楚陈子寒的情况。”
<br/>
<br/>        “你们都小看陈子昂了。在第一次见陈子寒也就是谷强的时候，陈子昂已经起疑心了。要不然她在见到最疼爱她的哥哥时会那么坦然的谈笑风生？”
<br/>
<br/>        “陈子昂的心机好深。”张倩楠低语。
<br/>
<br/>        “不，对她来说有些事情只是不屑为之，如果真的做起来，碾压夏颜、陆玉等人。”张函眼睛里闪着光，人性本善，在没有受到威胁之前，陈子昂一直是善良的，她绝对不是懦弱，她在很小的时候都会那么决绝的对待陈松年和她自己，何况现在在社会上打拼了这么多年？
<br/>
<br/>        陈子昂现在的软肋就是俩个孩子，孩子是陈子昂的软肋也是他的软肋，陈子昂不会任人宰割的，她一定会行动的。
<br/>
<br/>        我爱着你，我也理解你，更会尊重你，支持你，尊重你的选择，支持你的决定，如果我不能陪你到最后，我会用我的一生为你祝福，为你祈祷。
<br/>
<br/>        “那我下午将这些给潘青松送过去？”张倩楠问张函。
<br/>
<br/>        “给他打电话让他来一趟。”
<br/>
<br/>        .......
<br/>
<br/>        潘青松看到了张函面前的资料，气得浑身发抖，这些人是多么的无耻啊。
<br/>
<br/>        “很生气？”张函笑着问。
<br/>
<br/>        “你还笑得出来？”潘青松看着张函和张倩楠俩个人淡定的样子:“你们是不是早都知道了？”
<br/>
<br/>        “是啊，早都知道了，那又怎么样？我想陈子昂自己都会知道吧？都过了这么久，你又何必生气呢？”张函淡定的说。
<br/>
<br/>        “张函，我有事情问你。”
<br/>
<br/>        “什么事情？”
<br/>
<br/>        “闫欢你认识？”
<br/>
<br/>        “高中同学。”
<br/>
<br/>        “不止是高中同学的关系吧？是你将她送到陈子悦的身边。”
<br/>
<br/>        “你想说什么？”
<br/>
<br/>        潘青松自己给自己续了杯茶：“闫欢曾经在甸城呆过一段时间，是你将她找回来的，你很清楚甸城的情况吧？”
<br/>
<br/>        “我不清楚甸城的情况，当时我也只是想去找陈子昂，偶遇闫欢。”
<br/>
<br/>        “呵，我们子澜在甸城出事就说明了一切，如果你丧心病狂，那么请你远离陈子昂和你的俩个孩子，这份文件我拿走了，我会为陈子昂讨回公道的。”潘青松的语气透着决绝。
<br/>
<br/>        “你就这么想着我是个丧心病狂？”张函微微一笑。
<br/>
<br/>        “你.......”潘青松气结。
<br/>
<br/>        “清者自清，浊者自浊，我何必要解释？该干什么不该干什么你应该比我清楚，我是陆家的女婿，自然会和陆家站在一起的。”张函清冷的声音让潘青松心里一惊，他是陆家的女婿，自然要和陆家站在一起。
<br/>
<br/>        张函不说张家却说陆家。
<br/>
<br/>        “哈哈哈哈，你是陆家的走狗吧？陆家当你是女婿了吗？如果陆家当你是女婿的话，你现在还是这幅狗模样？”潘青松直接就骂出了口。
<br/>
<br/>        “我愿意当走狗怎么了？你管得着吗？有本事你动我啊，动一动啊，试试看，敢不敢？”张函还是一副不急不缓的模样。
<br/>
<br/>        “你们俩人吵来吵去有意思吗？放着正经事不干，在这里吵吵闹闹的。”张倩楠看着俩个人你一句我一句的吵闹不休，出声阻止。
<br/>
<br/>        潘青松拿起文件，摔门而去。
<br/>
<br/>        “你怎么这样说？明明你不是这样的。”张倩楠有些心疼。
<br/>
<br/>        “那要我怎么说？大家不都是这样想的嘛？我本来就是陆家的女婿。”张函看着窗外阴沉的天空，心如同这天气一样压抑。
<br/>
<br/>        “哥，我陪你一起去甸城吧。你走哪我跟哪。”
<br/>
<br/>        “你去干什么？你不要去趟这趟浑水了，好好的找个人嫁了，好好过日子！”
<br/>
<br/>        “这趟浑水你都趟了，我们张家能不跟着趟吗？你如今想要站在陈子昂的对立面就不怕陈子昂伤心吗？”
<br/>
<br/>        “已经伤过一次也就不怕在多伤一次，这里就交给潘青松了，潘青松嫉恶如仇，不光是陈子昂的事情，有些事情也要让他知道，他会一点一点的让有些东西浮出水面的。”
<br/>
<br/>        “光潘青松一个人不行吧？”
<br/>
<br/>        “当然不会只是潘青松一个人，会有人和潘青松站在一起的。倩楠，过几天我就要去甸城了，行程估计会和陈子昂相差无几。你就在这里和潘青松一起吧，我需要你们在这里作为我的后盾。”
<br/>
<br/>        “你知道西城的情况，甸城你不清楚，你过去有什么作用？”
<br/>
<br/>        “正因为不是很清楚我才要过去，陆战君在很久之前就和甸城的秦寿有所来往，他们到底有着什么样的瓜葛我们并不清楚，还有最重要的一点，陈子寒已经当了谷家的义子很多年。陈子寒的身份你我都明白，为什么陈子寒迟迟不回家？”
<br/>
<br/>        “陈子昂的父亲生病了，陈子昂会回来，你要去看她吗？”张倩楠岔开话题，他们心里都有一种感觉，只是没有得到证实而已。
<br/>
<br/>        “看情况吧，我去看看陈子澜，然后收拾回北京，准备去甸城。”
