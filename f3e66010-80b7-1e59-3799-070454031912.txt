　　楚天墟的天赋，非常不凡，虽然不是万古天骄，但和万古天骄相比，也不差多少。
<br/>　　其靠的就是通天神树元神，只不过，他并没有血脉之力，也没有其它的不凡之处。
<br/>　　“你逃不了了！”
<br/>　　浩荡的冲击波之中，苏莫大步的走出，眸中闪过犀利的杀机。
<br/>　　不过，他正准备吞噬这通天神树元神，周围的十数人再次疯狂的攻击了起来。
<br/>　　十几道强大的攻击，如同一只只咆哮的巨兽，向苏莫身上飞扑而来。
<br/>　　苏莫见此，立刻出剑抵挡，十几位万古天骄级别天才的攻击，他也不敢无视。
<br/>　　随即，苏莫一边催动吞噬元神，束缚楚天墟元神的逃走速度，一边挥剑抵挡众多的攻击。
<br/>　　轰！轰！轰！！
<br/>　　一道道攻击被苏莫斩碎，在他全力出手之下，这些人根本伤不到他。
<br/>　　而那些攻击苏莫之人，都知晓苏莫的强大，根本不敢靠近。
<br/>　　楚天墟的通天神树元神，受到了吞噬之力的束缚，逃走的速度骤降。
<br/>　　嗖！
<br/>　　苏莫身形如电，一边不停的挥剑，一边洞穿层层的冲击波，瞬间就接近了楚天墟的元神。
<br/>　　咻！
<br/>　　楚天墟的元神眼见苏莫接近，那略显虚幻的神树之上，一根枝干急速膨胀，化为了一条神鞭，向苏莫抽击而来。
<br/>　　但是，只剩下元神的楚天墟，实力和苏莫相比天差地别，苏莫大手一抓，轻易的就将对方擒拿，然后彻底吞噬。
<br/>　　“杀！”
<br/>　　朗喝声震天，是贺兰殇带领大批人手杀了过来。
<br/>　　有贺兰殇等人的加入，第六城和第八城之人，立刻便落入了下风。
<br/>　　轰隆隆！！
<br/>　　对于苏莫的围杀并没有停止，十几名第六城和第八城之人，拼命的攻击苏莫。
<br/>　　贺兰殇身形如电，一边出手为苏莫抵挡攻击，一边飞向苏莫，呼吸之间便来到了苏莫的身侧。
<br/>　　“贺兰兄，多谢了！”苏莫朗声说道，手中动作不停，不断的挥剑。
<br/>　　“苏兄，你我两城既然互为盟友，我自然回来支援！“贺兰殇笑了笑。
<br/>　　然而，下一刻，异变陡生，贺兰殇一掌猛然拍向苏莫。
<br/>　　这一掌迅疾无比，掌劲如同亿万大山，恐怖绝伦，这是贺兰殇凝聚巅峰实力的最强一掌。
<br/>　　他有信心，这一掌之下将苏莫轰杀，虽然他知道苏莫防御超强，但是，这一掌的威力足以摧毁一切防御。
<br/>　　浩瀚的掌劲，如同汹涌的大海，瞬间喷发，直接苏莫的脑门。
<br/>　　如此强劲的一掌，就算苏莫的肉身抗住了，识海之中的元神也会被湮灭。
<br/>　　“就知道你会如此！”
<br/>　　苏莫眸中杀机一闪，电光火石之间，他来不及阻挡，但是却将终极神剑一横，挡在了面前。
<br/>　　轰！
<br/>　　无可匹敌的一掌，重重的轰击在了终极神剑之上，苏莫全身一震，立刻被轰飞了出去，倒飞千余丈。
<br/>　　“什么？”贺兰殇见此，顿时面色一变，脸色瞬间难看了起来。
<br/>　　居然挡住了！
<br/>　　再这一掌之下，苏莫只是被轰退，居然没有受伤。
<br/>　　“这……！”
<br/>　　“怎么回事？”
<br/>　　“什么情况？”
<br/>　　此时此刻，周围的很多人，发现这一幕之后，都不禁目瞪口呆。
<br/>　　第二城和第三城不是结盟了吗？
<br/>　　贺兰殇不是来支援苏莫的吗？
<br/>　　现在怎么攻击苏莫？
<br/>　　这是什么情况？
<br/>　　所有人都是不明所以，很多人甚至纷纷停手了，就连远处的古魔太子和叶凌天，都暂时停住了。
<br/>　　那些贺兰殇带出来的第二城之人，甚至都不知道，是该攻击谁了。
<br/>　　“混账！”
<br/>　　第二城之中，龙羽见到外界的情况，立刻大声怒骂，这贺兰殇居然反水了。
<br/>　　一众第二城之中的人，均是不明所以，原来贺兰殇主动要求出去，不是支援苏莫，而是要对付苏莫。
<br/>　　一直毫无动静的第五城之中，一大群人伫立在半空之中，遥望着外界的情况。
<br/>　　这些人之中，为首的是龙腾和霸云公子。
<br/>　　看到外界的情况，龙腾的面色凝重不已，但是，他却是不好出去。
<br/>　　“龙腾，我们现在最好的办法，就是静观其变，坐收渔翁之利！”身材雄壮无比的霸云公子，声音洪亮的说道。
<br/>　　在这第五城之中，他一直在制衡着龙腾，不然对方早已带人出去了。
<br/>　　“霸云公子，我想要出去，你拦不住我！”龙腾声音低沉的说道。
<br/>　　“你可以出去，我不拦着你，但是，我们这些人不会和你一起！”霸云公子冷笑道。龙腾和苏莫的关系，众所周知，没有人会跟着龙腾去帮助苏莫。
<br/>　　龙腾面无表情，不过，他并不急着出去，因为他明白，苏莫的实力自保完全没问题。
<br/>　　别说是贺兰殇，就是古魔太子，能不能击败苏莫还是一个问题。
<br/>　　……
<br/>　　外界，苏莫身形伫立在半空之中，冷漠的目光遥望着贺兰殇。
<br/>　　此刻，他承受了对方一掌，虽然是用终极神剑挡住了，但是，体内的气血也是被震得不停的翻涌。
<br/>　　主要是，他之前正在攻击其它人，没有做足防御。

<br/>　　此刻，他心中已经有了猜测，不死神朝估计已经站在了帝一魂或者是拓跋青云一方了。

<br/>　　贺兰殇冷笑一声，道：“这里是千选战，只有淘汰了你，我才有更大的希望获胜！”

<br/>　　苏莫眸中杀机一闪，虚空踏步，一步百丈，向贺兰殇大步走了过去。

<br/>　　古魔太子脸色一沉，他适才的确是想出手，不过，有叶凌天拦着，他没有什么机会。
