　　“裂天剑派？”萧逸和其余长老同时惊疑一声。
<br/>　　萧逸是本身没离开过紫云城，对这个世界很多东西知之甚少。
<br/>　　其余长老则是面露崇拜和向往。
<br/>　　大长老道，“裂天剑派，乃是我们北山郡第一门派，更是我们北山郡所有武者向往的武道圣地。”
<br/>　　“只有资质极佳者才有机会进入那里修习，门派内随便一个弟子都是绝世天才，随便一位长老都是修为高深的强大武者。”
<br/>　　“若萧逸你能进入裂天剑派，北山慕容家便奈你不何。”
<br/>　　“裂天剑派如此厉害？”萧逸吃了一惊。
<br/>　　北山慕容家如此庞然大物，竟也不敢得罪裂天剑派？
<br/>　　大长老笑道，“你未曾出过紫云城，不知外面的世界何等精彩，又是何等的藏龙卧虎。”
<br/>　　“紫云城，不过是偏安一隅的小城，在北山郡一百五十城中甚至排不上号。你见识过最厉害的武者，也不过是先天境，不，洞玄境。”
<br/>　　“但，那些真正的大家族，大势力，洞玄境不过是为他们效劳的一份子罢了。强大的破玄境，以一人之力可抵万军的地元境，乃至更强的武者，你日后都可能会遇到。”
<br/>　　“裂天剑派，乃是北山郡王都要礼遇三分的门派，你可想而知他们的强大。”
<br/>　　说罢，大长老认真地看着萧逸，道，“萧逸，你的天赋，是我生平所见。进入裂天剑派修习，绝对不是难事。”
<br/>　　“呵呵。”萧逸自嘲一笑，道，“我这样，算不算即将逃离的丧家之犬？”
<br/>　　其实，就算这次没有北山慕容家之事，萧逸也早就想出外历练，见识一番。
<br/>　　只是，以这种姿态离开，却是他不愿意的。
<br/>　　但，现在的他，毕竟是萧家萧逸，有了牵挂，有了重视的人，他必须为他们考虑，也要为家族考虑。
<br/>　　若自己不走，只会让三长老和其余长老们为难和担忧。
<br/>　　大长老看着萧逸的自嘲之色，收起了笑容，脸色唏嘘。
<br/>　　才刚解决了萧家危机，而且最大的功劳是萧逸。但，现在这个萧家第一天才，却被逼到不得不逃离的地步。
<br/>　　说到底，还是萧家太弱了，没有能保护萧逸的实力。
<br/>　　对此，众位长老相当愧疚，却也无可奈何。
<br/>　　萧逸看了眼长老们脸上的愧疚之色，暗暗摇了摇头，暗道，“何必让长老们烦忧呢。”
<br/>　　萧逸当即收起了脸上的不爽和自嘲，恢复了以往自信的淡淡笑意。
<br/>　　“还是那句话。”萧逸忽然认真地说道。
<br/>　　“嗯？”众长老纷纷抬头看向萧逸。
<br/>　　萧逸淡笑道，“帮我照顾好依依，快则两三年，慢则五六年，你们会听到我名扬北山郡的事。我会以凌驾于北山慕容家的姿态，重新回来。”
<br/>　　“额。”众长老忽然脸色变了变，有些欲言又止。
<br/>　　“怎么了？信不过我？”萧逸笑着问道。
<br/>　　“不是。”大长老摇摇头，道，“你不是个信口开河的人，至今为止，你答应我们的事，也从来没有让我们失望过。你的天赋，只要给你足够的时间，必然能成为名动一方的武者。”
<br/>　　“只是...”大长老支支吾吾的。
<br/>　　最后还是三长老率先从怀中拿出一物，交给萧逸，并道出了依依之事。
<br/>　　萧逸听罢，神色突变，但他掩饰得很好，仍旧淡淡的模样，“依依被带走了？”
<br/>　　萧逸看了眼手上的东西，那是一个令牌，通体锃白，正面写着‘圣月宗’三字，背后是一轮洁白无邪的圆月。
<br/>　　“你们知道这圣月宗吗？”萧逸出言问道。
<br/>　　众长老纷纷摇头，“不知道，听都没听过。不过一看就知道绝非籍籍无名的小宗派，你在外历练时，兴许会遇到。”
<br/>　　虽然萧逸掩饰得很好，但熟悉他的三长老还是一眼看出了他脸色的变化。
<br/>　　“逸儿。”三长老安慰道，“我当时就在依依身旁，大概听到了她们的一些对话，如果我没有猜错的话，那位强大的武者，应该是起了收依依为徒的心思，你不用担心...”
<br/>　　“我没有担心。”萧逸神色自若地打断道。
<br/>　　“以那神秘武者挥手灭掉慕容家众武者的实力，完全可以强行掳走依依。但她没有，而是出言询问，证明不是大奸大恶之辈。”
<br/>　　“临走前，她还替你们稳住了自爆的气息，证明她并不是那种恶毒之人。”
<br/>　　“留下这令牌，大概也是想让我们这些依依的族人放心。”
<br/>　　萧逸快速地说了一遍。
<br/>　　而后问道，“大长老，我准备即日便离开紫云城。待会儿稍作准备一番，今晚趁夜离开。”
<br/>　　“嗯，早点走也好，免得夜长梦多，毕竟不知道北山慕容家的人何时会找上门。”大长老点点头。
<br/>　　“我出去走走。”萧逸自顾地下了床，往门外走去。
<br/>　　房内，众长老齐齐叹气。
<br/>　　“那小子，嘴上说没有担心，实则比谁都要紧张吧。”二长老摇了摇头。
<br/>　　“他那些解释的话，与其是说给我们听，倒不如说是给自己听的，让自己强作安心。”大长老和三长老也摇了摇头。
<br/>　　“卧槽，我懂了。”六长老忽然后知后觉地说道，“我就知道，那小子可不是怕事的主儿。走得这么急，可不像是怕北山慕容家的人追杀他，八成是为了去找依依吧。”
<br/>　　六长老话语刚落，其余长老纷纷翻了个白眼，像看白痴似地看着他。
<br/>　　.....
<br/>　　萧家府邸之外，萧逸再次拿出令牌，用力捏了捏。
<br/>　　他发现，自己全力之下，甚至双手捏得生疼，都无法撼动令牌半分。
<br/>　　甚至于，自己捏着令牌时，竟有一种心清目明的感觉，很立于修炼。
<br/>　　单看这令牌的不凡，就知道那圣月宗绝非泛泛之辈。
<br/>　　而且，看众长老的迷惘脸色，看来这个宗派并非是北山郡的宗派。
<br/>　　“圣月宗吗？”萧逸脸色肃然，喃喃自语着，“带走了依依，却留下这个令牌，呵，等着，我会亲自找上门去的。”
<br/>　　萧逸其实心里清楚得很。
<br/>　　如果不留下令牌还好，证明那神秘武者终有一日会带依依回来。
<br/>　　若留下令牌，留下信息，又明知自家宗派不在北山郡内，与紫云城隔着千山万水，路途遥远又危险无比，一个小家族的武者绝不可能安然无恙地到达。
<br/>　　那么，这个神秘武者的意思就很清楚了。令牌我已经给你了，有本事便自己寻来，没本事，也怪不得我。
<br/>　　万幸，观那神秘武者，似乎不是坏人，而且对依依颇为重视，倒不用太过担心依依的安危。
<br/>　　只是....
<br/>　　萧逸蓦然一笑，“那位神秘武者，你可能对依依没有歹念，而且会对依依很好。但，依依不能在我这个少爷身边照顾着，她会很不习惯，很念着我的吧。”
<br/>　　其实，萧逸想说的是，他没有依依照顾着，会很不习惯，很念着依依。
<br/>　　他没有说出口，但心里始终重复着，始终是依依一言一笑的画面。
