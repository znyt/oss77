夏夕绾雪白的耳垂上迅速蔓延上了一层红晕，都这个时候了，他竟然还敢笑话她！

<br/>        虽然，她的确很不好意思，刚才因为太气愤太着急了，所以才会出手抢内裤，现在被他这么一笑话，夏夕绾都想钻地洞了。
<br/>
<br/>        夏夕绾用力的踩了一下他的脚，泄愤。
<br/>
<br/>        陆寒霆搂紧了她的纤腰，恨不得将她的腰给折在他的臂膀里，“疼…陆太太饶命，我下次不敢了！”
<br/>
<br/>        这个男人，她虽然是很用力的踩他的，但是她才多大力气啊，根本就是给他挠痒的，他竟然贴在她耳畔喊疼。
<br/>
<br/>        夏夕绾扭动了两下，想要挣脱他的怀抱，“你再这样的话我就真的不理你了！”
<br/>
<br/>        陆寒霆将她禁锢在自己的怀里，不许她乱动乱蹭，薄唇贴在她清纯的长发上用力的亲了两口，“陆太太，这事我是一点都不知情，我是冤枉的，我向你发誓我陆寒霆真的没做过拿一条内裤来勾引女人这档子事。”
<br/>
<br/>        他信誓旦旦的誓言，带着愉悦和耐心覆在她耳畔哄她，夏夕绾的心情迅速变好了，但是想到那个女人可以随便出入他的房间，她还是觉得不爽，她像是古代媚惑君王的妲己般任性的跟他吹耳边风，“我不管，我就是不喜欢那个清洁阿姨，你将她辞退了，不单是这个，下一个，下下一个，你都不许找年轻漂亮的女人！”
<br/>
<br/>        像这些年轻漂亮的女人怎么可能来做保洁工，她们都是冲着陆寒霆这个人来的。
<br/>
<br/>        陆寒霆蹭着她滑腻的脸蛋，“都听你的，这事是我没有顾及到，我会交代下面的人办好的，将这一个辞退了，找个老实年长的，幽兰苑的女佣都有奶奶把关的，我们的卧室也只有吴妈能进去收拾，这事是我的错，不气了陆太太。”
<br/>
<br/>        夏夕绾的心情这才平复了下来，被他哄好了，像这种私密的事情奶奶从世阀豪门里走出来的自然都知道，有多少年轻漂亮的姑娘想要爬上主子的床，更何况像陆寒霆这种魅力十足的男人，垃圾桶里的纸，凌乱的床单甚至是昨晚闹了好久的沐浴间…哪里能让那些小姑娘进去收拾？
<br/>
<br/>        夏夕绾就是不喜欢。
<br/>
<br/>        “算你听话，那我就原谅你一次了。”
<br/>
<br/>        陆寒霆的大手落在她平坦的小腹上，然后从衣角里悄然钻了进去，嗓音低哑道，“陆太太，不用怕，自信一点，除了你，我是不会被其他女人给勾引走了。”
<br/>
<br/>        他略显粗粝的指腹落在了她肤如凝脂的娇肌上，夏夕绾吓得往他怀里缩，他右手腕上还戴着名贵的钢表，落在了她的衣服外面。
<br/>
<br/>        夏夕绾俏脸通红，迅速去推他的大手，“你干什么，现在是白天，还在公司里，你不许胡来！”
<br/>
<br/>        陆寒霆不肯松开她，反而将她箍的更紧，“这是我的公司，你是我的人，谁能管我胡来不胡来，陆太太，你也看到了，外面的狐狸精可是很多的，所以你要将我拽紧了。”
<br/>
<br/>        “…”
