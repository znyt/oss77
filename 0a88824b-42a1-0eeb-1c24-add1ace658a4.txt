　　这是觉醒生物的对决，也是食物链天敌的对决。
<br/>　　正常来说，猫对付老鼠太轻松了，几乎是生存本能，但是眼前这只小白鼠可是二阶精怪的层次，而黄金瞳不过19级，还未入二阶。
<br/>　　那么两者实力就形成了一个不小的差距。
<br/>　　只见拥有老虎之威，狮子之势的黄金瞳一脸凶相，一双阴阳眼光芒大亮的盯着对面的那只小老鼠，却没有轻举妄动，虎视眈眈等一个最佳的时机攻击。
<br/>　　面对威势滔滔的黄金瞳，这只小白鼠直立着小身子，口吐人言道：“好一只血统不凡的猫类，本包子大人吃过不少猫，还从来没有碰到如此血统的猫，极品啊”
<br/>　　小小老鼠，如此神态，将苏羽惊的三观尽毁，而余子童脸色难看起来，他是捉妖师，虽然是初级的那种，但也知道能做到口吐人言的鼠精已经不是他这个一品捉妖师可以对付的。
<br/>　　况且平时出去捉妖，他都是全靠自己的黄金瞳。
<br/>　　要是自己的黄金瞳拿不下这只鼠精，就麻烦大了，更别说还有一位生物掌控者。
<br/>　　另外他也有点担心自己的黄金瞳，那可是他的宝贝，这只猫要是有什么好歹，他都没法在白家混。
<br/>　　“嗷...”
<br/>　　黄金瞳不能说话，只能不断的冲着小小鼠精怪叫，叫着叫着，全身炸毛的扑了上去。
<br/>　　一猫一鼠开战了，一只化为金光，一只化为白光，在工地上不断的来回闪烁，来回拼杀。
<br/>　　这算是很有意思的战斗，众人都开始关注这场猫跟老鼠的厮杀。
<br/>　　林凡胸有成竹，那只猫虽然非常的不凡，但是境界太低，还没入二阶，怎么可能打的过自己的鼠精。
<br/>　　能拼的你来我往，已经非常的了不起。
<br/>　　“余大师，你不准备出手吗”
<br/>　　涂小安忽然浅笑吟吟的看着他。
<br/>　　“出手什么”余子童看着戏，微微一愣。
<br/>　　涂小安白了一眼：“归根结底，那个黑袍才是问题的关键，你把他干掉，事情不就解决了”
<br/>　　苏羽顿时看向余子童：“余大师，苏某的工地就全靠你了”
<br/>　　迄今为止，这位捉妖师貌似没做什么，都是这只猫出马解决，他顶多拿起一把剑，砍砍几只老鼠，并没有露出什么真本事。
<br/>　　余子童见涂小安在一边挑拨，气得脸色涨红，浑身有些颤抖，指着落井下石的涂小安说：“你懂什么，那人是生物掌控者，是那么好对付的吗”
<br/>　　“要是好对付，苏老板请你过来干嘛，当啦啦队？”涂小安可记住他之前嘲笑看不起自己的样子。
<br/>　　什么白家捉妖师，白家涂小安都没不放在眼里，还教训了白家两个人了。
<br/>　　也不在乎多让白家的一个人长长记性。
<br/>　　苏羽此时用怀疑质疑的目光看着余子童，然后道：“余大师，你莫非有什么难处，你要是觉得自己对付不了黑袍，那么我就跟你白家退订单，我的钱不能白白付”
<br/>　　所谓拿人钱财与人消灾，跟黑袍一样。
<br/>　　你看别人做的多敬业，你在这里拿一只猫出来，糊弄鬼呢，自己根本不出手。
<br/>　　余子童被说的脸上有点挂不住，如今的局势逼着他不得不出手。
<br/>　　其实他虽然是捉妖师不假，但是却是白家最低级的那种，以为碰到一个小订单，鼠灾什么，不是很容易搞定吗，他有黄金瞳，出马解决一切。
<br/>　　没想到惹出一个生物掌控者，更有一只二阶的鼠精，这已经远远超出了他的业务能力范围。
<br/>　　余子童微微一想，现在还没打呢，心理压力已经非常大了。
<br/>　　但不管怎样，余子童都必须要出手了，再不出手，面子上也挂不住。
<br/>　　况且他的黄金瞳现在也指望不上了。
<br/>　　只见余子童阴郁着脸，将掉在地上的长剑拿了起来，朝着黑袍林凡逼近了一步：“本大师在给你一次机会，离开工地，要是在纠缠不清，就别怪我手中剑不客气”
<br/>　　“不客气？”林凡一脸古怪的看着余子童：“你真的是白家的人？”
<br/>　　余子童冷哼哼的说：“这还能是假，若是怕了，现在就走”
<br/>　　林凡森然的笑了笑：“我不是怕，我是好奇白家派出来的人怎么那么弱，一品捉妖师也敢出来丢人现眼？”
<br/>　　“混帐，居然敢羞辱我，看剑”
<br/>　　余子童勃然大怒，泥人还有三分火，提着手中长剑，冲着空气挥舞了几下，有剑吟声顿起，配上他的中山装，还真有几分气势。
<br/>　　“不知好歹”黑袍男子宠辱不惊，冷笑连连，见余子童提剑杀来，一脸的轻蔑之色。
<br/>　　只闻砰的一声，还没看到什么刀光剑影，余子童就倒飞了出去，他自己甚至都没能作出任何反应，便撞在了一面坑坑洼洼的石壁上。
<br/>　　剧烈的疼痛，让余子童忍不住吐出一口鲜血。
<br/>　　“这么强？”
<br/>　　余子童喘着粗气，心中惊涛骇浪，他其实知道自己八成不是对方的对手，可没想败的那么随意。
<br/>　　冲过去，对方身形飘逸恍若幻影起来，他就胸口一痛的飞了出去。
<br/>　　看着还在原地的黑袍男子，他心中暗道：“完蛋，丢脸丢大了”
<br/>　　没有黄金瞳，他的战斗力简直虚无一样。
<br/>　　余子童不禁有些绝望，这样的情况下，他算是把白家的招牌砸光了。
<br/>　　本来已他的实力，是没资格成为白家的捉妖师，之所以能在大树下好乘凉，是因为他有一只好的宠物。
<br/>　　说起宠物黄金瞳，也是余子童命好，他家中一直有养猫的习惯。
<br/>　　忽有一日，家中母猫也不知道怀上了那只流浪猫的野种，就等待产了，余子童开始也没在意。
<br/>　　慢慢的，母猫的腹部大了起来，随着时间的流逝，三个月后，母猫依然不见产仔，这就引起了余子童的注意。
<br/>　　都说人是十月怀胎，但猫科动物，一般二月可产仔，可这母猫在三个月的时候，完全没有产仔的意思。
<br/>　　五个月后，母猫腹部越来越大，大的力气，但就是没有产崽。
<br/>　　余子童觉得非常反常，他就想看看，自己家的母猫什么时候能产崽。
<br/>　　这一等，足足十个月，母猫终于要产崽了。
<br/>　　他非常的诧异，十月怀胎？一只猫既然跟人一样，怀孕十个月才产崽。
<br/>　　这不科学呀。
