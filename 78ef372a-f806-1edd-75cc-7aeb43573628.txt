　　不得不说的是，进了太极宫的东宫之后，让那些商人大失所望，残破的宫墙上长满了杂草，并不怎么宽大的小广场上依稀能够看到刚刚整理过的痕迹，估计应该是这几天才临时整理出来的，否则也应该是遍地杂草吧。
<br/>　　至于说宫殿，这个倒是挺有气派的，可是屋脊上红一块绿一块的琉璃瓦算怎么回事？难道就不能统一一下么？又或者是实在没钱了，东一块西一块拼凑起来的？
<br/>　　当然，最要命的就是地上的那些地砖和台阶，这些石质的东西几乎没有一块是完整的，不是碎裂就是缺口。凡此种种无一不证明皇帝陛下以前的日子过的实在是不怎么样，否则这东宫也不可能如此破败。
<br/>　　不过这些显然只是用来满足那些商贾们的好奇心的，对招标会并没有什么实际的影响。
<br/>　　辰时一到宫门封闭之后，所有还没有入场的人全部被关在了外面，长安杯马球大赛广告招标会正式开始。
<br/>　　长乐、林晓晓、杨雨馨三人分别着一品公主正装与二品仪人正装，一前两后成品字型从明德殿内走出，在身后无数宫娥的簇拥下，显得贵气逼人。
<br/>　　而虽着三女出现，四周警戒的禁军，内侍，“轰隆”一声拜了下去，山呼海啸般的声音响起：“拜见长公主，拜见两位仪人！”
<br/>　　一群商贾哪里见过如此场面，见状之后亦纷纷身不由己的拜了下去，口中参差不齐的喊道：“草民拜见长公主，拜见两位仪人！”
<br/>　　长乐淡淡扫了一下半面躬身施礼的人群，微微一笑，摆手说道：“都起来吧！一会儿都好好计算一下，不要算的错了，机会只有一次，错过了便不会再有。”
<br/>　　“喏！”商贾们平日里根本没有见到长乐的机会，此时虽然离的远但多少也是见过了，一个个顿时显的激动异常。
<br/>　　杨雨馨见时间差不多了，越过长乐走到前面：“好了，大家都回去忙吧，记得，午时之前一定要把自己的底价交上来，否则按弃权处理。”
<br/>　　这一切都是她们三个刚刚在里面商量好了的，亮个相，刺激一下那个商贾，说不定这帮家伙脑子一热，还能再把价格提高个两、三成。
<br/>　　也不知道这三个丫头到底是怎么想的，如果这事儿她们三个提前问问小武，小武那丫头一定会告诉他们，这不可能，死了这条心吧，那些商贾一个个鬼精鬼精的怎么可能会提价。
<br/>　　……
<br/>　　长乐三女在露了一面之后便回了明德殿，商贾们也抓紧时间回到属于他们各自的隔间，也就是一个个大帐篷里面。
<br/>　　现在是辰时，距离午时满打满算也只有两个时辰，他们这些人必须在这最后的两个时辰里对前面三天里的判断做出最后的决定，而这个决定关系到他们的未来。
<br/>　　不过，因为这次的招标是以暗标形式存在的，他们根本就不知道别人会投入多少钱，这就是一件让人十分尴尬的事情了，就算是提前打好了招呼只怕在这个时候只怕都有被坑的危险。
<br/>　　彭家是做绸缎生意的，这一次来投标的人里，彭家东主彭城见到了四家与自己家里生产规模差不多的生意对手，这让老彭一瞬间紧张了起来，不自觉得开始怀疑自己的判断。
<br/>　　在此前的三天，他几乎将分布在长安各处的两千个广告位走了个遍，选择了三十个比较理想的位置，同时心中也大致上有了一个基本的报价——三万贯！
<br/>　　可是在见到其他几个互为对手的家伙之后，他开始对这个底价有不大确定，要知道，这可是只有一次机会，如果失败了，那就再也没有机会了。
<br/>　　而且最重要的是，广告位是一回事，随之而来的商业洽谈才是正餐，如果不能争到一个广告的位置，那么接下来与那些异族大商人和大贵族的商业洽谈自然也就无从谈起。
<br/>　　而因此带来的后果就是，他只能看着别人家发展壮大，而彭家只能渐渐没落。
<br/>　　黑啊，真黑，太黑了！彭城已经不知道这是第几次在暗中骂街，可是最终却不得不暗中咬牙，在自己的心理底线上再次加了一万贯的筹码。
<br/>　　这已经是他的最后能动用的资金了，如果还不能争到一个广告位的话，那只能说是命运弄人。
<br/>　　……
<br/>　　时间一点一点过去，小广场的气氛也越来越紧张，数百商户各有各的打算，有人成竹在胸，也有人心中忐忑。
<br/>　　可他们不知道，其实明德殿里的三个女人此时比他们还要紧张。
<br/>　　长乐好些，毕竟生于皇宫大风大浪见过不少，是三女中表现的最轻松的一个，可是因为她是这次马球赛的负责人，招标成功与否关系到马球赛是否会成功，也让她这个大唐长公主暗暗握紧了拳头。
<br/>　　杨雨馨跟着李承乾十余年，多少也算是见过一些风雨，但这次招标会是由她负责的，如果失败了，个人丢了面子事小，以后要被人笑话一辈子才是大事，这让原本信心十足的她也是万分紧张。
<br/>　　最后就是林晓晓了，这个李承乾在大唐接触过的第一个女人并没有什么特殊的能力，性格也不是十分强悍，如此高压之下让她没来由的紧张，就算是和她的切身利益没有什么相关，也还是紧张的手心里面全是汗水。
<br/>　　终于，午时还是到了，外面小广场上钲鸣声声，提醒着众人，投标正式开始。
<br/>　　……
<br/>　　商人们并不是第一次参加招标，自然知道其中的规矩，一个个早在午时之前就已经把标书准备妥当，只等着内侍过来收取。
<br/>　　长乐三女坐直了身体，结果马上就要出来了，一切只看这第一阶段位置最差的几百个广告位能卖出多少钱，如果卖的少了，估计后面的那些位置也完蛋了。
<br/>　　而随着一份份投标书被送进来，一群宫女则开始统计了起来，前面的那些废话统统不看，直接挑重点，查看标书中的价格，然后加减乘除一顿狠算，最后的结果被列于一边的公告板上，以便长乐几女能够看的清楚。
<br/>　　一个报价，两个报价，随着越来越多的报价被列出来，长乐三女的心渐渐落回了原位，信心也再次恢复过来。
