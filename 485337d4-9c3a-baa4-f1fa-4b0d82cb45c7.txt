　　在归墟天的其他地方，楚风眠都没有碰到其他武者的存在，可如今，他们既然是隐藏在了这山谷之中。
<br/>　　这山谷中，肯定是有着天大的宝藏，才会让他们都隐藏在这里，不知道是在等待这什么。
<br/>　　“这山谷之外，另有乾坤，先离开这山谷再说！”
<br/>　　楚风眠打量了周围一眼，心中暗自想到。
<br/>　　如今他感觉的到，在这山谷之外，更是还有着极为宽阔的空间。
<br/>　　楚风眠的身形一动，连连在这山谷之中穿行，突然就在前方，似乎是到来了一处尽头，黑暗无比。
<br/>　　在这黑暗之中，有着一道空间玄门，不知道是通往哪里。
<br/>　　“既来之，则安之，走！”
<br/>　　楚风眠打量了一眼这空间玄门，这空间玄门的背后，楚风眠以灵识进入其中，都无法看穿任何一点。
<br/>　　不过既然来了，也没有在退缩的道理。
<br/>　　楚风眠脚下一动，直接埋入到了这空间玄门之中去，顿时周围景色，突然变幻。
<br/>　　在楚风眠的脚下所出现的，便是一处巨大无比的平原，而抬起头来看。
<br/>　　就在天际之上，一座巨大无比的山谷，清晰可见。
<br/>　　这山谷，正是刚刚楚风眠所经过的山谷。
<br/>　　只是如今这山谷，高高在上，在这空间之中，楚风眠还远远无法动用遁光，根本无法再次回去。
<br/>　　而在另一边，则是有着一座巨大无比的宫殿。
<br/>　　不止一座，而是一群宫殿，凌空立于天空之上，无数宫殿，位于群山之上，却全部都漂浮在空中。
<br/>　　“浮空城！云羽天宫！”
<br/>　　楚风眠看到这宫殿，猛然响起了记载中的传说，据说这云羽天宫的所在，便是就在九天之上。
<br/>　　曾经云羽天宫弟子，将宫殿建造完毕后，由当时云羽天宫的大能，以无上手段，将这宫殿全部托举在九天之上。
<br/>　　永不沉落。
<br/>　　当时在古籍中，楚风眠见到这一段的时候，还以为只是无稽之谈。
<br/>　　毕竟将一方宫殿，全部托举在天上，这是什么手段？现在七大宗门就算是倾尽全力，都无法做到。
<br/>　　可如今一看，这传闻却是半点不假，眼前这天宫，就在眼前。
<br/>　　而除了这天宫，下方平原前，却是还有着无数山峰。
<br/>　　隐约之间楚风眠足以感觉到，这山峰之中，还有着其他武者的存在。
<br/>　　“果然，在这山谷之外，还是另有天地，这简直是一方独立的空间，独立在归墟天之外！”
<br/>　　楚风眠一边观察这，一边向着无数群山的方向走去，这里，正是最为接近到云羽天宫的地方。
<br/>　　如今云羽天宫，高高在上，楚风眠也无法上去，只能是走一步看一步，寻找机会才行。
<br/>　　楚风眠一边接近山峰，一边便是发现，这山峰上，也有着不少破落的宫殿。
<br/>　　如今已经是不知道经过了多少年的岁月，上面已经是破败不堪，但是隐约之间还是足以看出曾经的繁华。
<br/>　　“在强大的宗门，也终于是有着毁灭消失的一人，不过其中的道统，却足以一代一代的传承下来，也算不灭。”
<br/>　　楚风眠感叹一声。
<br/>　　这云羽天宫如今破败，倒是成为了现在这个时代之中，无数武者所向的藏宝之地了。
<br/>　　曾经留下的无数东西，都成为了宝藏。
<br/>　　楚风眠在这山峰之中，不断穿行。
<br/>　　突然之间，一道眼睛，便是锁定在了楚风眠的身上。
<br/>　　这一道眼睛，充满杀意，露出无数凶光。
<br/>　　“敢踏足本尊的领地！死！”
<br/>　　一声冰冷的声音突然响起，只见一道身影，猛然从天而讲，似乎是从上方的山峰上，一跃而下。
<br/>　　一尊巨大无比的掌印，便是冲着楚风眠轰杀过来。
<br/>　　这一掌，蕴含无尽血气，一掌打出，周围尽是浓厚的血腥味，乃是一道鲜血时掌印。
<br/>　　“有人？”
<br/>　　楚风眠脸色都是微微变化这，这人的突然出现，就连他都没有察觉到，似乎是隐藏的极好。
<br/>　　不过楚风眠却并不慌张，在这一瞬之间，他也感觉到了这人的实力，还没有达到碎命之境的地步，虽然也是生死境巅峰的境界，但是最多也就跟剑苍穹平起平坐左右的实力。
<br/>　　这种实力，楚风眠自然不需要任何慌张。
<br/>　　“战龙之爪！”
<br/>　　楚风眠心神一动，战龙之爪突然凝聚而出，向着天上猛然轰击过去，将这鲜血掌印，直接轰破。
<br/>　　在力量上，除非是碎命之境的武者，不然一打一，楚风眠不惧怕任何人。
<br/>　　“恩？好强大的力量！这一届新人中，怎么会有如此人物？”
<br/>　　楚风眠头顶上，突然传来了那人惊讶的声音。
<br/>　　这一个交手，那人便是将楚风眠的实力知晓的一清二楚，决不好对付。
<br/>　　一招不成，这身影便是调转身形，毫不留恋，直接一步向着远方逃去，准备离开。
<br/>　　“偷袭完了就想逃？至少要表露身份吧！”
<br/>　　楚风眠嘴角露出几分冷笑，突然之间大手一挥，战龙之爪，穿越无数群山，猛然之间冲着那一道身影，抓了过去。
<br/>　　一个瞬间，那一道身影，便是就被楚风眠抓在了手心之中。
<br/>　　这是一名中年男子，这是一尊浑身血衣的衣服，他的衣服显现出一种妖艳的红色，看样子，似乎是由鲜血所染红的。
<br/>　　身上蕴含这浓浓的血腥味，不知道是沾染过多少鲜血。
<br/>　　九魔城的弟子。
<br/>　　楚风眠一眼便是认出来，此人是九魔城的弟子，除了九魔城弟子，不然没有其他武者，会修炼这等鲜血魔功。
<br/>　　“小子，我是九魔城的鲜血魔尊，速速放开我，不然等我九魔城的强者降临，倒时候你想死都难！”
<br/>　　这鲜血魔尊，被楚风眠抓在手中，还是一脸狂妄的吼叫着。
<br/>　　“鲜血魔尊？你是九年前，进入到这归墟天中的武者？”
<br/>　　楚风眠回忆了一下这个名字，突然想起来，在九年前，九魔城进入到归墟天的武者中，的确是有着这样一个人物。
<br/>　　在外面，都盛传这鲜血魔尊，应该已经是死在了归墟天中了。
<br/>　　但是现在一看，这鲜血魔尊哪里陨落了，还好好的在归墟天中活着了。
