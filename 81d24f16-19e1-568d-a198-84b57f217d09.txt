　　萧逸感受着冷焱剑的异动，看了眼空气中飘荡的金光。
<br/>　　如果说传言是真的，那么创立万金府的那位上古强者，到底强到了什么地步？
<br/>　　他感知过这些金光，里头蕴含着的，明显是金属性的气息，锐利惊人。
<br/>　　这些金光，历千万年而不消。
<br/>　　但武者吸收不了，也触摸不了，宛若空气。
<br/>　　上古时期的武者，当真强到了这般发指的地步？当真手段强悍到这般，让人感到匪夷所思？
<br/>　　萧逸脸上，闪过一丝向往。
<br/>　　但这丝向往，很快便被脸上的淡漠所掩盖。
<br/>　　手中火焰一凝，快速将这颗金光幽狼的内丹炼化为丹药。
<br/>　　下一秒，怀着一丝迟疑，缓缓将冷焱剑伸向空气中的氤氲金光。
<br/>　　剑身触及到金光，原本宛若空气般的金光，这一次，竟未被穿透。
<br/>　　剑身，横亘于金光之中。
<br/>　　渐渐的，金光涌入剑内。
<br/>　　不过片刻，待得面前金光吸收殆尽。
<br/>　　整把冷焱剑，除了原本的冷白异色外，还平添了一层金光之色。
<br/>　　看起来，就像是一层金光，包裹了剑身。
<br/>　　但这层金光，又极薄，宛若无物。
<br/>　　萧逸双眼一眯，“还真的被吸收了。”
<br/>　　萧逸感知了一下，发现此刻的冷焱剑剑身，锋锐到了极点。
<br/>　　锵…
<br/>　　萧逸以剑撑地。
<br/>　　嗤…的一声，整把剑，忽然没入了地面。
<br/>　　坚硬的地面，竟宛若纸糊的一般。
<br/>　　“嘶。”萧逸倒吸一口凉气。
<br/>　　如果说，剑气是无形之锋利，那么这些金属性气息，便是锋锐之极致。
<br/>　　萧逸蓦地闪过一丝想法。
<br/>　　看来，这些金光，除了常年生活在此险地的妖兽能吸收借用外，武器也能。
<br/>　　不过，若是所有武器都能借用之，那岂非等闲武器都能成为神兵利器？
<br/>　　萧逸试了试，手中一把亚圣器即便的剑凭空而现。
<br/>　　这是他以前战斗时，收获的乾坤戒内的战利品。
<br/>　　手中持剑，靠近这些金光。
<br/>　　果然，手中之剑，可以吸收这些金光。
<br/>　　空气中飘荡的这缕金光，不多时便被吸收殆尽。
<br/>　　然而，仅仅是数秒后。
<br/>　　咔…咔…咔…
<br/>　　剑身，忽然一阵碎裂。
<br/>　　数秒后，整把剑，寸寸尽碎。
<br/>　　剑身的碎片，哗哗跌落地面，唯手柄剩于萧逸手中。
<br/>　　“原来如此。”萧逸明白了过来。
<br/>　　武器，确实可以吸收这些金光。
<br/>　　只不过，这把武器能否承载这些金光，就是两说了。
<br/>　　冷焱剑乃是中品圣器，自是能承载。
<br/>　　可刚才手中之剑，不过是一把亚圣器，根本承受不住金光的极致锋锐。
<br/>　　萧逸的目光，再次看回冷焱剑。
<br/>　　此时的冷焱剑，无疑威力比之前增加了数分以上。
<br/>　　剑的威力提升，同样亦是剑修的实力之提升。
<br/>　　萧逸甚至相信，现今手持冷焱剑，即便不使用一剑出十数剑齐至的手段；单靠星光长河，便足以秒杀一位圣皇境九重强者。
<br/>　　萧逸这般想着，面露喜意。
<br/>　　然而，这抹喜意，仅仅出现了数分钟，便忽然消散。
<br/>　　因为，冷焱剑上的金光，忽然消散，再不复踪。
<br/>　　萧逸一愣，随即反应了过来。
<br/>　　看来，即便武器能吸收这些金光，但也同样会如妖兽内丹一样，留不住这些金光太久。
<br/>　　最多不过数分钟就会消散。
<br/>　　“这有何用？”萧逸瞥了眼那些金光，撇撇嘴。
<br/>　　“还吸吗？”萧逸看回冷焱剑。
<br/>　　冷焱剑剑身微震，随后再无异状。
<br/>　　萧逸不禁莞尔一笑，“刚才还异动连连，现在倒是嫌弃这些金光力量了。”
<br/>　　“算了。”
<br/>　　萧逸收回了冷焱剑。
<br/>　　与其打这些金光的注意，还不如多去猎杀些妖兽炼制金光丹。
<br/>　　整个金光险地，无比辽阔，妖兽的数量更是数之不尽。
<br/>　　这里，足够自己历练许久的了。
<br/>　　想罢，萧逸继续深入前行。
<br/>　　然而，恰在此时，一声声兽吼，忽然响彻整片森林。
<br/>　　剧烈的吼声，带着嗜血味道的嘶吼，让人闻之胆颤。
<br/>　　这一瞬间，整个金光险地的妖兽，忽然开始沸腾，开始暴走。
<br/>　　萧逸双眼眯了眯。
<br/>　　这般妖兽暴走一般的声音，他不是现在才听到。
<br/>　　早在之前不断深入，不，更早些前，金光险地陷入夜色时，这些响彻森林的兽吼声他就时不时能听到。
<br/>　　只不过，现在的兽吼声，更为剧烈。
<br/>　　空气中的暴走气息，也更加明显。
<br/>　　“兽潮吗？”萧逸脸上霎时闪过一丝凝重之色。
<br/>　　“不对。”
<br/>　　萧逸忽然又摇了摇头。
<br/>　　若是兽潮，兽吼声必是集于一点。
<br/>　　而且，一般会由妖兽森林的深处中爆发，声音亦最是剧烈。
<br/>　　可现在的兽吼声，来自于四面八方。
<br/>　　与其说这是兽潮，倒不如说，这是妖兽在欢呼之声。
<br/>　　那抹肆虐嘶吼内，还带着一股瘆人的嗜血之味。
<br/>　　嘭…
<br/>　　忽然，一道破空声，从萧逸侧边而来。
<br/>　　萧逸瞬间反应过来，脚步一退。
<br/>　　嗖…一道火红色风刃，在他眼前一划而过。
<br/>　　“疾风烈鸟？”萧逸看了眼不远处。
<br/>　　一头护身火红羽毛的鸟类妖兽，泛着凶光。
<br/>　　一双火红翅膀，剧烈抖动下，罡风惊人。
<br/>　　疾风烈鸟，皇境妖兽。
<br/>　　本是火属性鸟类妖兽，操控猛火。
<br/>　　但其是鸟类，本身天生就有御空纵风之能。
<br/>　　故这种妖兽，有同时操控风火的手段。
<br/>　　火借风势，威力陡增；风凭火之爆发，速度激增。
<br/>　　如刚才那道火红色风刃，几乎是萧逸刚刚察觉到这股风刃的瞬间，风刃已然打来。
<br/>　　若那时他没有退后一步，而是拔尖而出试图抵挡的话，恐怕，这道风刃已经打在他身上了。
<br/>　　论实力，疾风烈鸟，还在金光幽狼之上。
<br/>　　“原来如此。”萧逸霎时恍然。
<br/>　　金光险地的夜晚，确实很热闹。
<br/>　　一支支人类队伍，倾巢而出，斩杀妖兽，炼制珍贵的金光丹。
<br/>　　但，这般热闹，不仅来自于人类，亦来自于妖兽。
<br/>　　人类斩杀妖兽，妖兽亦同样仇视着人类，食之而后快。
<br/>　　夜晚，是它们吸食金光，借助金光力量之时，实力大增之时，亦是击杀人类武者的最好之时。
<br/>　　人类武者与妖兽的战斗，在入夜之时，已注定了激烈。
<br/>　　金光险地的夜晚，注定是个不眠之夜。
