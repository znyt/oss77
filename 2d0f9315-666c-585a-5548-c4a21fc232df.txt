　　罗昊眉头皱得很深。
<br/>　　何帆带给他的消息，的确有些出乎他的意料。
<br/>　　AT4单兵反坦克导弹，“毒刺”单兵防空导弹，这些军用级武器装备会出现在一个贩毒集团的手中，的确是令罗昊感觉到意外。
<br/>　　“能查到背后支持他的人是谁吗？”罗昊对何帆问道。
<br/>　　何帆摇头，回答道：“不过我们根据普提查所销售的高纯度cocaine分析，普提查手中所掌握的高纯度cocaine是南美洲那边的加工工艺。”
<br/>　　“南美洲……”
<br/>　　罗昊已经开始在脑海中思考，这次绑架的背后是谁的手笔。
<br/>　　“我说你们，不，他们T国警方也真是够菜的，一个毒贩居然让他嚣张了这么长时间。”
<br/>　　司徒空撇嘴，显然是对T国警方的办事能力感到不足。要是在国内，军警联合行动早就把对方的老巢都给铲平了。
<br/>　　“这里不是国内，而且毗邻金三角。”何帆苦笑一声，说道：“更何况，他们还拥有自己的眼线，普提查也有自己的眼线，这也是他能多次逃脱的原因所在。所以这次行动，你们必须要保持低调，我这边不能提供给你们任何帮助，不然很有可能会提前被普提查获悉消息。”
<br/>　　“能确定普提查的具体位置吗？”罗昊对何帆和葛铭问道。
<br/>　　罗昊决定要加快行动节奏，很明显，这次绑架是普提查背后那人的手笔，如果自己这边不加快行动节奏的话，佟千雪很有可能会被转移出T国，把佟千雪在T国境内拦截下来，毫无疑问是最佳的选择。
<br/>　　葛铭打开一张卫星地图，用一支红色的记号笔在一个位置画了一个圈，说道：“就是在这里。这里以前是个渔村，被普提查霸占后，就成了普提查的老巢，普提查在这里经营了三年时间。”
<br/>　　“三年的时间，T国警方居然无动于衷？”
<br/>　　齐凡冷哼一声，显然是对T国警方的不作为觉得意外；不过，转念细想，普提查在T国警察内部有他的人，而对方能从普提查身上获取客观的利益，又何必冒着生命危险去把普提查的势力打掉呢？
<br/>　　“渔村里的毒贩人数在150~200人之间，装备精良。”葛铭向战术中队众人介绍渔村中的情况，说道：“渔村中除了普提查的人外，还有原本生活在那里的渔民，那些渔民男人和老人成了普提查的劳动力，女人成了**。”
<br/>　　“在行动的时候，我们会多加留意。”罗昊点头，对葛铭问道：“能确定这些渔民被关押的位置吗？”
<br/>　　葛铭摇头，回答道：“我们的人没办法进入渔村，所以渔村内部的情况就只能依靠你们自己去了解。你们的装备在二楼。”
<br/>　　罗昊把那张卫星地图拿到自己面前，手指轻敲了几下，对葛铭说道：“葛队长，你这里有没有潜水用具？”
<br/>　　葛铭摇了摇头，回答道：“氧气罐需要定期维护，不方便长期储存，如果需要的话只能向国内申请支援。”
<br/>　　“如果从渔村西北方向的丛林抵近渔村，可以避免使用潜水用具，但是普提查在丛林中一定有所准备，如果是我，就一定会在丛林中布置陷阱和诡雷。”罗昊分析着说道：“就算我们可以在不被发觉的情况下通过丛林接近渔村，但是那样会浪费很长的时间，天色一亮就会失去优势。”
<br/>　　“我们在T国有空投点，我现在可以跟龙穴进行联系，要求龙穴空投潜水用具。”葛铭说道。
<br/>　　“如果是这样，那就最好不过。”罗昊说道。
<br/>　　葛铭没有任何耽搁，立刻起身离开了客厅，去跟龙穴取得联系；罗昊则是去了二楼，选择明天晚上行动所需要的武器。
<br/>　　……
<br/>　　T国东南方向，距离T国海岸线大约五十海里的位置，一艘关闭了引擎的快艇停在海面上，被涌动的海水推得左摇右晃。
<br/>　　快艇上，依旧是一件花格子衬衫、牛仔中裤、大墨镜打盘的葛铭正抽着烟等待着什么。
<br/>　　一架运-9运输机尾部舱门缓缓打开，被放置在滑轨上、四周绑上了漂浮装置的大箱子被推下运输机。运-9运输机在投下箱子后，立刻关闭了尾部舱门，直接离开。
<br/>　　从运输机上被推下后，箱子上的降落伞随即打开，朝着海面降落下去。箱子上的定位装置向葛铭手中的PDA发出信号，葛铭扔掉手中的烟头，发动快艇引擎朝着PDA上不断闪烁的红点位置接近过去。
<br/>　　噗通！
<br/>　　箱子落入海水中，猛地下沉下去，箱子没入海水一半的时候，绑在箱子四周的漂浮囊阻止了箱子继续下沉，白色的降落伞盖在箱子上，装着十几套潜水装置的箱子漂浮在海面上。
<br/>　　因为有定位装置的定位，葛铭很快就驾驶着快艇找到了箱子，关闭引擎，让快艇减速靠近箱子，用两根绳子连在箱子的牵引钩上，加以固定后，驾驶着快艇把箱子拖走。
<br/>　　……
<br/>　　深夜，一艘没有动力的小渔船在海面上飘荡着。
<br/>　　海浪打在小渔船上，令得小渔船不断上下颠簸着。
<br/>　　渔船船舱中，战术中队队员已经是换上了利用生物钢制作的仿生鲨鱼皮潜水服，背上了氧气瓶，拉下头上的潜水镜，走出船舱，坐在船舷上，咬住吸氧嘴，身体后仰，翻下渔船，沉入到海水中。
<br/>　　噗通！噗通！噗通！噗通！
<br/>　　伴随着小渔船不断的摇晃，十几个战术中队队员依次潜入到海水中，朝着普提查所控制的渔村方向游去，葛铭站在渔船上已经完全看不到漆黑海水中的情况。
<br/>　　葛铭继续在海面上待命，他后面的任务就是等待罗昊给他的信号，在得到罗昊发出的信号后，他驾驶渔船向渔村靠近，然后接应战术中队从渔村离开，经由水路跟一艘停泊在海中的货轮汇合，战术中队也将跟随这艘货轮撤离。
<br/>　　渔村中，那些被作为劳动力的渔民已经被驱赶回一间大木屋中关押起来，而那些被作为**的女人则是被毒贩带进自己所居住的地方。夜晚，对于这些关押起来的女人来说，就是噩梦的开始……
