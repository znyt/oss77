　　第十层空间之中，龙腾已经彻底的化为了洪荒祖龙，与之前龙灵所化的洪荒祖龙一般无二。
<br/>　　不过，龙腾的龙躯依旧在增长，已经达到了接近五千丈的地步。
<br/>　　其身上的气息，也同样在增长，已经达到了三品低级，但并没有停止的势头。
<br/>　　“可怕！”苏莫深深的吸了口气，龙腾这次的造化，真的是不同凡响。
<br/>　　龙腾成为了洪荒祖龙，将是龙族无上的存在，对于他的帮助，将会极大。
<br/>　　甚至，一举打破他目前的窘境，压制住帝一魂和拓跋青云两人。
<br/>　　昂~~~
<br/>　　龙腾扬天长啸，龙吟声震动诸天环宇，虽然他极为的痛苦，但是，声音中的兴奋却是不言而喻。
<br/>　　三品中级！
<br/>　　三品高级！
<br/>　　约莫七八个呼吸之后，龙腾的修为气息，达到了三品高级，和苏莫相同了。
<br/>　　但是，其气息依旧在增长，还是没有停止的趋势。
<br/>　　不过，这个时候，其气息增长的速度，已经在减缓了，没有之前那么凶猛了。
<br/>　　“应该能过四品吧！”
<br/>　　苏莫暗暗期待了起来，期待龙腾能一举达到四品的修为，按着现在的势头，有很大的可能。
<br/>　　他自然希望龙腾的修为能冲过四品，龙腾的修为越高，对于他来说也更加的保险。
<br/>　　时间缓缓流逝，又过了大约五个呼吸之后，龙腾的龙躯停止了增长，停在了九千余丈左右。
<br/>　　而龙腾的气息，已经达到了三品高级的极限，距离四品只有半步之遥了。
<br/>　　“冲不过去了吗？”苏莫见此，眉头微皱，不过，下一刻，龙腾身上的气息，立刻冲破了桎梏，猛然大涨，一举达到了四品低级的地步。
<br/>　　苏莫见此，顿时眼眸一亮，不过，这一刻，龙腾的气息也彻底的停止了增长，看来祖龙精血的力量，已经完全耗尽了。
<br/>　　昂！
<br/>　　龙腾庞大的身躯扭动，巨大的头颅抬起，扬天发出一声猛烈的长啸，震碎大片的星空。
<br/>　　这声龙啸之中，充满了兴奋与激动，蕴含着无法言喻的喜悦。
<br/>　　随即，彩色的光芒一闪，再次化为身穿锦袍的龙腾。
<br/>　　不过，此时此刻，龙腾的气质已然大变，变得悠远而深邃，变得威严而霸气。
<br/>　　嗖！
<br/>　　龙腾身形一闪，便来到了苏莫的身边，满脸喜色的道：“苏莫，这一次我真是要一飞冲天了？”
<br/>　　“明白！”苏莫微笑着点了点头，这是不用说的事情，对方现在几乎是第二个洪荒祖龙，自然要一飞冲天了。
<br/>　　“哈哈，你真的福星啊，跟着你果然奇遇无穷！”龙腾哈哈大笑着说道，现如今，他不仅血统变成了和洪荒祖龙一模一样，更是得到了洪荒祖龙的传承。
<br/>　　这传承之中，有洪荒祖龙修炼的道，还有各种强大的绝学。
<br/>　　“先不要高兴的太早，现如今，祖龙塔之外，肯定已经人满为患了！”苏莫笑容收敛了下来，面色凝重的说道。
<br/>　　龙腾闻言，面色也变得郑重了下来，沉声道：“是福是祸，只能见机行事了！”
<br/>　　“嗯！”
<br/>　　苏莫点了点头，随即环视四周，问道：“如何出去？”
<br/>　　此时，第十层的考验已经过去，但是，却是并没有离开的路。
<br/>　　“这个简单！”
<br/>　　龙腾点了点头，随即沉默了少许，手掌抬起，掌中有龙力涌动，缓缓的形成了一个海碗大小的符文。
<br/>　　龙腾大手一按，符文遁入了虚空深处，不见了踪迹。
<br/>　　少倾，这一方浩大的虚空，瞬间消失无踪，苏莫和龙腾两人，出现在了一个圆形的石室之中。
<br/>　　这石室就是祖龙塔的第十层，石室的墙壁，都是呈现青铜之色，其上刻画着大量的神龙图案，密密麻麻。
<br/>　　这祖龙塔，乃是一件超级神器，堪比终极神剑的存在，此刻，此神器已经被龙腾掌控了。
<br/>　　“走，我们出去吧！”龙腾招呼一声苏莫，便随着边缘处的楼梯口，向下一层而去。
<br/>　　此时此刻，祖龙塔之中，除了苏莫和龙腾之外，还有少量的龙族之人。
<br/>　　这些人数量不多，约莫二十余人，均是在苏莫之前进入的祖龙塔，不过由于祖龙精血被龙腾融合，所有人都被困在了所在层的虚空之中，无法出去。
<br/>　　而龙腾动用的符文，不仅解除了第十层的虚空，下方九层的虚空，也全部都消失了。
<br/>　　祖龙塔内的十层空间，都变成了正常的塔楼空间。
<br/>　　“怎么回事？”
<br/>　　“发生了什么事情？”
<br/>　　祖龙塔的的龙族之人，见到这种情况，均是心中惊疑不定，他们由于进来的早，一直在缠斗，并不知道外界发生的事情。
<br/>　　“你们，都出去吧！”
<br/>　　就在此刻，龙腾的声音，响彻在每一层空间，而后，祖龙塔之内传出一股浩大的力量，直接将众人强行送了出去。
<br/>　　……
<br/>　　祖龙塔之外，所有人依旧在等待，每一个呼吸的时间，对于众人来说，都度日如年。
<br/>　　时间越长，便越发的让众人心中惊疑，难道那屠霄，能有和龙灵所化祖龙一战的实力？这未免太过耸人听闻了。
<br/>　　就在此刻，祖龙塔第十层的塔壁之上，祖龙图案的光芒，陡然暗淡了下去。
<br/>　　不仅如此，如第二层、第三层、第四层等塔壁神龙图案之前同样闪亮的几层，也几乎是在同一时刻，全部暗淡了下去。
<br/>　　“这是……？”
<br/>　　所有龙族之人，看到这一幕，无不心神一震，个个都瞪圆了眼眸。
<br/>　　这是什么情况？
<br/>　　屠霄已经败了吗？
<br/>　　若是屠霄败了，只会是第十层暗淡下去，下面那几层的神龙图案，为何全部暗淡了下去？
<br/>　　难道屠霄胜了？
<br/>　　这个猜测，几乎让所有人心中狂震，如遭天雷轰击，大脑轰鸣不止。
<br/>　　不过，没有人急着冲进祖龙塔之中，因为，距离塔门最近的龙帝，都没有行动。
<br/>　　此刻，龙帝的心中，和普通的龙族之人差不多，同样大震。
<br/>　　不过，他的一双眼眸，却是留露出璀璨的神采，难道，屠霄真的得到了祖龙精血？
<br/>　　龙帝心中大动，他抬起脚步，就欲进入祖龙塔，因为祖龙精血一旦被人得到，祖龙塔内的阵法力量就会随之消散。
<br/>　　不过，就在此刻，祖龙塔的大门之处，空间一阵剧烈的波动，有数十道身影同时抛飞了出来。
<br/>　　唰！唰！唰！
<br/>　　约莫有二十余位龙族之人，全部在同一时刻，被强行送出了祖龙塔，差点撞在龙帝的身上。。
<br/>　　这让龙帝的脚步，为之一顿。
<br/>　　唰！唰！
<br/>　　就在此刻，两道挺拔的身影，紧随众人之后，大步的从祖龙塔内走了出来。
