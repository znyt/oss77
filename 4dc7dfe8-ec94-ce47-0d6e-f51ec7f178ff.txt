<br/>　　月影斩划开地面，木叶和木天瞬间朝着两旁躲开老远，他们看到过楚涵第二击月影斩的恐怖，那一刀劈开树林的可怕力量，让两名不同种族的王者都深深忌惮。
<br/>　　木叶还没来得及恢复伤势，就又一次的被迫在这片废墟地带逃窜，憋屈感让他后悔莫及，早知道就该第一时间来狼牙，取了楚涵的性命！
<br/>　　木天则更为情绪复杂，同样不断躲避逃窜之中，他还有更多的不解和惊恐。
<br/>　　因为这道月影斩的气息……
<br/>　　“你这招是从哪学来的？”木天忽然停在远处，朝着楚涵犀利质问，眼中满是不可思议。
<br/>　　刷！
<br/>　　楚涵手中的修罗战斧在半空中一个伦璇，甩出漂亮的弧度后单手握在身侧，他的双目直射向眼前这头丧尸，面无表情之中声音平静的婉如一汪寒潭。
<br/>　　“关你何事？区区丧尸。”
<br/>　　木天的瞳孔剧烈收缩起来，这是收到过最大的挑衅和羞辱。
<br/>　　区区？丧尸！
<br/>　　从头至尾，楚涵都没有把他当成过生命看待，而是与外面那些行尸走肉一样，不过是一头丧尸，一头稍微强大些的丧尸而已！
<br/>　　“木天，我们这次必须先撤。”忽然间，木叶开口，眼底满是忌惮和不甘。
<br/>　　木天死咬着牙，不甘心和愤怒，让他的双脚站定在原地不肯离开。
<br/>　　风声呼啸依旧，月影斩带来的恐怖能量不断在这片地带盘旋，地上裂开的巨大缝隙幽幽黑火灼烧依旧，范围颇大的地带到处都是无比恐怖的危险感，让木天和木叶所能移动的地方极其有限，他们不敢去触碰那些黑火，莫名的不敢。
<br/>　　楚涵却在这时候眼神一闪，双目眯起：“木……天？”
<br/>　　木天的呼吸一紧，心脏忽然激烈跳动起来，一种不好的预感油然而生！
<br/>　　楚涵却在这时候，毫无预兆的拉开回炉系统面版，迅速找到了木天的名字，忠诚度依然是两年前的那个数据，但木天的名字颜色却与任何一人都不同。
<br/>　　既不是普通人的正常颜色，也不是死去之人的灰色，而是尤其晃眼的绿色！
<br/>　　绿色？
<br/>　　楚涵的思维顿了顿，而后收起回炉系统的面版，朝着木天冷笑：“好一个丧尸王。”
<br/>　　并不张狂的声音带着嘲讽，没有太多的言语，却偏偏让木天瞬间暴怒。
<br/>　　“你算什么东西！闭嘴！”木天忽然大吼出声。
<br/>　　发狂中他能量暴涨，而后向一颗巨型炮弹般冲上前，朝着楚涵就扬起拳头！
<br/>　　噌——
<br/>　　楚涵不退不让，单手伦璇着修罗战斧，一下子挡住了这一拳！
<br/>　　刷刷刷！
<br/>　　充满爆裂感能量波动的威猛一拳，就这么与修罗战斧的斧刃面相碰，如同旋转的风扇一般，修罗战斧不断的伦璇将全部的能量都拦下，甚至还将部分反弹过去，导致木天的手背瞬间裂开了道道伤口。
<br/>　　木天暴怒着，另一只手也扬起，带着同样恐怖的威能，再次朝着前方狠狠一砸。
<br/>　　轰！
<br/>　　拳头与修罗战斧相撞，发出一声爆响。
<br/>　　但此时的楚涵却已经不知道什么时候，变成了双手握斧，不断伦璇的黑斧也顿在半空，就这么依靠着斧刃面，将木天的两只拳头都死死抵住。
<br/>　　“放肆！”一声冷喝，从楚涵的口中发出，带着强烈的压制感。
<br/>　　木天没由来的气势一滞，而后没由来的心头一阵恐慌，他快速退后好几步，将自己退到了现场最边缘的地带，远远的与楚涵避开。
<br/>　　旁边的尚九娣和陆玟丞都一脸莫名，不明白忽然间这丧尸王是怎么回事，明明可以穷追猛打的状态，却忽然收回了气势，硬是被楚涵压了一头。
<br/>　　楚涵却抬眼，眼底冒着即将爆发的怒火：“好一个木天！”
<br/>　　木天，木叶的亲弟弟，曾经死城之战中与楚涵并肩作战的一员。
<br/>　　但是眼下，他却是一头丧尸，还是丧尸王！
<br/>　　楚涵曾经让陈煜天试探过木叶身边那个异种阿大的身份，但却没有想到阿大是木天的同时，竟然还进化成了丧尸王。
<br/>　　这是何等的偶然？
<br/>　　还是命运的玩笑！
<br/>　　早已难以抑制的愤怒，让楚涵一下子爆发。
<br/>　　轰！
<br/>　　又是一道的月影斩成型，像是从天而降的巨大黑月，朝着木天狠狠砸去！
<br/>　　“杀戮鲜血让你忘了身为人类的底线了吗？”
<br/>　　“死城之战的荣耀你可还历历在目？”
<br/>　　“向黑暗低头你有过羞耻吗？”
<br/>　　“你这个懦夫！”
<br/>　　最后一声楚涵近乎是嘶吼的咆哮而出，迸发着对所有在今日死去狼牙战士们的感情，以及难以挽回的白允儿伤势，全部朝着木天压了过去！
<br/>　　这期间木天一动不能动，就像是被定身了一般的站在原地，他的心头满是恐惧和不甘心，以及渺小如同尘埃般的反抗。
<br/>　　他不服！
<br/>　　他是丧尸王！
<br/>　　但是为什么，明明已经不再是人类，还会对楚涵有诡异的臣服感？
<br/>　　眼前这个男人，到底对他施了什么魔咒？

<br/>　　此时的四周哪还有异种王的身影？
<br/>　　早在木叶将木天的名字报出口，让其吸引了楚涵全部的火力后，便趁机消失的无影无踪！

<br/>　　好一个兄弟情！

<br/>　　想要跪下，认错。
<br/>　　楚涵的双目冒着熊熊烈火，心头愤恨让他毫不犹豫的至强一击出手，要斩了这头丧尸！
<br/>　　可就在那黑月的恐怖威能触碰到木天发丝的瞬间——
<br/>　　哗！
