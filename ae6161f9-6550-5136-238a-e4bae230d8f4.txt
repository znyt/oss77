　　　　“冰河时代”
<br/>　　一道充满冰冷的声音，在这密密麻麻猴类的现场凭空响起。
<br/>　　霎时，诡异的事情发生了，已涂小安为中心的地面居然已肉眼可见的速度冻结了起来，一股深蓝色寒冰朝着四面八方蔓延，寒冰所到之处的地面全都变成了厚厚的冰。
<br/>　　最近距离的赤阳巨猿跟猿猴王看到自己的脚下冻结成冰，微微一震身子就溃散破解了。
<br/>　　“雕虫小技”赤阳巨猿轻蔑一笑，戏很足。
<br/>　　脚下旁的猿猴王却脸色大变，像想到了什么，抬头望去，他们的猴子猴孙们被深蓝色寒冰蔓延，连尖叫都来不及，就成了一只只冰雕。
<br/>　　这些冰雕那叫一个栩栩如生啊，每一个冰雕的动作跟面部表情都不一样，就像兵马俑一排一排的成型。
<br/>　　深蓝色寒冰就这样四面八方的一覆盖，万物皆冰，远远望去，还以为是深蓝色的海洋，一望无际，美不胜收。
<br/>　　涂小安淡淡邪笑，雕虫小技？你来一个我看看。
<br/>　　元素属性能力，最大的一个特点就是群杀绝技，现场无数觉醒猴，精怪猴，全部一个转眼成了冰雕，三阶以下，没人能挡得住涂小安的冰霜之力。
<br/>　　冰封完成，涂小安缓缓的站起来，不知何时，他的周身一柄一柄薄如蝉翼的飞刀环绕的飞起来，从一柄增加到了九柄。
<br/>　　九柄飞刀随着涂小安的一个念想，咻咻咻的在空气中发出破空声，一闪而过。
<br/>　　只见一只冰冻的觉醒猴的脑袋从肩膀上硬邦邦的滑落，澎...的一声掉在地上，就跟一个西瓜掉在地上一样成了稀碎，碎冰一地。
<br/>　　咻咻咻...嘭嘭嘭...
<br/>　　一只只猴子冰雕的脑袋被飞刀一穿而过，掉在地上，终结生命。
<br/>　　飞刀成了镰刀，收割觉醒猴跟精怪猴的生命，它们死的没有一点痛觉，甚至死了都不知道自己已经死了，莫名其妙就从这个世界离开。
<br/>　　“叮...宿主杀死一阶15级觉醒猴，获得75点进化能源!”
<br/>　　“叮...宿主杀死一阶16级觉醒猴子，获得160进化能源！”
<br/>　　“叮...宿主杀死二阶21级精怪猴，获得580进化能源！”
<br/>　　叮...叮...叮...
<br/>　　飞刀掠过，涂小安的进化能源顿时暴涨起来，叮叮当当的系统声响彻不断。
<br/>　　如此多的觉醒生物集中站在一起，简直成了涂小安的练级刷怪的绝佳场所，进化能源来的简直不要太轻松。
<br/>　　由于是击杀，而不是吞噬，所以获得的进化能源会减半，但是这已经无关紧要了，因为丝毫不影响进化能源的暴涨。
<br/>　　短短不过几十秒，上百只觉醒猴跟精怪猴就毫无知觉的死了。
<br/>　　“住手，快给本王住手”
<br/>　　猿猴王回神，勃然大怒，死的都是猴族的精英呀，一下次就死了上百，简直元气大伤。
<br/>　　要知道刚才那一伙人类闯进来大杀四方，死的猴族数量或许还没有现在那么多呢。
<br/>　　觉醒猴，还好，但是培养一只精怪猴可就不容易了，最少需要上百年的时间啊。
<br/>　　“蛇精，你在找死”
<br/>　　若在不阻止对方，猴儿山的精兵强将都要被杀死。
<br/>　　赤阳巨猿石柱般的粗暴手臂延伸，朝着涂小安就一拳砸了过去，空气直接扭曲。
<br/>　　这一拳，何止万斤，一拳如山。
<br/>　　涂小安随意的瞥了一眼，无动于衷，牛妖王直接进入战斗状态，残暴的气息迸发，成为牛头人，一拳迎了上去。
<br/>　　轰的一声。
<br/>　　牛妖王跟赤阳巨猿都双双倒退数步。
<br/>　　“牛妖王，你敢拦我，以前小打小闹可以，但你这次可是过火了”赤阳巨猿咆哮怒河，金刚般的吼叫起来，就跟牛妖王混打在了起来。
<br/>　　这一打，气浪席卷，一只一只冰雕怦然粉碎。
<br/>　　这一幕，吓得赤阳巨猿连忙收住力气，但却被牛妖王狠狠一拳的打飞出去。
<br/>　　涂小安不管不顾，系统中的进化能源涨的让他振奋，短短时间，已经比吞噬一头三阶妖灵还要夸张了。
<br/>　　放眼望去，数以万计的冰雕密密麻麻的林立，当然，绝大部分还是一些没有觉醒的普通猴。
<br/>　　可要是将数以万计的冰雕全部震碎，那么对于这个猴儿山来说，绝对是致命的打击，不说灭绝，起码死了一大半。
<br/>　　毕竟这也不是猴儿山的全部猴儿，但是精英绝大部分都来了。
<br/>　　九柄飞刀漫天掠闪，破空声连连，带着一只只觉醒猴的性命。
<br/>　　赤阳巨猿被牛妖王缠住，脱不了身，于是，精瘦的猿猴王终于坐不住了，这已经不是蛇精有没有资格像它挑战的问题了。
<br/>　　他被逼的不得不出手。
<br/>　　涂小安的眼前空间一阵涟漪，猿猴王原地浮现，他怒气冲天，瞪起了眼，脸上暴起了一道道青筋，愤怒地盯着他。
<br/>　　嗯？
<br/>　　好快！
<br/>　　涂小安还没反应，直接倒飞出去，一飞就是数百米之外，他的周身金光气焰凝固，金属之体显化。
<br/>　　没有这个金属之体，涂小安现在就成了一条紫金蛇了。
<br/>　　“没有人敢如此在我猴儿山为非作歹，妖王都不敢，你区区一条蛇精到底谁给你胆”
<br/>　　猿猴王声音还在数百米外，可下一秒，又出现在涂小安的眼前，一只手掐住了涂小安的喉咙，直接给提了起来，怒火中烧！
<br/>　　“放给我”涂小安脖子被掐住，并不惊慌，声音淡漠的说。
<br/>　　猿猴王蹙眉，手更加的用力，如没有金属之体，涂小安这虚假的身体早就爆了。
<br/>　　“不放吗，你可别后悔”涂小安的意识力骤然投放，投放到了附近的几百斤冰雕之上。
<br/>　　他妖治的眼眸浑身一眯。
<br/>　　澎...澎...澎...
<br/>　　顷刻间，几百斤冰雕在一瞬之间，全部粉碎。
<br/>　　“不...不....”猿猴王肝胆俱裂，猴儿山几百精英就这样莫名其妙的没了。
<br/>　　“叮...恭喜宿主杀死265只觉醒猴，获得一波14030进化能源”
<br/>　　进化能源条：81670/100000。
<br/>　　短短时间，涂小安就在猴儿山获得了三万多的进化能源，仅仅刚刚一会儿。
<br/>　　如若他想，这数以万亿冰雕的猴子要全部死绝，想增加多少进化能源都没有问题。
<br/>　　如此大的杀戮，还是涂小安第一次那么做，但是他并没有什么太大的感觉，被冰冻的猴子，死了连一丝血都没有溢出。
