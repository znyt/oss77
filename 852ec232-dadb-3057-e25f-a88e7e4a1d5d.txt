　　“三清圣地，擅闯者死！”
<br/>　　元始刻下此八字，特别是其中两字，元始还用上了神通，其中，“圣”字元始用上不周印，平时不会怎样，但要是有人想要破坏石碑，圣字就会显现出滔天威压，镇压天地。
<br/>　　而另一个字是“死”字，元始用上了杀道，字内充满狂暴的杀意，杀气冲天，尸山血谷，煞气滔天，叫人防不胜防防。
<br/>　　将石碑插入山口，以此警示世人。
<br/>　　“大兄，没想到你的阵法水品如此之高，有时间我们讨论讨论。”这时，通天走了过来，兴奋道，像是找到了同道。
<br/>　　通天本身就是阵道大家，盘古所传承的阵法他全都学会，而且还自创了许多阵法，威力不俗，现在见识到元始布下的阵法，神威玄妙非凡，顿时有了探讨的心思。
<br/>　　“不急，我们先前往东海，到了那儿，我们再一起论道也不迟。”元始淡言道。
<br/>　　“善！”两人同意，决定先到东海三仙岛。
<br/>　　既然有了决定，元始三人也就不再耽搁，他就是说一不二的人，立刻前往东海三仙岛。
<br/>　　就在元始布置大阵之时，西昆仑玉京山上，鸿钧老祖似有所感，他睁开双眼，掐指一算，但此时正值天地大劫，天机混乱，他也并没有算出什么。只是喃喃道：
<br/>　　“变数的气息？又会是谁呢？”
<br/>　　之后，便不再多管，继续闭目参悟大道，毕竟他的大劫也要到了，没有太多时间来管这些事，就算要管，也是等大劫过后再说。
<br/>　　同时，西昆仑的另一个地方，一个道人身披青色长袍，满头银发飘散，也不打结，任由它随风飞舞。他满脸潇洒之意，腰挂碧玉箫，手提血红葫芦，斜卧在地，但却毫不在意。
<br/>　　此时，他也许是察觉到什么，微微一笑，道：
<br/>　　“哈哈，有趣，有趣！潇潇洒洒任我游，自自在在散圣仙。一道飞虹飞天外，三界五行任遨游！哈哈！”
<br/>　　……
<br/>　　就在元始三人前往三仙岛这段时间，洪荒之中三族大战也刚刚停止，这些年，龙族大举进攻麒麟一族，凤族也跟着凑热闹，想在两族大战之中捡便宜，可是事态出乎意料，凤凰族这一凑合，龙族与麒麟族大战就渐渐演变成了三族大战。
<br/>　　刚开始龙族还占绝对上风，可是后来随着凤族的掺和，三族互相提防，谁也不信任谁，龙族靠着绝对的实力也很难取胜，这一场战争整整持续了两百年。
<br/>　　大战不止，小战更是不断，三族都伤亡惨重，龙族就算取胜也是惨胜，最后不得不停止大战，各自回家休养生息，等待下一次大战。
<br/>　　同时，凤凰族与麒麟族都认识到了龙族的强大之处，两族都想要联合，共同抵抗龙族。但两族之中，谁不是心高气傲之辈？他们谁也不服谁，各怀鬼胎，联盟一点也不团结，看似强大，实际上脆弱不堪。
<br/>　　这场大战影响深远，天地间众多大能都被三族的底蕴震惊了，纷纷避世不出，在各自的道场参悟大道。
<br/>　　五年后，元始带领老子与通天登上永恒岛。永恒岛上，还是一如既往地漂亮辉煌，两人初次见到，就喜欢上了这里。
<br/>　　“永恒岛！好名字。”通天称赞道。
<br/>　　“哈哈，三弟你却是不知，当初吾初到此岛，此岛名为‘蓬莱’，但吾却是不甚欢喜，便将它换了名称。”元始笑道。
<br/>　　“蓬莱？蓬莱也不错啊，不过，既然大兄不喜，那换了又何妨？我辈修士，本就是求个自自在在，不喜欢就不要它。”通天道。
<br/>　　“来，今日为兄定叫你们领教一番我这永恒岛玉虚宫。”元始领着两人沿着白玉石阶而上，领略一路风光，最后来到玉虚宫之前停下。
<br/>　　“玉虚宫，好名字，岛好，名好，宫也好。”
<br/>　　通天两人见到美轮美奂的宫殿，也的确是被惊到了，不是没有比这更好的，而是他们之前一直住在洞府之中，也没有太过在意，尽管洞府收拾了一番，算是很好了，但比起眼前这座玉虚宫，明显是大大的不如。
<br/>　　两人一见到这里，就喜欢上这里，嚷着要住在这里。
<br/>　　“此处有三座仙岛，呈三才之势，另外两岛分别为‘方丈’、‘瀛台’，等会儿为兄带你们前去，你俩一人一座，自己想要布置成怎样就布置成怎样，吾不会管你们。”元始道。
<br/>　　就这样，元始带着两人来到另外两座仙岛，经过一番争论，主要是通天争论，老子无所谓，最后，通天选了瀛台岛，老子选了方丈岛。
<br/>　　通天的瀛台岛上有顶级灵根先天五行果树，而老子的方丈岛上虽然也有顶级灵根，但没有像通天那样的顶级先天灵根，但老子无为，好清静，无所谓，反正都是自家兄弟，今后要吃，难道兄弟还会不给吗？
<br/>　　值得一提的是，不管是通天，还是老子，对于仙岛的名字也都有所不满，都想要修改岛名。要是没有元始的前例，两人还不会觉得怎么，但有了元始做榜样，两人顿时有了改名的想法。
<br/>　　最后，通天因岛上有先天五行果树，故而岛名直接改为“五行岛”，而老子的方丈岛却直接以他的名字来命名，为“太清岛”，符合他的清净无为。
<br/>　　太清岛？这名字也很好，搞得元始都想着要不要自己也换一个岛名，叫玉清岛，通天的改为上清岛，从此后三仙岛就成了三清岛。
<br/>　　呃？想多了，最后，元始还是决定没有改名，就按如今这个岛名。
<br/>　　故而，之前的三仙岛蓬莱、方丈、瀛台变成了现在的三清岛永恒、太清、五行。
<br/>　　“你们先按自己想法布置自己的岛屿，吾就先走了，百年之后，再来我玉虚宫论道。”
<br/>　　最后，元始对着两人说道，毕竟那是他们自己今后要住的地方，本应该自己布置，他也不想打扰。
<br/>　　同时，他又扔给两人各自一枚玉牌，那是元始自己所炼阵牌，并叮嘱两人，持此玉牌，进出永恒岛畅通无阻。
<br/>　　元始走后，两人也回各自仙岛布置起来。
<br/>　　百年之后，老子通天来到永恒岛玉虚宫，元始将他们引进演道场，接着，元始拿出两枚黄中李，让与他们。
<br/>　　“这——这是黄中李？！”通天惊叹道。
<br/>　　就连老子那一向面目表情的脸上都充满了惊诧，实在是十大先天灵根太出名了，每一种都绝世无双，现在他们竟然有机会偿到，由不得他们不惊讶。
<br/>　　“不错，这就是十大先天灵根之一的黄中李，它便是吾在永恒岛上发现的。黄中李十万年一开花，十万年一结果，再过十万年，方能成熟，整整三十万年才结出九个果子，确实十分珍贵。你们就不要浪费了，赶紧服下。这黄中李内蕴含道则，看看你们能否领悟？”
<br/>　　两人也知道元始的脾气，那是说一不二，就不再矫情，接过黄中李，立即服下，紧接着闭目炼化，感悟其中隐藏的道则。
<br/>　　元始也取出一个黑色黄中李果子，放到口中，入口即化，化为精纯的能量充斥全身。那能量之中，隐隐带有黑色神纹，那是破碎的道则碎片。
<br/>　　“这…这是灵魂法则？”元始惊讶道。
<br/>　　之后，便不再乱想，赶紧闭目参悟。
