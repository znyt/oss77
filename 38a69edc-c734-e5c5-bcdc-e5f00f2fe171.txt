　　在白文墨的提醒下，王玄策感觉颠覆高昌，然后搅动西域风云是一个不错的选择，但是这其中也有很多问题，比如：颠覆高昌的过程不能让人看出来其中有大唐的手脚之类。
<br/>　　那么他就必须通过一些特别的方式来推动事情的发展，以达到自己的目的。
<br/>　　当然，在高昌内部，选一个合适的代理人，让这个代理人成为一切事件的幕后推后，一担事有不谐直接把这个家伙干掉，这是最好的选择。
<br/>　　而这种选择的一个弊端就是，代理人不能太精明，太精明的人总是不那么容易摆弄，弄不好偷鸡不成蚀把米，自己丢人事小，搅乱大局才是最要不得的。
<br/>　　王玄策不想给自己留下办事不利的坏名声，所以他对选人方面十分慎重，像马万杰这种人，到底要不要用，他是一直持保留态度的。
<br/>　　不过马万杰很快就给了王玄策一个意外的惊喜，在他犹豫不决的时候，试探着问道：“王史君，大唐会干预西域国主的更替么？”
<br/>　　王玄策压下心头的喜悦，表面上不动声色的说道：“不会，西域诸国是主权国家，如何选择君王是你们自己的事情，与大唐没有关系。”
<br/>　　“哦，史君不要误会，我的意思是最近国主身体有些欠安，总是说自己心口有些痛，我也怕……”没有意料之中的斥责，马万杰的胆子也渐渐大了起来。
<br/>　　该死的鞠文泰他真是伺候够了，这个心高气傲又极度自卑的家伙，整天就活在自己的世界里，在这个大唐军队就驻扎在城外的时刻，一个不好可能把整个高昌都拖进无底的深渊。
<br/>　　而且鞠文泰这老头子，不顾马家三代忠良辅佐高昌数任国主，不管事情对错，动不动就对他非打即骂，一连脸面都不给马万杰留，这也是老马对他生出逆反之心的原因。
<br/>　　马万杰和王玄策，两人都是精明无比，都在试探着对方，但是因为各自的靠山不同，老马无疑是处在弱势的一方，不管干什么他首先都要考虑王玄策，或者说大唐的态度。
<br/>　　如果说大唐并不希望高昌有变，那么他所有的计划都必须胎死腹中，反之的话，那操作性就大了许多，甚至如果有可能的话，老马以为自己或许可以试试登顶的滋味。
<br/>　　王玄策闻歌知雅意，马万杰特地说明不要误会，分明就是让人去误会一下嘛。
<br/>　　王玄策心中冷笑，表面上却不动声色：“我没有误会什么，国主身体欠安就要好好休息，如果支持不住的话就选一位王子出来暂代国主之责，必竟以后的日子还长，身体累坏了对高昌，对大唐都是一个很大的损失。”
<br/>　　“是啊，多谢王史君的关心，我回去一定嘱咐国主，督促国主安心养病。”马万杰眉头舒展，脸上露出一丝笑容。
<br/>　　他想要投靠大唐，但是却得不到信任，这原本就是意料之中的事情，如果王玄策真的一口答应下来，说不准他还真就不敢投了。
<br/>　　这是他对王玄策的第一个试探，本是抱着可有可无的心态，就算是将来有一天被王玄策在鞠文泰面前揭发，他也可以用想要打入大唐内部，探听消息为由来解释。
<br/>　　唐军虽然是他们自己请来的，但是如果说唐军真是一心为了扫平马贼而来，这话估计连其蠢如猪的鞠文泰自己都不会信。
<br/>　　所以如果马万杰用打入唐军内部探听唐军目的为理由的话，在鞠文泰那里的确可以蒙混过关。
<br/>　　当然，马万杰的第一个试探失败了，而在失败的同时，也从侧面证明了，唐军在西域的确是另有所图，而且所图甚至大。
<br/>　　其中道理很简单——无欲则刚！
<br/>　　如果大唐没有任何图谋，完全可以理直气壮的将他保下来，而不会担心因为保下他而带来的其他变故，影响到自己的计划。
<br/>　　就因为这样，马万杰撞着胆子进行了第二次试探，而试探的结果无疑是让他又惊又喜。
<br/>　　惊的是，大唐使者竟然会不管属国国主的死活。这是一种极不合常理的作法，就算是属国之间有战争，大唐都有调解的义务，更别说事关属国国主的生死大事。
<br/>　　喜的是，大唐如果连高昌国主的死活都不管，那么所图一定甚大，很有可能是整个西域。这样一来的话，自己作为第一个投靠过去的人，得到的好处无疑将是最多的。
<br/>　　马万杰是聪明人，很聪明，这是王玄策对他的评价。
<br/>　　这种人就像是武器中的双刃剑，用的好可攻可防，用的不好，伤人伤已。
<br/>　　所以回到大营之后，王玄策十分犹豫，主动找到白文墨，打算和这个好为人师的老家伙好好聊聊。
<br/>　　“玄策，你想的多了。”白文墨在听了王玄策的顾虑之后，笑着摇摇头：“在绝对的实力面前，一切阴谋都不足畏惧。”
<br/>　　言罢看着依旧有些不解的王玄策说道：“不要忘了，你身后有五万大唐将士在为你撑腰，既然你说那马万杰是聪明人，那么他就一定会看到这一点，而看到这一点之后，你认为他会怎么作？”
<br/>　　王玄策沉吟着，琢磨着白文墨话中的含义，半晌之后才说道：“先生的意思是那马万杰值得相信？”
<br/>　　结果白文墨却再次摇了摇头，看着王玄策怜悯的说道：“玄策，难道你现在还不明白？这一次让你过来只是殿下给你的一个锻炼机会，你在这里可以尽情的折腾，完全没有必要担心其它。就算那马万杰背叛了又怎么样，就算是他把你的计划说给鞠文泰听又能怎么样？对于最后的结局完全没有任何改变。”
<br/>　　“可是如果被人看出大唐打算吞并西域呢？到时候大唐就会陷入被动。”王玄策接过话头，将白文墨打断。
<br/>　　白文墨淡然一笑，看着王玄策说道：“这个天下有很多聪明人，难道你以为现在就没有人看出大唐有吞并西域的目的？”
<br/>　　“那，那我们还在等什么？直接动手不好么？”王玄策突然有一种深深的挫败感，原来一切算计都是徒劳。
<br/>　　“不，这并不一样，这事就像是做生意，所有人都知道商人是想把东西卖给自己，把自己的钱装到他的口袋里，但是有些商人可以让人花钱花的心甘情愿，而有一些却需要强行把东西塞给顾客。”白文墨在王玄策的肩膀上拍了拍，结束了这次谈话，将沉思中的王玄策一个人留在帐篷里，独自离开。
