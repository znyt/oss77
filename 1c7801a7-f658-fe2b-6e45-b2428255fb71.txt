        “本来就是妈妈先看中的，我为什么要夺妈妈的心头所好？因为妈妈您喜欢，所以我就拿来送给您。”童筱颖如实回答。
<br/>
<br/>        她这声“妈妈”倒是叫得顺口了，冷夫人听着也挺顺耳。
<br/>
<br/>        或许是因为她是自己闺蜜的亲生女儿的缘故，冷夫人对童筱颖也有了几分亲近感。
<br/>
<br/>        刚刚还一脸严肃，转瞬间，冷夫人的脸上挂上了笑容：“那我就收下了。”
<br/>
<br/>        “嗯。”童筱颖随即起身，向冷夫人微微欠了欠身，“那妈妈早点休息，我回房去了。”
<br/>
<br/>        “等等！”冷夫人又唤了声。
<br/>
<br/>        童筱颖停住脚步，等着冷夫人的下文。
<br/>
<br/>        “你若是真不能生，其实也并不一定非得跟小沉离婚。大不了，和小沉去领养一个便是，最好领养一儿一女，只要他们都姓冷，叫小沉爸爸，叫我奶奶。”冷夫人低声沉吟着。
<br/>
<br/>        “嗯，谢谢妈妈。”童筱颖欣慰地笑道，“妈妈，晚安。”
<br/>
<br/>        离开冷夫人的院子后，走在回去的长廊上，童筱颖只觉自己的脚步轻盈如飞。
<br/>
<br/>        傅家大庄园里有亭台楼榭，住在这里，往往会有种好像回到古代王侯府邸的错觉。
<br/>
<br/>        这里比起外头那些现代化的豪华别墅，多了一份时空交错的美感与历史带来的沧桑，让人忍不住对这里有了憧憬。
<br/>
<br/>        回去的路上，童筱颖走得格外慢了点。
<br/>
<br/>        心血来潮时，她又坐在廊椅上，看夜空里的星星。
<br/>
<br/>        之前在实验室里，透过玻璃屋顶，她也看过这夜空的繁星，有了先后的对比，原来拥有自由的星星，比被禁锢的星星更美。
<br/>
<br/>        在房里等了许久都未见童筱颖归来，傅夜沉只好出来寻找。
<br/>
<br/>        当他走到去自己母亲的院子长廊时，隔着老远就看到了灯笼下，坐在廊椅上的童筱颖，暗自松了口气。
<br/>
<br/>        傅夜沉走过去后，脱下了自己的外套，披在了童筱颖的身上，童筱颖这才坐直身子，看向傅夜沉。
<br/>
<br/>        他在她的身旁坐下，轻声问道：“我妈为难你了？”
<br/>
<br/>        童筱颖微微摇头，眉开眼笑地回答：“妈妈说，如果我真不能生孩子，也可不用跟你离婚，因为，我们可以去领养一对孩子。”
<br/>
<br/>        “领养一对孩子？”傅夜沉错愕。
<br/>
<br/>        童筱颖笑着接着说道：“就是一儿一女啊！”
<br/>
<br/>        “你都没跟我试过，怎么知道你不能生？”傅夜沉嘴角微扬，抬起手来，轻轻地撩拨了一下面前这个小女人的下巴。
<br/>
<br/>        童筱颖瞬间面红耳赤地偏过头去，望向长廊外的荷塘。
<br/>
<br/>        她对这个男人的感觉很奇怪，有个时候会脸红心跳，但有个时候又会对他有那么一点点的抗拒。
<br/>
<br/>        与其说是抗拒，倒不如说是害怕与这个男人亲热。
<br/>
<br/>        她也不知道自己为什么会有这种莫名其妙的心悸感。
<br/>
<br/>        难道跟自己的过去有关吗？
<br/>
<br/>        “筱颖，你在想什么？”傅夜沉轻声问道。
<br/>
<br/>        童筱颖回过神来，转脸看向傅夜沉的俊脸。
<br/>
<br/>        波光粼粼反射到他的脸颊上，让他棱角分明的脸部线条看起来格外硬朗。
<br/>
<br/>        她隐约间能感觉得出，这个男人以前一定不是现在这种性子。
<br/>
<br/>        他对她表现出来的，是极其的耐心，还有无微不至的关心，以及奋不顾身的舍命，点点滴滴都让她很是感动。
<br/>
<br/>        她能接受他是自己的老公这一事实，也能接受他的家人，可为什么接受不了将彼此的关系更近一步？
<br/>
<br/>        这个心结的问题到底在哪里？
<br/>
<br/>        就好像那几天，她去威尼斯旅游，仍旧会在每个夜晚梦见他。
<br/>
<br/>        明明知道他留给她的联系号码，却又强制自己不去跟他联系。
<br/>
<br/>        好吧！
<br/>
<br/>        她去旅游的那几天，其实他也没主动联系过她。
<br/>
<br/>        两人的关系，真的很奇怪。
<br/>
<br/>        明明甚是想念，却不与对方寒暄。
<br/>
<br/>        “大哥，你是不是有事瞒着我？”想到这里，童筱颖若所有思地问。
<br/>
<br/>        傅夜沉淡定地笑了笑：“与其说我有事瞒着你，倒不如说，你想从我这儿知道什么事。”
<br/>
<br/>        童筱颖抿了抿唇，眼珠子一转，问的却是一句无厘头的话：“大哥，是不是我想要什么，你都会给我什么？”
<br/>
<br/>        “嗯。”傅夜沉不假思索地应了声。
<br/>
<br/>        童筱颖双手一拍，十指交握在胸前，微微一笑：“我想要天上的月亮。”
<br/>
<br/>        她故意这么皮的，只为逗逗他。
<br/>
<br/>        傅夜沉仰头看向夜空，今夜只有繁星，没有月亮。
<br/>
<br/>        她原本以为这事为难到他了，岂料，他忽而笑了笑：“现在是无法给你摘月亮了，要不，我们先回房去休息，等到半夜三更，我再把月亮捧到你面前来。”
<br/>
<br/>        “真的假的？”童筱颖一脸狐疑。
