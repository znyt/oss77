        明司寒穿好衣服后，走到门口看到连星儿独自一个人把所有的事情往身上揽，故意冷嘲热讽地说道：“真是个蠢女人！你在这儿发什么大善心啊？再说了，我喜欢的人一直都是阿妍。如果你不是傅夜沉的女儿，我还真的懒得理你。”
<br/>
<br/>        “你给我闭嘴！”明护国看向明司寒，冷斥道。
<br/>
<br/>        连星儿直起身子，一脸难以置信地看向明司寒。
<br/>
<br/>        “从你选择我的那一刻起，我就想玩玩你。说真的，凭什么是你来选择我，而不是我选择你？就凭你傅家有钱有势吗？我告诉你，连星儿，我最看不起你们家这种仗势欺人。”明司寒故意提高了音调，傲慢无礼地说道。
<br/>
<br/>        贺思思的父亲大概明白是怎么一回事了，便对身边的下属说道：“看样子，这是两家的私人恩怨。既然如此，笔录里你就如实记录吧！这个案子就这么结了！”
<br/>
<br/>        “是！”笔录员点了点头。
<br/>
<br/>        贺思思的父亲，又礼貌性地向明护国颔首道：“明叔，既然是你们明家和傅家的私事，我就不插手了。小寒也不用跟我们去警局了，警局那边的事情，我会替你们去扫尾。”
<br/>
<br/>        “辛苦你了，谢谢。”明护国感激地点了下头。
<br/>
<br/>        贺思思刚想插嘴说什么的时候，却被自己的父亲给强行拽走了。
<br/>
<br/>        此刻的总统套房里，像死一般的沉寂。
<br/>
<br/>        明护国叹了口气，接着说道：“既然你们为了解除婚约这事，闹腾成这样，我便如了你们晚辈的意愿，解除你们的婚约便是。”
<br/>
<br/>        “明爷爷……”连星儿欲言又止。
<br/>
<br/>        明启辉上前搀扶着明护国，与随行的助理一起离开。
<br/>
<br/>        此刻的总统套房里，就只剩下连星儿和明司寒，还有裴萱妍。
<br/>
<br/>        连星儿再次看向明司寒，疾步走到他跟前，生气地质问道：“你这出的什么馊主意啊？你知不知道，你这么做差点把你自己给毁了！”
<br/>
<br/>        “至少我两的婚约解除了不是吗？答应过你的事情，我做到了。”明司寒不痛不痒地说道。
<br/>
<br/>        此时，裴萱妍只裹着一条白色浴巾，赤脚走到了明司寒的身后，对连星儿咧嘴一笑：“小丫头，你不要的男人，我收下了。谢谢你的不嫁之恩。”
<br/>
<br/>        连星儿被裴萱妍这句话弄得脸色半紫半红，气也不是，不气也不是。
<br/>
<br/>        甚至，她也不知道自己为什么会突然脱口而出地继续质问：“明司寒，你真的跟裴萱妍……”
<br/>
<br/>        “怎么？只许你跟别的男人暧昧不清，就不许我跟别的女人有染了吗？”明司寒大手一伸，打断连星儿的话时，也直接将裴萱妍揽入了怀中。
<br/>
<br/>        连星儿揪心地看着明司寒亲密地搂着裴萱妍的肩头。
<br/>
<br/>        裴萱妍还一副小鸟依人的样子，紧紧地贴着明司寒的身子。
<br/>
<br/>        连星儿不明白自己此刻为何会有一种想哭的冲动，一阵揪心过后又是隐隐的刺痛感袭来。
<br/>
<br/>        她深吸了口气，脸上艰难地挤出一抹微笑：“那祝你们幸福啊！毕竟，咱们也算哥们一场。”
<br/>
<br/>        连星儿说完，毅然转身，迈开步伐后，本想淡定高傲地走出去，却情难自控地仓皇而逃。
<br/>
<br/>        看着连星儿的身影彻底消失在玄关口，开门的声音过后迎来关门声。
<br/>
<br/>        明司寒立马翻脸，冷酷无情地推开了裴萱妍。
<br/>
<br/>        裴萱妍抹了抹额前的刘海，瘪了瘪嘴，反问道：“这戏，我陪你演完了！别忘了你答应过我，要加入我们组织的。”
<br/>
<br/>        “我可没答应要加入你们，我只答应你，你们有需要我的时候，我会尽力而为。”明司寒冷冷地说道。
<br/>
<br/>        裴萱妍嘴角微扬：“那也行吧！有你的黑客技术，能为我们组织锦上添花不少。”
<br/>
<br/>        明司寒睨了裴萱妍一眼，不再吭声。
<br/>
<br/>        裴萱妍看着明司寒，脑海里忽然想到了连星儿，又补充了一句：“老实跟你说吧！就我女人的直觉来讲，我感觉连星儿看你的眼神变了。”
<br/>
<br/>        “当然变了。估计，会很感激我吧！”明司寒苦闷地应了声。
<br/>
<br/>        裴萱妍只笑不语。
<br/>
<br/>        这没有恋爱经验的男人，还真的是纯情得跟张白纸一样，非得在纸上写明白了才看得懂。
<br/>
<br/>        她第一次在明司寒家门口见到连星儿的时候，连星儿那时候的眼神里对明司寒确实没有爱意。
<br/>
<br/>        但是，这一次，连星儿看明司寒的眼神变了。
<br/>
<br/>        看着两个明明相互喜欢的人，彼此却不知道彼此的心意，作为旁观者的裴萱妍觉得很是有趣。
<br/>
<br/>        裴萱妍不点破，不撮合，就一脸看戏的态度。
