　　第149章
<br/>　　此事过后，程大雷没有在落叶城耽搁，整队返回蛤蟆寨。
<br/>　　“将军，咱们这次威风了，回去我也能搞个百夫长当当。”出得城门，高飞豹骑在马上兴奋道。
<br/>　　“将军什么将军，在自己人面前叫我大当家。”程大雷没好气的白了他一眼，还真人给根鸡毛就当令箭，不知道自己几斤几两了。
<br/>　　高飞豹无语，程大雷这话怎么和在城里说得不一样了。
<br/>　　“程将军，程将军……”
<br/>　　身后传来呼声，程大雷扭过头来，呵斥道：“不是说不要……”
<br/>　　来的是齐德强，程大雷识趣的闭上嘴巴。
<br/>　　“程将，要走了？”齐德强拍马赶来。
<br/>　　“是啊，虽然我舍不得老齐你，但山寨一堆事要处理，终不能耽搁，不如老齐去我山寨盘桓几日，你我兄弟秉烛达旦，促膝长谈。”
<br/>　　齐德强摆摆手：“我也是公务繁忙，不去打扰程将了。不过确实有几个人要去蛤蟆寨？”
<br/>　　“什么人？”程大雷微微皱眉。
<br/>　　“哈，是主公派去的人，都是主公身边的人才，派过去也是帮程将军做些事。”
<br/>　　得，监视的来了。
<br/>　　程大雷问：“来几个？”
<br/>　　齐德强一直观察着程大雷的表情，知道他心里肯定不满这件事，于是试探着说：“有七八人吧。”
<br/>　　“老齐，你对不住我啊！”程大雷大叫。
<br/>　　“这个是主公的安排，我也没有办法，确实……”齐德强吓了一跳。
<br/>　　“像这样的人才，你怎么不多派几个过来，我最缺的就是人才啊！”
<br/>　　“……”齐德强腹诽：事情怎么和自己想得不一样。难道程大雷这个憨货，真看不出来其中关键。
<br/>　　人要有自知之明，这是程大雷一直告诉自己的一件事。真以为穿上这套狗皮，就能在幽州一地横着走了。幽州王之所以不让自己死在落叶城，是因为他要自己和山寨所有弟兄死在攻打黑石城的事情上。
<br/>　　没有人会给老鼠喂火腿肠吃，除非火腿肠放在捕鼠器中。如今自己已经吃了火腿肠，能逃脱将要落下的笼盖么。
<br/>　　如何和幽州王玩好这场猫鼠游戏，才是接下来的重头戏。
<br/>　　别过齐德强，程大雷带领队伍返回蛤蟆寨。在路上他使用了系统奖励的抽奖次数。
<br/>　　那扭蛋机状的事物再次在脑海中出现，随着咕噜噜旋转，一物从里面掉了出来，随之程大雷收到一条系统提示。
<br/>　　嘟，获得剑法：阿飞快剑（第一式）
<br/>　　马上的程大雷皱起眉头，这次系统奖励不是以秘籍的方式，随着一道明悟，自己已经掌握了这招剑法。
<br/>　　而至于这『阿飞快剑』……
<br/>　　程大雷查看系统介绍。
<br/>　　阿飞快剑：第一式直来直往，绝世级，残缺。
<br/>　　这次系统介绍很简单，正如这招剑法一样，直来直往，却能取人性命。废话，谁不知道古书第一快剑，所谓阿飞的剑，李寻欢的飞刀，陆小凤的灵犀一指都是bug级的，例不虚发。
<br/>　　这是第一次有江湖武学出现在系统奖励中，但令程大雷困惑的并不是这点。而是，连阿飞快剑这样的武学，也只能排到绝世级别，那么真正的传说级该有多强大呢？
<br/>　　天下第三的小李飞刀，叶孤城的天外飞仙，乔帮主的降龙十八掌，如果这些都不算传说，难道是那失传已久的辟邪剑法……
<br/>　　程大雷一身寒意。
<br/>　　好吧，也不知自己不断完成任务，系统会不会把这些江湖武学搞出来。更不知真正的传说级高手会拥有怎样的力量：一剑千里、凌空虚度……反正程大雷现在一个没见到过。
<br/>　　他如今见到的高高手，也无非就是秦蛮、赵子龙这个等级了。程大雷约也明白，优秀到顶级是道坎，顶级到绝世是道天坎，至于传说，估计要拼脸。不然程大雷也不会时至今日一个都见不到。
<br/>　　但程大雷身上有个bug，只要他不断建设山寨，实力就会随之提升。
<br/>　　传说，未必不可期。
<br/>　　『普通人才会拼努力，拼天赋，我们这种人通常只拼BUG』
<br/>　　回到山寨后，程大雷一刻也没有停歇，这次在落叶城全身而退，程大雷也没有想着摆个庆功宴什么的，立刻让所有人投入到建设山寨的事情上。
<br/>　　伐木建屋，垒石为墙，在开春之前，程大雷必须将山寨升到三级，成为青牛山内一座坚不可摧的堡垒。
<br/>　　幽州王的果子不是那么好吃的，吃了吐的滋味程大雷可是一丁点都不想尝试。
<br/>　　……
<br/>　　山道上，一队人马缓缓而来。
<br/>　　裴喜之坐在马车内，望着周围的山景，脸上露出凝重的神色。
<br/>　　他是李善言座下的幕僚，这次前往蛤蟆寨是执行监视的工作，当然美其名曰协助。
<br/>　　裴喜之不是很喜欢这份工作，山贼毕竟是山贼，据说有些山贼还会生吃人肉。但军师交待下的任务，自己也是不能不接。
<br/>　　坐在马车内，脑海内又响起临别前李善言与自己的密谈。
<br/>　　“知道自己这次去的任务么，调查他们是否真心接受招安，打探他们的底细，分化他们内部的关系，蛤蟆寨吞并五十家山寨，我不相信他们会是铁板一块。”
<br/>　　“任务我都明白，但军师，我担心这些野蛮的山贼……”
<br/>　　“你说的我都明白，但只有大勇之人能完成这次任务。你记得，你是个读书人，拼武力你不行，你要和他们拼头脑。”
<br/>　　调查、打探，分化……不和他们拼武力，要和他们拼头脑。
<br/>　　想到这些山贼会如何粗鲁的对待自己，裴喜之就不寒而栗，他不断催眠自己：我是一个读书人，我要用头脑碾压他们，让他们见识到智慧的力量。
<br/>　　前行便进入青牛山，斜刺里突然冲出一波人，将车队拦住。
<br/>　　“我们是幽州王的人，奉命来此处，你们想要干什么！”
<br/>　　“欢迎欢迎，热烈欢迎！”
<br/>　　一个留着山羊须的猥琐中年人过来，一把抓住裴喜之的手，热情道：“是裴大人么，真可谓数九隆冬盼春光，三更半夜盼朝阳，单身的老头盼大娘，盼星星盼月亮可算把您盼来了，裴大人就是春光，裴大人就是朝阳，裴大人就是我们的大娘！”
<br/>
