　　闭关室内。
<br/>　　萧逸震惊地看着手中断裂的白骨手杖。
<br/>　　“这是真正的荒影蛛骨骸？”
<br/>　　原本，萧逸以为那荒影蛛之骨，只是毒袍根据自身武魂所凝聚。
<br/>　　可现在，真正看细了，却发现，这根本就是真正的妖兽骨骸。
<br/>　　武魂凝聚而出，还是根本就是实体，这有很大的区别。
<br/>　　难怪那时毒袍阴蛰的眼眸中如此自信。
<br/>　　萧逸陡然想了起来。
<br/>　　毒袍此人，果真谨慎无比。
<br/>　　难怪他那时自信无人能损坏他手中白骨手杖，难怪他那时自信凝聚出的太荒之火，能够击杀脚踏血煞、风煞两条龙形煞气的萧逸。
<br/>　　萧逸陡然明白了过来。
<br/>　　毒袍当时所施，乃是‘荒骨苍炎’。
<br/>　　这一武技，乃是邪君府有名的武技之一。
<br/>　　以骨化火，凝聚太荒之火。
<br/>　　毒袍以真正的‘荒影蛛’骨骸，去凝聚出的太荒之火，将强悍到极其可怕的地步。
<br/>　　可惜，最后火焰还未成，就因极度自信下，被萧逸毁了手杖。
<br/>　　火焰不攻自散。
<br/>　　不过，让萧逸惊骇的是，毒袍到哪去寻得这荒影蛛的尸骸和骨头？
<br/>　　荒影蛛，可是传说中的太荒十兽。
<br/>　　太荒十兽，天地初生，太荒时期，应运而生，天地间也仅这十头。
<br/>　　谁能见过荒影蛛？
<br/>　　而且，在大陆传说中，炎龙的八道龙炎吐息，方让这十头太荒之兽灭绝。
<br/>　　怎么可能还会有尸骸存在？
<br/>　　那么，毒袍到哪去寻到这根骨头的？
<br/>　　萧逸想不明白，也只能稳下心头惊骇，不再多想。
<br/>　　嗖…
<br/>　　手中光芒再闪，取出了乾坤戒内一些功法典籍之类的东西。
<br/>　　萧逸看了几眼，随便扔回了乾坤戒内。
<br/>　　这些功法典籍，是毒修一类的功法。
<br/>　　不过，在萧逸的判断中，这跟万毒宝典相比，简直就是一堆垃圾。
<br/>　　倒是里头有大量的毒物和修炼之物。
<br/>　　萧逸将整个乾坤戒感知了一番，打算看看有没有关于邪君府的其他情报。
<br/>　　不过，如他所料，没有。
<br/>　　邪君府的邪修，向来谨慎，毒袍就更不用说了。
<br/>　　乾坤戒内，连半分情报卷宗都没有。
<br/>　　萧逸摇了摇头，嗖…嗖…嗖…
<br/>　　手中光芒大作。
<br/>　　十万邪修的乾坤戒，在萧逸一番感知后，被悉数取出。
<br/>　　里头，有不少邪器。
<br/>　　算到每个邪修本身，修炼之物、毒物，都极少。
<br/>　　但十万邪修加起来，却是数量不小了。
<br/>　　而且，妖兽精血，更是大量。
<br/>　　萧逸一直追杀这些邪修的目的，其中原因之一，便是要快速收集妖兽精血。
<br/>　　萧逸没有浪费时间，直接开始吸收炼化。
<br/>　　那些毒物，一一吸收殆尽。
<br/>　　体内毒丹，壮大了些许，但并不明显。
<br/>　　说起来，万毒宝典的修炼方式，几乎颠覆了整个正常的修炼体系。
<br/>　　真要验证，或者修炼个纠结，现在还做不到。
<br/>　　他日，毒丹若能继续成长，达到蜕变的地步，或许，方才是真正将万毒宝典验证透彻之时。
<br/>　　所有修炼之物，也一一吸收。
<br/>　　这些邪修身上的修炼之物，本就不多，虽然十万之数加起来看似不少；但对比萧逸的庞大小世界和气泉来说，同样不怎么够看。
<br/>　　嘭…
<br/>　　萧逸身上一阵气势爆发。
<br/>　　修为，突破了一重，达到圣皇境八重多些。
<br/>　　接下来，是那大量的妖兽精血。
<br/>　　这些妖兽精血，倒是为数不少。
<br/>　　一一吸收后，修罗战体，直接打到圣皇境九重的层次。
<br/>　　对比之前的圣皇境七重，直接跳了两重。
<br/>　　“圣皇境八重修为，修罗战体圣皇境九重层次。”萧逸捏了捏有力的拳头，脸色略带喜意，但仍旧微微皱眉。
<br/>　　“这等实力，有一定把握，但还是不够。”萧逸皱眉自语着。
<br/>　　他这些天的追杀邪修，也是变相的历练的一种。
<br/>　　他自然也要为之后的古帝之墓开启而做准备。
<br/>　　上次离开剑帝碑时，距离古帝之墓，还有七月时间左右。
<br/>　　现今算算时间的话，大概还有两月。
<br/>　　“还有两月时间。”萧逸握了握拳头。
<br/>　　以他现今实力，足以无视中域各大天骄。
<br/>　　至始至终，萧逸就从未将年轻一辈放在眼中，这等自信，他还是有的。
<br/>　　可他的目的，可不是单单地将年轻一辈看作对手。
<br/>　　这些年来，什么叫做打了小的，老的出来报仇，萧逸无比了解。
<br/>　　故他一直以来视作对手的，都从来不是年轻一辈。
<br/>　　另外一点，要真正闯过古帝之墓，绝非易事。
<br/>　　按当初云渊前辈所说，古帝之墓给下的考核，估计是按上古天骄所设下的。
<br/>　　而且，即便是上古天骄，也无几分把握能破。
<br/>　　古帝之墓，存在的岁月，不知凡几。
<br/>　　可至今，无人能破。
<br/>　　萧逸不知道上古时期的天骄，到底有何等惊才绝艳。
<br/>　　但大概也能有所猜测。
<br/>　　上古时期的手段，远非现今天骄能比。
<br/>　　那么，古帝之墓，连上古天骄都没几分把握能破，难度可见一斑。
<br/>　　在萧逸的对比中，现今自己实力，要说闯古帝之墓，乃是有把握的，可要说百分百的把握，却绝对不够。
<br/>　　“或许，此次猎妖殿总殿之行，能有收获。”萧逸思索着，自语一声。
<br/>　　最后的这两月时间，将是他最后的实力提升机会。
<br/>　　……
<br/>　　收拾好一切，萧逸离开了闭关室。
<br/>　　刚出闭关室，迎面便看到灸炎与唐音两位殿主。
<br/>　　“嗯？”萧逸皱了皱眉。
<br/>　　两位主殿主，如若‘门神’一般守在闭关室外，着实让他微微惊讶。
<br/>　　“两位殿主，这是…”萧逸皱了皱眉。
<br/>　　“呵。”灸炎笑笑，“易霄分殿主，你可是出了名的神出鬼没。”
<br/>　　“不看着你，怕是你又走没影了。”
<br/>　　“看来一番闭关，你伤势已痊愈；走吧，随我二人去一趟总殿。”
<br/>　　萧逸微微皱眉，“总殿，我会去，不过走前，我还有些事要做。”
<br/>　　“何事？”灸炎二人，皱了皱眉。
<br/>　　“没什么大事。”萧逸摇摇头，“只是谨慎下，去看看毒袍等邪修的尸体罢了。”
<br/>　　毒袍，还有那十万邪修的武魂力量，可还未吸收呢。
<br/>　　萧逸本想着，闭关之后，再去吸收，随后随这二人去总殿一趟。
<br/>　　灸炎点点头，“那走吧，我二人陪易霄分殿主去。”
<br/>　　“我想自己去。”萧逸语气微冷。
<br/>　　“不行。”灸炎摇摇头，语气强硬。
<br/>　　“那随你们。”萧逸眯了眯眼，紫炎火翼一振，瞬间御空而离。
<br/>　　“哼。”灸炎冷哼一声，同样化作一道火光，御空追上。
<br/>　　“好快的速度。”萧逸眼眸一惊。
<br/>　　嗖…嗖…
<br/>　　不过片刻，三人便回到了之前大战之下的废墟中。
<br/>　　“嗯？”萧逸环视四周，面具之下的脸色，顷刻大变。
<br/>　　“毒袍的尸体呢？”
<br/>　　周遭虽已是一片废墟，凌乱不堪。
<br/>　　但对于强者而言，一道感知，即可知晓这里所有情况。
<br/>　　“不好？”灸炎二人，亦是脸色一变。
<br/>　　“好个毒袍，之前诈死？”
<br/>　　“他应该还跑不远，我们追。”
<br/>　　嗖…嗖…
<br/>　　灸炎与唐音，连忙御空飞离。
<br/>　　萧逸皱了皱眉，未跟上，而是看向空气中，那十万邪修飘荡在天地间的武魂力量。
