　　　　轰隆隆！！
<br/>　　巨大的蛟龙速度快到极致，风驰电逝一般，庞大的身躯碾碎空间，急速到来。
<br/>　　而后，蛟龙在众人还未反应过来之前，巨口一张，一口将火炼八荒阵吞进了腹中。
<br/>　　“不好！”
<br/>　　阳鼎天见此，顿时心中一惊，因为阵法被东海龙王吞掉，他立刻就失去了对阵法的控制。
<br/>　　嗖！
<br/>　　远处再次有破空声响起，一名锦袍中年人，紧随蛟龙之后而来。
<br/>　　这名中年人龙行虎步，身上气息浩大精纯，赫然也是一名准帝强者。
<br/>　　“帝释！”
<br/>　　众人眼眸一凝，这紧随而来之人，正是帝玄宫宫主帝释。
<br/>　　“诸位别来无恙！”帝释快速到来，面色冷漠的说道。
<br/>　　“帝释，你这个丧家之犬，居然还敢现身？”殷棕冷笑着说道。
<br/>　　上次他和二魔老、北堂平华，三人联手追杀帝释，追杀了小半个苍穹世界，从中洲追到西海，又从西海追到北海。
<br/>　　“为何不敢现身？”帝释眸中冷意连连，眸光扫视殷棕五人。
<br/>　　若是他自己，他自然不会现身，但现在有东海龙王再次，他自然无惧。
<br/>　　言罢，帝释目光转动，最后落在了上官昊的身上。
<br/>　　“昊儿，你可真是让为师刮目相看啊！”帝释面色淡淡，仿佛对于上官昊所做的一切，并没有多少愤怒一般。
<br/>　　“哼！帝释，你我早已不是师徒！”
<br/>　　上官昊闻言冷笑一声，随即他身形稍稍移动，向二魔老等人靠近了一些，显然担心帝释突然出手。
<br/>　　“这个世界有因就有果，为师会清理门户！“帝释沉声道。
<br/>　　实际上，帝释的心中五味陈杂，有愤怒、有后悔、有失望，但更多的却是痛苦。
<br/>　　他一生阅人无数，没想到唯一的弟子却是看错了人。
<br/>　　其实，这也不能怪帝释看走了眼，以前的上官昊虽然为人狂傲，但心智还没有如此不堪。
<br/>　　只是武魂的丢失，加上帝释对于苏莫的看重，以及苏莫和夕儿的婚事，彻底的刺激了上官昊。
<br/>　　让得他陷入了极端，心里严重的扭曲了！
<br/>　　“清理门户？你有这个实力吗？”上官昊面露不屑之色。
<br/>　　嗖！嗖！嗖！！
<br/>　　就在此刻，二魔老、大魔老、殷棕等五大强者，身形齐动，瞬间便将帝释和东海龙王包围了起来。
<br/>　　其他人见此，立刻时速后退，不敢靠近。
<br/>　　大战一触即发！
<br/>　　……
<br/>　　神图空间中。
<br/>　　在火炼八荒大阵被东海龙王吞掉的那一刻，他立刻便睁开了眼眸，停止了修炼。
<br/>　　一位他明显的感觉到了阵法之力的变化，瞬间下降了一个档次。
<br/>　　“怎么回事？”苏莫心中惊疑，不明所以，因为大阵的缘故，他无法察觉到大阵之外的情况。
<br/>　　嗖！
<br/>　　苏莫立刻离开了山洞，飞到了高空之中，仔细查看外面的情况。
<br/>　　“怎么了？”
<br/>　　“难道空间宝器撑不住了！”
<br/>　　“不会吧？”
<br/>　　神图空间中的无数武者，见到苏莫现身，立刻都被惊动了。
<br/>　　其中有一些武皇强者，也飞上了高空之中，来到了苏莫的身边。
<br/>　　海明江也是第一时间出现，来到了苏莫的身前。
<br/>　　“苏宫主，什么情况？”海明江沉声问道。
<br/>　　苏莫并未回答海明，面色凝重，仔细的查看外界的情况。
<br/>　　“这……！”其他武皇强者见此，心中不明所以，但他们也不好再追问。
<br/>　　少倾之后，一直面色凝重的苏莫，眉头一挑，面上露出了一丝喜色。
<br/>　　“苏宫主，怎么了？”海明江见此，再次出口相问。
<br/>　　“海阁主，火炼八荒大阵在急速消融，要不了几个呼吸，差不多就能彻底消散了！“苏莫笑道。
<br/>　　“真的？”
<br/>　　海明江闻言大喜，随即又惊疑的问道：“这是怎么回事？不会是他们的阴谋吧？”
<br/>　　海明江有些不太理解，他们被困在大阵中，已经成了瓮中之鳖，但现在大阵怎么突然就要消散了！
<br/>　　这很有可能是阳鼎天等人，要用其他方法对付他们了。
<br/>　　其他人亦是一脸的惊疑，脸上并没有多少喜色。
<br/>　　“呵呵！只要大阵消散，我们就有逃走的机会”
<br/>　　苏莫轻笑一声，继续道：“你们放心吧，只要有一丝一毫的机会，我会带你们逃出去。”
<br/>　　众人闻言稍稍心安，他们对于苏莫的实力，还是非常信任的。
<br/>　　毕竟，苏莫名声在外，他们之前也都目睹了苏莫的实力，比之天涯海阁阁主海明江丝毫不差。
<br/>　　“苏莫小友，今日老夫若是能脱身，日后任凭差遣！”一名身形高大、胡须泛白的老者向苏莫抱拳。
<br/>　　苏莫扫了此人一眼，发现此人居然是武皇境八重修为，实力着实不弱。
<br/>　　“不知前辈来自何门何派？”苏莫沉声问道。
<br/>　　“呵呵！老夫一介散修，无门无派！”老者笑着说道。
<br/>　　苏莫闻言暗暗点头，一介散修，能达到如此修为，的确不简单。
<br/>　　“不知前辈可想过加入宗门，若是愿意的话，可入我苍穹神宫！“苏莫沉声道。
<br/>　　“这……！”老者闻言一怔，倒是没有想到苏莫会邀请他加入苍穹神宫。
<br/>　　略一沉吟，老者点了点头，苦笑道：“今日若能逃过一劫，他日老夫毕竟前去！”
<br/>　　老者没有拒绝苏莫的要求，毕竟他也知道，苍穹神宫可是远古霸主，底蕴强大无比，若是能加入苍穹神宫，对他绝对是有天大的好处。
<br/>　　当然，一切都有一个前提，那就是今日能逃过一劫。
<br/>　　“苏宫主，若是今日能大难不死，在下也愿意加入苍穹神宫！”这时，一名低阶武皇立刻向苏莫抱拳说道。
<br/>　　此人之前在外面差点葬身火海，幸的苏莫所救，所以也表示愿意加入苍穹神宫。
<br/>　　“苏宫主，我也愿意加入！”
<br/>　　“在下只有武王境六重修为，不知道苏宫主可愿意收留在下？”
<br/>　　“苏宫主，还有我们，我们也愿意加入苍穹神宫！”
<br/>　　随着老者和那名低阶武皇表示要加入苍穹神宫，霎时之间，整个神图空间之中，起码有一半的人表态，愿意加入苍穹神宫。
<br/>　　这些人之所以要加入苍穹神宫，则是由几重原因。
<br/>　　其一是苏莫对他们有救命之恩；其二则是苍穹神宫不同于其它势力，传承底蕴强大；其三则是很多人都有亲人朋友死于火炼大阵之下，他们跟随苏莫之后，才能有机会报复黄泉魔宗和阴阳神宫。
