　　揉了揉朦胧的美眸，望着负手立在窗前沉思的慕清霄，小医仙俏皮一笑。
<br/>　　蹑手蹑脚的迈着猫步，轻轻走到窗前，娇躯紧紧贴在他结实的后背上，莲藕一般的玉臂紧紧的环在他的腰上。
<br/>　　两团饱满圆润的峰峦挤压在背部，能够清晰的感觉到弹性是何等的惊人。
<br/>　　回过神来，慕清霄苦笑一声，这小妮子一大清早就勾引自己，难道跟自己在一起，真的学坏了？
<br/>　　守着这样一个倾国倾城的女子，能看能摸，唯独不能吃，慕清霄都觉得有些蛋疼。
<br/>　　转过身来，温柔的捏了捏她小巧的鼻子，道：“快去洗漱吧，今天我们就启程去迦南学院，那里对你现阶段修炼也有帮助。”
<br/>　　“嗯。”
<br/>　　小医仙乖巧的点了点小脑袋，却没有离开，而是静静的靠在他的怀中，贪婪的嗅着他的气味，美女对男人来说是毒药，男人又何尝不是。
<br/>　　温存片刻，小医仙才恋恋不舍的离开他的怀抱。
<br/>　　......
<br/>　　......
<br/>　　迦南学院，是一座斗气大陆著名的学府。
<br/>　　作为斗气大陆上最古老的学院，里面的天才更是不知何几，学院分为内外两院，内院中的天才更是万里挑一...
<br/>　　站在上空，望着脚下的枫城，药皇韩枫一死，这里恐怕也要乱作一团了吧，空间一阵扭曲，天空中的两道身影就凭空消失。
<br/>　　半个时辰后，慕清霄就按照地图上的路线，带着小医仙越来越接近迦南学院，空间一阵扭曲，两道身影就凭空出现在了一片山脉上空。
<br/>　　放眼望去，正好能瞧见，远处两座雄伟巨峰的夹角处，有着一座若隐若现的小镇，那便是和平镇，黑角域城镇之一。
<br/>　　山脉中央有条直通小镇的黄土大道，这便是从黑角域中，进入迦南学院周边第一座小镇的唯一道路！
<br/>　　视线中，小镇门口左面不远处的有一颗大树，大树成漆黑之色，树冠四面八方的蔓延而开，张牙舞爪，在夕阳余晖下，透着一抹淡淡的阴冷。
<br/>　　站在上空，目光缓缓在树上扫过，只见在那树枝叉间，一些尸体，直接被穿插着挂在上面，轻风刮来，摇摇晃晃，发出令人毛骨悚然的嘎吱声响。
<br/>　　这种渗透出阴森气息的树木，便是死灵树。
<br/>　　在黑角域中，这株死灵树的名声，几乎已经达到了某种让人闻风丧胆的地步，很多在黑角域中待了一些年头的人，永远都不会忘记，那一次黑角域与迦南学院间的一场殊死血战。
<br/>　　当年，那场血战的确切起因，已经没有太多的人去考究。
<br/>　　唯一能够让得人知道的，便是那场血战后，两名斗王以及一名斗皇强者的尸体，被以一种近乎残酷的手段，挂在了这株死灵树之上...
<br/>　　至从那之后，迦南学院周边范围，便是陷入了一片与外界格格不入的宁静地带，再没有任何黑角域的强者，敢携带着满脸杀气，闯进这些小镇。
<br/>　　即使偶尔几次的骚乱，那骚扰之人，也会在不到一个小时后，成为一具尸体被插在死灵树上...
<br/>　　这些年间，死灵树的恶名，也是几乎扩散到了整个黑角域，因此，即使是那些穷凶极恶之徒，也是很少再有胆量进入这对于他们来说，便是坟墓的地所。
<br/>　　小镇的街道上，有着不少行人，街道两旁，摆着各种小摊，一些小孩在街道上穿梭打闹嬉笑，这一副安详融洽的氛围，与黑角域中，几乎是两种截然不同的表现。
<br/>　　而这座小镇，也有着相对的规则，凡是进入和平镇的人，必须报出自己身份，姓名等等，否则，将会直接被驱逐出去。
<br/>　　望着视线中和平温馨的小镇，慕清霄微微一笑，搂在小医仙飘然落在了通往小镇的道路上。
<br/>　　道路上有着不少行人，两人并没有套上黑袍，而是以真面目示人，之前穿上黑袍，是因为城市中秩序混乱，但这里完全不同。
<br/>　　周围行人，许多明显是从黑角域了过来的，他们体内明显有许多凶戾之气，但眉宇间却没有一丝杀气溢出。
<br/>　　只不过，这种隐匿内心杀气的模样非常的不自然，看上去就像是强行压制一般，因此，道路上大部分行人的脸色，都有显得极其怪异。
<br/>　　走在道路上，两人的容貌自然吸引来了许多异样的目光。
<br/>　　慕清霄自然不用说，神骏完美的容貌，高贵无比的气质，小医仙经过七彩韵灵果，淬体液长时间的改造，气质也远不是往日能比的。
<br/>　　看到小医仙一身白裙，前-凸后-翘，犹如仙子，路上许多男子眼中都抱有贪婪，却不敢付诸行动。
<br/>　　小医仙紧紧跟在慕清霄身旁，完全不理会周围的目光，顺着树木葱郁的黄土道路，缓缓来到小镇附近。
<br/>　　这里的气息，明显没有黑角域中的压抑，黑角域中的混乱气氛与秩序，完全被隔绝在了外面。
<br/>　　几分钟后，慕清霄停下脚步。
<br/>　　抬头望了一眼小镇门口处的匾额，上面的名字极其普通平庸，若是放在黑角域任何一处地方，恐怕第二天一早就会被砸的稀巴烂。
<br/>　　在慕清霄与小医仙前脚刚踏进镇门，便有一道奇异波动在两人身上扫过。
<br/>　　没有理会带有检测味道的奇异波动，慕清霄抬头望着繁荣温馨的街道，不禁有些感慨，迦南学院不愧是大陆最古老的学院。
<br/>　　能在混乱的黑角域中，将周围变成一片净土，威慑力不可谓不大。
<br/>　　街道上的行人，显然也注意到了慕清霄与小医仙，只是见两人容貌完美，看似人畜无害的模样，眼中的警惕也松懈了一些。
<br/>　　慕清霄在行人身上扫了一眼，其中斗者居多，甚至有些连斗之气旋都没有凝聚，但眼中却没有一丝惊恐，因为，有迦南学院守护着这片土地。
<br/>　　刚要进入小镇，却见屋顶有几十道身影疾驰而来，片刻后，身影就来到小镇门口处，拦在了慕清霄与小医仙的面前。
<br/>　　“凡是进入和平镇的人，必须报出自己的身份，姓名等等，否则，将直接被驱逐出去。”
