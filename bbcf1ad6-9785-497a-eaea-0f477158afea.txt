　　一座山清水秀的小山之上，一个四五岁大小的小孩坐在高石之上，抬头看着天空上的星星，在发呆。若仔细看，便会发现，这个小孩眼中时而会闪着一丝沧桑。
<br/>　　这个小孩便是伏羲了。
<br/>　　伏羲自从出世后，便表现的异常聪明，三日能言，五日能走，在稍微大些，便能够对事物有自己的看法。
<br/>　　自从上次异象出现后，部落之人也知道了伏羲不是怪物，而是人族大贤，他们母子也被接回到部落之中。所以，如今，他是生活在部落之中。
<br/>　　“来到此界已有五年了，看来，我是回不去了。没想到，这世界与自己的世界有着惊人的相似度。而且，我竟然也叫伏羲，这难道是巧合吗？”
<br/>　　小孩喃喃自语道，声音轻微，只有自己能够听到。
<br/>　　若是此时有人能够听到小孩的话，心中便会掀起惊涛骇浪。眼前这个小孩竟然不是洪荒世界的。不错，眼前这个名为伏羲的小孩并不是洪荒世界的人，他来自一个名叫神州的地方。在他那个世界，与此界的洪荒世界有着惊人的相似度。神州也有洪荒，只不过那里的洪荒只是亿万星辰中的一颗，他们称之为帝星。
<br/>　　神州也分远古神话时代，与上古时代。神话时代，域外天魔真主带领无尽天魔入侵神州，道祖是当时唯一的仙，与真主大战，双双消失在宇宙中，之后不见踪影。
<br/>　　之后，双方爆发大战，神州至尊参战，大败天魔，天魔退去之后，因为神州各大势力忌惮洪荒帝星的强大实力，联手入侵帝星。大战爆发，妖皇死去，娲皇战死，佛皇被杀，通天战死，元始战死，……道尊最后不得不牺牲自己，将洪荒帝星封印，深藏星空，使得帝星之人，得以苟活。
<br/>　　上古时代，人族新生，茹毛饮血，食不果腹，妖魔横行，人族如无根浮萍，在夹缝之中生存。那时，伏羲出世，创八卦，定山河，趋利避害，证道至尊，带领人族走上自强。之后，他毅然走向星空，与当年大战洪荒的势力死战，为后人铺路，也正是那一战，他死了。
<br/>　　死亡，有时亦是新生。
<br/>　　没想到，伏羲死后却来到了这个名为洪荒的世界。
<br/>　　在这个世界，他发现，比他之前的世界更加危险，人物虽然都差不多，但在实力方面却比他之前的世界，强了无数倍。更不用说洪荒七圣那等毁天灭地，重塑乾坤的人物了，他简直不敢想象。
<br/>　　在这个世界中，却是没有远古神话时代的域外天魔，但却有龙汉大劫，巫妖大劫这等无量劫。
<br/>　　在这个世界，人族却比他想象中的要好得多，人族的实力也非常强大，有人族武祖创武道，人族武皇崛起，大战巫妖两族，如今更是深不可测，远远不是自己之前的世界可相比的。
<br/>　　思绪之中飘过以前的种种，伏羲看着自己如今小小的身体，深深的叹了一口气，道：
<br/>　　“罢了，罢了！既来之，则安之。既然我已经来到这是世上，我便要完成我的使命。若是我没有猜错的话，我的使命就是带领人族大兴。如今的人族虽然强盛，但还远没有达到大兴的地步。至于之前的世界，我已经尽力了，后面的就要看后人了。”
<br/>　　说完，伏羲眼中闪过一丝坚定，看向虚空，转身向着山下跑去。一步便是数丈，显然，他的实力并不像表面一样。
<br/>　　昆仑山，玉虚宫中，元始看着眼前的水镜，笑了一声：“有趣，有趣！”
<br/>　　他面前的水镜上，显示的正是伏羲抬头望星空的一幕。
<br/>　　“穿越者都出现了，这个世界越来越有趣了，呵呵！”
<br/>　　……
<br/>　　数十年过去了，如今的伏羲已经是一族之长了。在此期间，伏羲表现出非同凡人的思想，他教化人族，制定人伦纲常，叔伯兄弟，姑婶姐妹。而且这种影响也一点点的向外扩张出去，其他部落纷纷效仿。
<br/>　　他的武道修为也是堪称恐怖，如今才过去数十年，便已经到了金丹武宗境至极，只差一步，便可进入神通王者境。
<br/>　　同时，伏羲也多了一个妹妹，名为女娲。当然，此女娲并非圣人女娲，但要说两者之间没有关系，各大圣人谁也不会相信。
<br/>　　在此期间，伏羲带领部落编网捉鱼，狩猎采果，驯养温顺的野兽。就这样，伏羲之名渐渐传了开来，影响着周边的部落，并向着远处传去。
<br/>　　人族生活越来越好，部落发展也越来越快，周围的部落慢慢都前来投靠，风氏部落渐渐壮大，伏羲的声望也越来越高。
<br/>　　数百年就这样过去了。
<br/>　　伏羲之名如今已经传遍洪荒人族，在人族中的声望也越来越高。但伏羲并不快乐，他知道自己还差着最关键的一步，而且，有了自己的前世，他也知道差在哪里。
<br/>　　对于创八卦一事，他也想过，可惜，究竟是此界的法则高晦涩高深超上一世，他前世的八怪已经不适合此界了，所以他需要重新推演。但要推演八卦，却缺少了最重要的东西，否则会非常耗时。
<br/>　　想到此处，他便向自己的老师龟灵圣母说明情况，毕竟龟灵圣母乃是圣人的弟子，说不定会有什么解决之道。
<br/>　　拜见龟灵之后，伏羲直接说明来意，想要求得一法，为人族趋利避害，挣脱苦海。
<br/>　　龟灵一听，顿时眯起了眼睛，便开始演算天机。片刻之后，龟灵圣母对着伏羲笑道：“你的确是有一番机缘，与你所求之事有关，也罢，为师便替你跑一趟吧。”
<br/>　　说完，龟灵圣母便向着昆仑山飞去。
<br/>　　回到昆仑山，龟灵圣母直接向通天教主说明来意，通天听后大笑道：“此事我已知晓，你只管前去教导伏羲，时机不到，莫要强求，时机一到，机缘便到。”
<br/>　　“是，老师。”
<br/>　　尽管有些疑惑，但龟灵圣母并没有多问，而是直接返回人族。
<br/>　　在她走后，通天脸色变得严肃起来，他想了想，便直接传音将此事告诉女娲，毕竟，伏羲终究是女娲的大兄，对于伏羲证道一事，她比谁都关心。
<br/>　　女娲听后，脸上顿时露出一丝愤怒之色，道：“好你个通天，你门下弟子教导人皇，还需要本宫出手。事情不想着自己去解决，只想着拿功德，哼！”
<br/>　　对于通天的算计，女娲心知肚明，但是，此事也由不得她不烦忧啊，因为谁都知道，她对于伏羲的关心，超乎寻常，也正是如此，通天才会有机会算计她。
<br/>　　女娲眼神闪烁一阵，心中便有了思量，她默默道：“太上欠本宫的因果还没有还，看来此番因果还是要在此事上了结。”
<br/>　　说着，女娲直接动身，找到太上道主，说明来意。太上也知道此番因果，无可奈何，他只好答应了。同时，他心里也隐隐有些不快，那是对于通天的行为不快。通天教弟子，还要他出手。通天也早知道自己欠女娲因果，但他还是找上了女娲，难道他就没有考虑到这一层关系？他有些不信。不过，通天说到底也是自己的兄弟，他也没有太过在意，只是将这件事记在了心中。
<br/>　　看着女娲走后，太上想了想，心中便对此事有了计较，之后直接宣来自己的弟子玄都：
<br/>　　“来人，叫玄都来一趟。”
<br/>　　“是，老爷。”
<br/>　　……
