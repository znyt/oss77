　　其他巫族天才闻言，俱是露出了感兴趣之色。
<br/>　　从刚才苏莫出手，他们也看出了苏莫不简单，所以也想听听苏莫如何不凡！
<br/>　　巫森深吸了口气，凝重道：“此子武皇境四重之时，便能打败我！”
<br/>　　巫森没有隐瞒，也不怕折损自己面子，就是要告诉一众巫族天才，不能大意轻敌。
<br/>　　“武皇境四重打败武帝境？”众巫族天才闻言一惊，有这么恐怖？这怎么可能？
<br/>　　如此战力的天才，就算是在他们巫族主脉中，都极其罕见啊。
<br/>　　武皇境四重打败武帝，这种越级战斗能力，已经无比逆天了！
<br/>　　别说他们巫族，就算是整个天荒星域，都为数不多！
<br/>　　一个小小的苍穹世界，能诞生这么妖孽的天才？
<br/>　　这根本不可能吧！
<br/>　　不过，众人很快便回过神来，武帝和武帝可不相同。
<br/>　　巫森在苍穹世界修炼成武帝，那也只是苍穹世界的武帝，和外界的武帝根本不能同日而语。
<br/>　　在一众巫族天才看来，巫森虽然是武帝，但也就和外界普通的武皇境八、九重武者差不多，哪里能和外界的武帝相比。
<br/>　　所以，苏莫的战力虽然惊人，也没有什么逆天之处！
<br/>　　“好了！我知道了，你退下了吧！”
<br/>　　巫天御随意的挥了挥手，继续道：“待我们扫荡了苍穹世界，你带我们寻宝！”
<br/>　　巫森见巫天御面色淡然，显然没听进去他的告诫，心中暗叹。
<br/>　　若是对方大意的话，说不定会陨落于苏莫之手。
<br/>　　想到此处，巫森眸光一闪，道：“小友，你们带我离开苍穹世界，我感激不尽，特准备了一件礼物，还请笑纳！”
<br/>　　言罢，巫森身边空间一阵波动，一道倩影出现在了他的身边。
<br/>　　倩影是一名女子，容颜绝美，长发如瀑，但此女面色僵硬，眸中没有多少的神采。
<br/>　　此女不是别人，正是宏青璇。
<br/>　　“小友，此女乃是纯阴之体，万年罕见，可谓是极品炉鼎！”巫森面色郑重，继续道：“我将此女送于你，只求你诛杀苏莫！”
<br/>　　巫森对于苏莫，杀心极重，为了能杀死苏莫，不惜将宏青璇都送了出去。
<br/>　　当然，他一直保留着宏青璇，也是准备献给巫族试炼者，以搞好关系，离开苍穹世界。
<br/>　　不过，他也看出来了，其实根本不需要献上礼物，巫族试炼者也会带他离开。
<br/>　　但是，现在巫天御如此小觑苏莫，他还是决定将宏青璇献出来，以求对方不要大意。
<br/>　　“哦，纯阴之体？”巫天御闻言，眸中顿时闪过一道璀璨的神光。
<br/>　　其他巫族天才闻言，也均是面现意外之色。
<br/>　　纯阴之体？
<br/>　　苍穹世界这种小地方，居然还有这种亿万中无一的体质？
<br/>　　众人凝神一看，果然如此，只见宏青璇体内，蕴含浓郁的纯阴之气，的确是纯阴之体，天下少有的极品炉鼎。
<br/>　　只不过，此女的修为未免也太低了吧！连武皇境都不到！
<br/>　　“不错！”巫天御的嘴角勾起了一丝笑意。
<br/>　　“青璇？”
<br/>　　另一边，苏莫看到宏青璇出现，顿时一惊，而后他的双眸顿时赤红了起来，心中的杀意，如同沸腾的火山，轰然炸开。
<br/>　　“杀！”
<br/>　　毫不犹豫，苏莫怒吼一声，身形一闪，如同一道幻影，急速向巫天御冲了过去。
<br/>　　十亿魔剑瞬间出现在了他的手中，随即猛然一挥。
<br/>　　刹那之间，一道恐怖的万丈剑气，如同开天之刃，裹挟无匹锋芒，斩碎虚空，横贯天地，先一步向巫天御袭杀而去。
<br/>　　咻咻咻！！
<br/>　　与此同时，苏莫心神一动，道道剑光闪烁而出，二十八柄本命灵剑如同万箭齐射，裹挟惊天杀机，爆射长空，直击巫森。
<br/>　　苏莫这一连串的攻击，不是为了杀巫天御，也不是为了杀巫森，而是为了救宏青璇。
<br/>　　宏青璇一现身，他便再也抑制不住了，心中都有些疯狂了起来。
<br/>　　宏青璇，是他生命中最重要的人之一，却被当成物品送来送去，他心中怎能不怒！
<br/>　　当然，苏莫也没有失去理智，二十八柄本命灵剑，不仅能杀人，更能守护宏青璇的安危，最起码不会被攻击余波波及。
<br/>　　“杀！”
<br/>　　苏莫一出手，司空炎、海明江等近百名苍穹神宫的强者，便也立刻出手了，纷纷四散而开，祭出强大的攻击，杀向一众巫族天才。
<br/>　　刹那之间，各种恐怖的攻击照耀八方，浩荡九天，如同道道天地洪流，铺天盖地的向一众巫族天才杀去。
<br/>　　不说司空炎等九宫之主，海明江、帝释、钱副阁主等人，均修炼了苍穹神宫的帝级武学，实力早已不是几年前可比。
<br/>　　众人的攻击，威力滔天，如同怒海狂涛，杀机无限。
<br/>　　“自不量力！”巫天御见此眸中杀机一闪，他速度极快，袖袍一挥，宏青璇便立刻消失不见了。
<br/>　　唰！
<br/>　　随即，巫天御手中长枪一抖，紫光爆闪，犀利无匹的枪芒如蛟龙出海，瞬间破碎虚空，和苏莫的剑气相击在了一起。
<br/>　　轰！
<br/>　　爆炸惊天，毁灭之气席卷。
<br/>　　于此同时，眼见二十八柄灵剑袭来，巫森哪里敢抵挡分毫，立刻急速暴退。
<br/>　　巫娴则是身形一闪，拦在了巫森的前方，她玉手一拍，一道巨大的玄力手掌，碾压虚空，狠狠的向到来的灵剑镇杀而去。
<br/>　　轰！轰！轰！！
<br/>　　巨掌与灵剑相击，发出一声声震天的爆响，巨掌立刻炸裂，化为滔天气浪。
<br/>　　而二十八柄本命灵剑，则是纷纷弹开，四散而去。
<br/>　　苏莫的第一轮攻击，丝毫无功，被巫天御和巫娴两人，分别出手挡住了。
<br/>　　而一众巫族的天才，着实强大，面对司空炎等人的攻击，个个爆发出绝强的战力，将所有攻击尽数湮灭。
<br/>　　一时间，毁天灭地的气浪如同大海狂啸、怒海奔腾，现场一片混乱。
<br/>　　嗖嗖嗖！！
<br/>　　破空声不断的响起，无论是苍穹神宫的强者，还是巫族的天才，纷纷四散而开，各自挑选自己的对手。
<br/>　　不过，苍穹神宫人数太少，基本上都是一人面对数人的局面。
<br/>　　“杀！”
<br/>　　一击无功，苏莫怒喝一声，速度不减，身形穿过席卷的气浪，飞临高空，一剑狠狠的斩下，斩向巫天御。
<br/>　　宏青璇明显是被巫天御收进了某种空间中，所以，苏莫现在的首要目标，就是要拿下甚至是斩杀巫天御。
<br/>　　待斩杀了巫天御，再对付其他人就容易多了！
